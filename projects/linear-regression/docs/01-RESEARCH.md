# 01 - 市场调研

## 项目背景

线性回归是机器学习中最基础、最重要的算法之一。它由统计学家 Francis Galton 在 19 世纪提出，至今仍是数据分析和机器学习的核心工具。

## 线性回归历史

### 发展历程

| 年代 | 里程碑 | 贡献者 |
|------|--------|--------|
| 1805 | 最小二乘法 | Adrien-Marie Legendre |
| 1875 | 回归概念 | Francis Galton |
| 1900 | 高斯-马尔可夫定理 | Carl Friedrich Gauss |
| 1950s | 统计学习理论 | 多位统计学家 |
| 1970s | 正则化方法 | Hoerl & Kennard (Ridge) |
| 1996 | Lasso | Robert Tibshirani |
| 2005 | Elastic Net | Hui Zou & Trevor Hastie |

### 核心概念演变

1. **最小二乘法**：最小化残差平方和
2. **梯度下降**：迭代优化方法
3. **正规方程**：闭式解方法
4. **正则化**：防止过拟合
5. **特征工程**：提升模型表达能力

## 优缺点分析

### 优点

1. **简单易懂**：数学原理清晰，易于解释
2. **计算高效**：训练和预测速度快
3. **可解释性强**：权重直接反映特征重要性
4. **理论完善**：有坚实的统计学理论基础
5. **基准模型**：常作为其他模型的基准
6. **无需调参**：超参数少，易于使用

### 缺点

1. **线性假设**：只能捕捉线性关系
2. **对异常值敏感**：MSE 损失对异常值敏感
3. **多重共线性**：特征高度相关时不稳定
4. **特征独立假设**：忽略了特征间交互
5. **欠拟合风险**：对复杂模式表达能力不足

## 竞品分析

### 1. scikit-learn

**特点**：
- API 设计优秀（fit/predict 模式）
- 文档完善，社区活跃
- 性能优化好（底层 C 实现）
- 支持多种回归变体

**可借鉴**：
- 接口设计规范
- 参数命名约定
- 模型评估工具

### 2. Statsmodels

**特点**：
- 统计推断功能强大
- 提供 p 值、置信区间
- 适合统计分析

**可借鉴**：
- 统计指标设计
- 摘要报告格式

### 3. TensorFlow/PyTorch

**特点**：
- 支持自动微分
- GPU 加速
- 深度学习框架的一部分

**可借鉴**：
- 自动微分思想
- 批量训练方式
