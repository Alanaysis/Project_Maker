# 01 - 市场调研

## 项目背景

线性回归是机器学习中最基础、最重要的算法之一。它是理解更复杂模型的基础，广泛应用于预测、趋势分析等领域。

## 核心概念

### 1. 线性回归定义

线性回归是一种用于建模因变量（目标）与一个或多个自变量（特征）之间关系的统计方法。

**数学表示**：
```
y = w * x + b
```
其中：
- `y` 是预测值
- `w` 是权重（斜率）
- `x` 是输入特征
- `b` 是偏置（截距）

### 2. 梯度下降

梯度下降是一种优化算法，用于最小化损失函数。

**更新规则**：
```
w = w - learning_rate * ∂Loss/∂w
b = b - learning_rate * ∂Loss/∂b
```

### 3. 损失函数（MSE）

均方误差（Mean Squared Error）是线性回归最常用的损失函数：

```
MSE = (1/n) * Σ(y_pred - y_true)²
```
