# 02 - 需求文档

## 功能需求

### 1. 核心模型

#### LinearRegression 类
- **初始化**：设置学习率、迭代次数等超参数
- **fit 方法**：训练模型
  - 输入：特征矩阵 X, 目标值 y
  - 输出：训练后的模型
- **predict 方法**：预测
  - 输入：特征矩阵 X
  - 输出：预测值 y_pred

### 2. 损失函数

#### MSE Loss
- 计算均方误差
- 支持批量计算
- 返回标量损失值

### 3. 梯度计算

#### 梯度下降
- 计算权重梯度
- 计算偏置梯度
- 支持不同的学习率

### 4. 可视化

#### 训练过程可视化
- 损失曲线图
- 参数更新轨迹
- 拟合效果展示

## 非功能需求

### 1. 代码质量
- 清晰的代码结构
- 详细的注释
- 完整的类型提示

### 2. 测试覆盖
- 单元测试覆盖率 > 80%
- 边界条件测试
- 集成测试

### 3. 文档
- 详细的 README
- API 文档
- 学习笔记

## 技术约束

1. **Python 版本**：>= 3.8
2. **依赖**：仅使用 NumPy 和 Matplotlib
3. **无外部 ML 框架**：从零实现

## 用户故事

1. 作为学习者，我想查看损失函数的下降曲线，以理解训练过程
2. 作为学习者，我想观察参数更新，以理解梯度下降
3. 作为学习者，我想看到拟合效果，以验证模型正确性

## 验收标准

1. 模型能够收敛（损失下降）
2. 预测结果在合理范围内
3. 所有测试通过
4. 可视化清晰易懂
