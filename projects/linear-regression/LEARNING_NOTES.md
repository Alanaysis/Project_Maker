# 学习笔记 - 线性回归

## 1. 线性回归基础

### 1.1 什么是线性回归？

线性回归是一种用于建模因变量（目标）与一个或多个自变量（特征）之间关系的统计方法。

**核心思想**：找到一条直线（或超平面），使得数据点到这条直线的距离之和最小。

### 1.2 数学表示

**简单线性回归**（单特征）：
```
y = w * x + b
```

**多元线性回归**（多特征）：
```
y = w₁x₁ + w₂x₂ + ... + wₙxₙ + b
```

**向量形式**：
```
y = X @ w + b
```

其中：
- `X` 是特征矩阵，形状 (n_samples, n_features)
- `w` 是权重向量，形状 (n_features,)
- `b` 是偏置（标量）
- `y` 是预测值

### 1.3 几何解释

- 单特征：找到一条直线拟合数据点
- 双特征：找到一个平面拟合数据点
- 更多特征：找到一个超平面拟合数据点

## 2. 损失函数

### 2.1 什么是损失函数？

损失函数用于衡量模型预测值与真实值之间的差异。训练的目标是最小化损失函数。

### 2.2 均方误差 (MSE)

```
MSE = (1/n) * Σ(y_pred - y_true)²
```

**优点**：
- 数学性质好，便于求导
- 对大误差惩罚更重
- 有唯一最优解

**缺点**：
- 对异常值敏感
- 解释性不如 MAE

### 2.3 平均绝对误差 (MAE)

```
MAE = (1/n) * Σ|y_pred - y_true|
```

**优点**：
- 对异常值鲁棒
- 解释直观

**缺点**：
- 在零点不可导
- 可能有多个最优解

### 2.4 MSE vs MAE

| 特性 | MSE | MAE |
|------|-----|-----|
| 对异常值敏感度 | 高 | 低 |
| 可导性 | 处处可导 | 零点不可导 |
| 最优解唯一性 | 是 | 不一定 |
| 计算效率 | 高 | 高 |

## 3. 梯度下降

### 3.1 什么是梯度下降？

梯度下降是一种优化算法，用于找到函数的最小值。

**核心思想**：沿着梯度的反方向更新参数，因为梯度指向函数增长最快的方向。

### 3.2 数学原理

对于函数 f(x)，梯度是：
```
∇f = [∂f/∂x₁, ∂f/∂x₂, ..., ∂f/∂xₙ]
```

更新规则：
```
x_new = x_old - learning_rate * ∇f
```

### 3.3 线性回归的梯度

对于线性回归模型：
```
y_pred = X @ w + b
Loss = (1/n) * Σ(y_pred - y_true)²
```

对 w 求偏导：
```
∂Loss/∂w = (2/n) * X.T @ (y_pred - y_true)
```

对 b 求偏导：
```
∂Loss/∂b = (2/n) * Σ(y_pred - y_true)
```

### 3.4 学习率

学习率控制参数更新的步长：

- **太小**：收敛慢，需要更多迭代
- **太大**：可能震荡，甚至发散
- **合适**：稳定收敛

**经验**：从 0.01 或 0.1 开始，观察损失曲线调整。

### 3.5 梯度下降变体

| 变体 | 特点 |
|------|------|
| 批量梯度下降 (BGD) | 使用全部数据计算梯度 |
| 随机梯度下降 (SGD) | 使用单个样本计算梯度 |
| 小批量梯度下降 (Mini-batch) | 使用部分数据计算梯度 |

## 4. 模型训练流程

### 4.1 完整流程

```
1. 初始化参数 (w, b)
2. 重复以下步骤：
   a. 前向传播：计算 y_pred = X @ w + b
   b. 计算损失：Loss = MSE(y_true, y_pred)
   c. 反向传播：计算梯度 dw, db
   d. 参数更新：w = w - lr * dw, b = b - lr * db
3. 直到收敛或达到最大迭代次数
```

### 4.2 前向传播

前向传播是将输入数据通过模型得到预测值的过程：

```
输入 X → 线性变换 (X @ w + b) → 输出 y_pred
```

### 4.3 反向传播

反向传播是计算损失函数对参数的梯度的过程：

```
损失 Loss → 链式法则 → 梯度 dw, db
```

### 4.4 参数更新

使用梯度下降更新参数：

```
w = w - learning_rate * dw
b = b - learning_rate * db
```

## 5. 模型评估

### 5.1 R² 分数

R² 分数衡量模型解释数据变异的比例：

```
R² = 1 - SS_res / SS_tot
```

其中：
- `SS_res = Σ(y_true - y_pred)²` (残差平方和)
- `SS_tot = Σ(y_true - mean(y_true))²` (总平方和)

**解读**：
- R² = 1：完美拟合
- R² = 0：与均值预测一样好
- R² < 0：比均值预测还差

### 5.2 训练集 vs 测试集

- **训练集**：用于训练模型
- **测试集**：用于评估模型泛化能力

**划分比例**：通常 80% 训练，20% 测试

### 5.3 过拟合与欠拟合

| 类型 | 训练误差 | 测试误差 | 原因 |
|------|----------|----------|------|
| 欠拟合 | 高 | 高 | 模型太简单 |
| 过拟合 | 低 | 高 | 模型太复杂 |
| 良好 | 低 | 低 | 模型合适 |

## 6. 实践技巧

### 6.1 数据预处理

- **标准化**：将特征缩放到相似范围
- **归一化**：将特征缩放到 [0, 1]
- **处理缺失值**：删除或填充

### 6.2 特征工程

- **特征选择**：选择相关特征
- **特征组合**：创建新特征
- **特征变换**：多项式特征等

### 6.3 超参数调优

- **学习率**：尝试 0.001, 0.01, 0.1
- **迭代次数**：观察损失曲线
- **批量大小**：32, 64, 128

## 7. 常见问题

### Q1: 为什么损失不下降？

可能原因：
1. 学习率太大或太小
2. 数据没有标准化
3. 梯度计算错误

### Q2: 如何判断模型收敛？

观察损失曲线：
- 损失不再明显下降
- 损失在某个值附近震荡
- 达到预设的收敛阈值

### Q3: 如何处理多特征？

使用矩阵运算：
```python
# X: (n_samples, n_features)
# w: (n_features,)
y_pred = X @ w + b
```

### Q4: 如何选择学习率？

1. 从 0.01 开始
2. 观察损失曲线
3. 如果震荡，减小学习率
4. 如果收敛慢，增大学习率
