# K-Means 聚类算法调研报告

## 1. 算法概述

K-Means 是一种无监督学习算法，用于将数据集划分为 K 个簇。算法通过迭代优化目标函数（簇内平方和），使得每个数据点到其所属簇中心的距离最小化。

## 2. 核心原理

### 2.1 算法流程
1. **初始化**：随机选择 K 个数据点作为初始簇中心
2. **分配**：将每个数据点分配到距离最近的簇中心
3. **更新**：重新计算每个簇的中心（均值）
4. **收敛判断**：重复步骤 2-3 直到中心不再变化或达到最大迭代次数

### 2.2 距离度量
- **欧氏距离**：最常用，适用于连续数值特征
  ```
  d(x, y) = sqrt(Σ(xi - yi)²)
  ```
- **曼哈顿距离**：适用于网格状数据
  ```
  d(x, y) = Σ|xi - yi|
  ```
- **余弦相似度**：适用于文本数据

### 2.3 目标函数
最小化簇内平方和（Within-Cluster Sum of Squares, WCSS）：
```
J = Σ Σ ||x - μi||²
    i x∈Ci
```

## 3. 关键问题

### 3.1 K 值选择
- **肘部法则**：绘制不同 K 值对应的 WCSS，选择拐点
- **轮廓系数**：评估簇的紧密度和分离度
- **Gap Statistic**：比较实际数据与随机数据的 WCSS

### 3.2 初始化方法
- **随机初始化**：简单但可能陷入局部最优
- **K-Means++**：更智能的初始化，降低对初始值的敏感性

### 3.3 收敛条件
- 中心点变化小于阈值
- 达到最大迭代次数
- 簇分配不再变化

## 4. 应用场景

### 4.1 典型应用
- **客户细分**：根据购买行为将客户分组
- **图像压缩**：减少颜色数量
- **文档聚类**：文本主题分类
- **异常检测**：识别离群点

### 4.2 优缺点
**优点**：
- 简单易实现
- 计算效率高
- 结果易于解释

**缺点**：
- 需要预先指定 K 值
- 对初始值敏感
- 假设簇是球形的
- 对异常值敏感
