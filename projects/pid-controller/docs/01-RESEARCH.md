# PID 控制器 - 调研文档

## 1. PID 控制概述

PID (Proportional-Integral-Derivative) 控制器是工业控制中最广泛使用的反馈控制器。据统计，超过 95% 的工业控制回路使用 PID 或其变体。

### 1.1 历史背景

- **1911年**: Elmer Sperry 在船舶自动驾驶仪中首次使用 PID 控制概念
- **1922年**: Nicolas Minorsky 发表了 PID 控制理论的系统性分析
- **1940年**: Taylor Instruments 推出第一台商用 PID 控制器
- **1950年代**: Ziegler 和 Nichols 提出了著名的 PID 整定方法

### 1.2 应用领域

- **温度控制**: 烤箱、恒温器、化学反应器
- **运动控制**: 电机速度、机器人关节、CNC 机床
- **过程控制**: 液位、流量、压力控制
- **航空航天**: 飞行控制、姿态稳定
- **汽车**: 巡航控制、ABS 制动系统

## 2. PID 控制原理

### 2.1 基本公式

```
u(t) = Kp*e(t) + Ki*∫e(τ)dτ + Kd*de(t)/dt
```

其中:
- `u(t)` = 控制输出
- `e(t)` = 误差 = 设定值 - 测量值
- `Kp` = 比例增益
- `Ki` = 积分增益
- `Kd` = 微分增益

### 2.2 各项作用

| 项 | 作用 | 优点 | 缺点 |
|---|------|------|------|
| P (比例) | 对当前误差做出响应 | 快速响应 | 无法消除稳态误差 |
| I (积分) | 消除累积误差 | 消除稳态误差 | 可能导致超调和振荡 |
| D (微分) | 预测误差趋势 | 减少超调，提供阻尼 | 对噪声敏感 |

### 2.3 传递函数

连续域:
```
G(s) = Kp + Ki/s + Kd*s
```

离散域 (后向差分):
```
G(z) = Kp + Ki*T/(1-z^-1) + Kd*(1-z^-1)/T
```

## 3. PID 整定方法

### 3.1 Ziegler-Nichols 方法 (闭环)

1. 设 Ki=0, Kd=0
2. 逐渐增加 Kp 直到系统产生等幅振荡
3. 记录此时的 Kp 为 Ku (临界增益)
4. 记录振荡周期为 Tu (临界周期)
5. 使用公式计算 PID 参数

| 控制器类型 | Kp | Ki | Kd |
|-----------|-----|-----|-----|
| P | 0.5*Ku | - | - |
| PI | 0.45*Ku | 0.54*Ku/Tu | - |
| PID | 0.6*Ku | 1.2*Ku/Tu | 0.075*Ku*Tu |

### 3.2 Cohen-Coon 方法 (开环)

1. 对开环系统施加阶跃输入
2. 记录响应曲线 (S 形)
3. 识别参数:
   - L: 死时间
   - τ: 时间常数
   - K: 过程增益
4. 使用 Cohen-Coon 公式计算

### 3.3 Tyreus-Luyben 方法

修改的 Ziegler-Nichols 方法，产生更保守的整定:
- Kp = 0.45 * Ku
- Ki = 0.45 * Ku / (2.2 * Tu)
- Kd = 0.45 * Ku * (Tu / 6.3)

## 4. 系统模型

### 4.1 一阶系统

传递函数: G(s) = K / (τs + 1)

特点:
- 无超调
- 指数趋近稳态
- τ 决定响应速度

### 4.2 二阶系统

传递函数: G(s) = K*ωn² / (s² + 2ζωn*s + ωn²)

特点:
- ζ < 1: 欠阻尼 (有超调)
- ζ = 1: 临界阻尼 (最快无超调)
- ζ > 1: 过阻尼 (慢，无超调)

## 5. 实际考虑

### 5.1 积分饱和 (Integral Windup)

当执行器饱和时，积分项继续累积导致:
- 大的超调
- 长的恢复时间

解决方案:
- 积分限幅
- 条件积分
- 反计算抗饱和

### 5.2 微分噪声

微分项对高频噪声敏感，解决方案:
- 低通滤波器
- 只对测量值微分 (不微分设定值变化)

### 5.3 设定值冲击 (Derivative Kick)

设定值突变时微分项产生冲击，解决方案:
- 微分项只作用于测量值
- 设定值斜坡化

## 6. 改进 PID 变体

### 6.1 积分分离 (Integral Separation)

当误差较大时，暂停积分累积，防止积分饱和:
- 当 |e(t)| > 阈值时，停止积分
- 当 |e(t)| <= 阈值时，恢复积分
- 适用于大范围设定值变化的场景

### 6.2 微分先行 (Derivative on Measurement)

微分项只作用于测量值，而非误差:
- d_term = -Kd * d(PV)/dt
- 避免设定值突变时的微分冲击
- 本项目已实现此优化

### 6.3 不完全微分 (Incomplete Derivative)

在微分项上增加一阶低通滤波:
- d_filtered = β * d_raw + (1-β) * d_prev
- 进一步减少高频噪声影响
- β 越小，滤波越强

### 6.4 带死区 PID (Dead Zone PID)

当误差在死区内时，控制器不动作:
- 当 |e(t)| < 死区宽度时，输出为 0
- 适用于有摩擦或间隙的系统
- 避免在小误差时的频繁动作

## 7. 系统模型扩展

### 7.1 延迟系统 (Delay System)

传递函数: G(s) = K * e^(-Ls) / (τs + 1)

特点:
- L: 死时间 (延迟)
- 输出对输入的响应有时间延迟
- 常见于管道流量、化学过程、网络控制

### 7.2 实际应用系统

- **温度控制**: 一阶 + 延迟，加热器热惯性
- **电机控制**: 二阶系统，惯量 + 摩擦
- **液位控制**: 一阶系统，非线性阀门特性
