# 线性规划研究文档

## 1. 问题背景

### 1.1 什么是线性规划

线性规划 (Linear Programming, LP) 是数学优化的一个分支，用于在一组线性约束条件下，最大化或最小化一个线性目标函数。它是运筹学中应用最广泛的数学方法之一。

一般形式：
```
max/min  c^T x
s.t.     Ax  <=/>=/==  b
          x >= 0
```

其中：
- x ∈ R^n 是决策变量
- c ∈ R^n 是目标函数系数
- A ∈ R^{m×n} 是约束系数矩阵
- b ∈ R^m 是右端项

### 1.2 历史发展

- **1947年** - George Dantzig 提出单纯形法
- **1950年代** - 对偶理论建立
- **1979年** - Leonid Khachiyan 提出椭球法 (多项式时间)
- **1984年** - Narendra Karmarkar 提出内点法
- **现代** - 大规模 LP 求解器 (CPLEX, Gurobi, GLPK)

### 1.3 应用领域

| 领域 | 应用 |
|------|------|
| 生产管理 | 生产计划、库存控制、人员排班 |
| 物流运输 | 运输问题、车辆路径、网络流 |
| 金融投资 | 投资组合优化、风险管理 |
| 通信网络 | 流量分配、带宽分配 |
| 能源系统 | 电力调度、能源分配 |
| 机器学习 | SVM、线性回归、特征选择 |

## 2. 单纯形法理论

### 2.1 几何直觉

线性规划的可行域是一个凸多面体。最优解一定在顶点处取得（如果存在）。

单纯形法的核心思想：
1. 从一个顶点（基可行解）出发
2. 沿着边移动到相邻的更优顶点
3. 重复直到到达最优顶点

### 2.2 标准形式

引入松弛变量将不等式约束转为等式：
```
max  c^T x
s.t. Ax + Is = b
      x >= 0, s >= 0
```

其中 I 是 m×m 单位矩阵，s 是松弛变量。

### 2.3 基可行解

将变量分为基变量 x_B 和非基变量 x_N：
```
B x_B + N x_N = b
x_B = B^{-1}(b - N x_N)
```

当 x_N = 0 时，x_B = B^{-1} b 称为基可行解（如果 x_B >= 0）。

### 2.4 最优性条件

检验数（reduced cost）：
```
σ_j = c_j - c_B^T B^{-1} A_j
```

- 若所有 σ_j <= 0，当前解最优
- 若存在 σ_j > 0，对应的非基变量可以入基

### 2.5 单纯形表

单纯形表将所有信息整合在一个矩阵中：

```
| B^{-1}A | B^{-1}I | B^{-1}b |
|---------|---------|---------|
|  c - z  |    0    |   z     |
```

其中 z = c_B^T B^{-1} b 是当前目标函数值。

## 3. 大M法与两阶段法

### 3.1 为什么需要这些方法

标准单纯形法要求：
- 初始基可行解容易找到（松弛变量构成初始基）
- 所有约束为 <= 类型

对于包含 >= 或 == 约束的问题，需要额外处理。

### 3.2 大M法

引入人工变量 a_i，并在目标函数中加入惩罚项 -M * a_i：
```
max  c^T x - M * sum(a_i)
s.t. 原约束 + 人工变量 = b
      x, a >= 0
```

M 是一个很大的正数，使得最优解中人工变量必须为 0。

**优点**：实现简单，一次求解
**缺点**：M 的选择影响数值稳定性

### 3.3 两阶段法

**Phase I**：最小化人工变量之和
```
min  sum(a_i)
s.t. 原约束 + 人工变量 = b
```

若最优值 = 0，找到可行解；否则原问题不可行。

**Phase II**：在 Phase I 的可行基上优化原目标函数

**优点**：数值稳定，不依赖 M 的选择
**缺点**：需要两次求解

## 4. 对偶理论

### 4.1 对偶问题构造

原问题 (Primal)：
```
max  c^T x
s.t. Ax <= b
      x >= 0
```

对偶问题 (Dual)：
```
min  b^T y
s.t. A^T y >= c
      y >= 0
```

构造规则：
| 原问题 | 对偶问题 |
|--------|----------|
| max | min |
| <= 约束 | y >= 0 |
| >= 约束 | y <= 0 |
| == 约束 | y 无约束 |
| x >= 0 | >= 约束 |
| x <= 0 | <= 约束 |
| x 无约束 | == 约束 |

### 4.2 弱对偶定理

对任何可行解 x 和 y：
```
c^T x <= b^T y
```

证明：
```
c^T x <= (A^T y)^T x = y^T A x <= y^T b = b^T y
```

### 4.3 强对偶定理

若原问题有最优解 x*，则对偶也有最优解 y*，且：
```
c^T x* = b^T y*
```

### 4.4 互补松弛条件

在最优解处：
```
y_i^* (b_i - A_i x*) = 0,  对所有 i
x_j^* (A_j^T y* - c_j) = 0,  对所有 j
```

含义：
- 若约束 i 严格成立（有松弛），则 y_i^* = 0
- 若 x_j^* > 0，则对应的对偶约束必须紧

## 5. 敏感性分析

### 5.1 目标函数系数变化

对于非基变量 x_j：
- σ_j = c_j - z_j <= 0 才能保持最优
- c_j 可以增加到 z_j 而不改变最优基

对于基变量 x_j：
- 变化会影响所有非基变量的检验数
- 需要重新计算允许范围

### 5.2 右端项变化

最优解 x_B = B^{-1} b 会变化：
- 需要保持 x_B >= 0（原始可行性）
- 影子价格 = 对偶变量 = 目标函数对 b 的偏导数

### 5.3 影子价格

影子价格 y_i^* 表示：
- 右端项 b_i 增加 1 单位，目标函数增加 y_i^*
- 仅在允许范围内有效

## 6. 运输问题与指派问题

### 6.1 运输问题

m 个供应点, n 个需求点, 最小化总运输成本。

特殊结构：约束矩阵是全幺模矩阵，LP 松弛自然得到整数解。

初始解方法：
- 西北角法
- 最小元素法
- Vogel 近似法

### 6.2 指派问题

运输问题的特例：m = n，所有供给 = 需求 = 1。

匈牙利算法：O(n^3) 时间复杂度。
