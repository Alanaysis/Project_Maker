# 密码学库需求文档

## 1. 功能需求

### 1.1 哈希算法模块

#### MD5
- [x] 支持任意长度输入
- [x] 返回32字符十六进制字符串
- [x] 支持分块更新
- [x] 支持字符串和字节输入

#### SHA-1
- [x] 支持任意长度输入
- [x] 返回40字符十六进制字符串
- [x] 支持分块更新
- [x] 支持字符串和字节输入

#### SHA-256
- [x] 支持任意长度输入
- [x] 返回64字符十六进制字符串
- [x] 支持分块更新
- [x] 支持字符串和字节输入

#### HMAC
- [x] 支持MD5、SHA-1、SHA-256
- [x] 支持密钥和消息输入
- [x] 支持验证功能
- [x] 常量时间比较（防时序攻击）

### 1.2 对称加密模块

#### AES
- [x] 支持AES-128、AES-192、AES-256
- [x] 支持ECB、CBC、CTR模式
- [x] PKCS7填充
- [x] 随机IV/nonce生成
- [x] 单块加密/解密

#### DES
- [x] 支持64位密钥
- [x] 支持ECB、CBC、CTR模式
- [x] PKCS7填充
- [x] 单块加密/解密

#### 分组模式
- [x] ECB模式（不推荐）
- [x] CBC模式（推荐）
- [x] CTR模式（推荐）
- [x] PKCS7填充/去填充

### 1.3 非对称加密模块

#### RSA
- [x] 支持512-4096位密钥
- [x] 公钥加密/私钥解密
- [x] 数字签名/验证
- [x] Miller-Rabin素性测试

#### ECDH
- [x] 支持secp256k1曲线
- [x] 支持P-256曲线
- [x] 密钥对生成
- [x] 共享密钥计算

#### 数字签名
- [x] RSA签名
- [x] ECDSA签名
- [x] 签名验证
- [x] 文件签名

### 1.4 编码模块

#### Base64
- [x] 标准Base64编码/解码
- [x] URL安全Base64
- [x] 填充处理

#### Hex
- [x] 十六进制编码/解码
- [x] 大小写支持
- [x] 格式化输出
- [x] 整数转换

### 1.5 实用工具模块

#### 密码存储
- [x] PBKDF2密钥派生
- [x] 随机盐值生成
- [x] 可配置迭代次数
- [x] 密码验证

#### 数据加密
- [x] AES/DES加密
- [x] 多种加密模式
- [x] 字符串加密
- [x] JSON序列化

#### 数字签名工具
- [x] RSA/ECDSA签名
- [x] 文件签名
- [x] 签名验证

## 2. 非功能需求

### 2.1 性能需求
- 哈希计算：> 1MB/s
- AES加密：> 10MB/s
- RSA签名：> 100次/秒（2048位）

### 2.2 安全需求
- 使用安全的随机数生成器
- 常量时间比较（防时序攻击）
- 安全的密钥存储
- 避免内存中的敏感数据残留

### 2.3 可用需求
- 清晰的API设计
- 完整的文档和示例
- 易于扩展的模块结构

### 2.4 兼容需求
- Python 3.7+
- 跨平台支持（Windows、Linux、macOS）

## 3. 接口需求

### 3.1 统一接口
```python
# 哈希
hash_value = SHA256.hash(data)

# 加密
encryptor = DataEncryptor('aes', 256, 'cbc')
encrypted = encryptor.encrypt(data)

# 签名
sig = SignatureUtil('ecdsa')
signature = sig.sign_message(data)
```

### 3.2 链式调用
```python
hasher = SHA256()
hasher.update("part1").update("part2")
result = hasher.hexdigest()
```
