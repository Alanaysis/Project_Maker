# 密码学库研究文档

## 1. 密码学概述

密码学是研究编制密码和破译密码的技术科学。现代密码学主要包含以下领域：

### 1.1 哈希函数
- **定义**：将任意长度的输入转换为固定长度输出的函数
- **特性**：单向性、抗碰撞性、雪崩效应
- **应用**：数据完整性校验、密码存储、数字签名

### 1.2 对称加密
- **定义**：加密和解密使用相同密钥的加密算法
- **分类**：分组密码、流密码
- **应用**：数据加密、文件加密、通信加密

### 1.3 非对称加密
- **定义**：使用公钥加密、私钥解密的加密算法
- **基础**：数学难题（大数分解、离散对数、椭圆曲线）
- **应用**：密钥交换、数字签名、身份认证

## 2. 哈希算法研究

### 2.1 MD5
- **输出长度**：128位（16字节）
- **轮数**：4轮，每轮16步
- **状态**：已被证明存在碰撞，不建议用于安全场景
- **学习价值**：理解哈希函数的基本结构

### 2.2 SHA-1
- **输出长度**：160位（20字节）
- **轮数**：4轮，每轮20步，共80步
- **状态**：已被证明存在碰撞攻击
- **应用**：Git版本控制、旧系统兼容

### 2.3 SHA-256
- **输出长度**：256位（32字节）
- **轮数**：64步
- **状态**：目前安全，广泛使用
- **应用**：TLS/SSL、比特币、数字证书

### 2.4 HMAC
- **定义**：基于哈希的消息认证码
- **结构**：H(K XOR opad || H(K XOR ipad || message))
- **应用**：API认证、会话令牌、数据完整性验证

## 3. 对称加密研究

### 3.1 AES
- **分组大小**：128位（16字节）
- **密钥长度**：128/192/256位
- **轮数**：10/12/14轮
- **操作**：SubBytes、ShiftRows、MixColumns、AddRoundKey
- **状态**：目前最广泛使用的对称加密标准

### 3.2 DES
- **分组大小**：64位（8字节）
- **密钥长度**：56位（实际64位，8位校验）
- **轮数**：16轮
- **操作**：初始置换、Feistel网络、逆初始置换
- **状态**：已被破解，仅用于学习

### 3.3 分组模式
- **ECB**：简单但不安全，相同明文产生相同密文
- **CBC**：需要随机IV，串行处理，安全性好
- **CTR**：将分组密码变为流密码，可并行处理

## 4. 非对称加密研究

### 4.1 RSA
- **数学基础**：大整数分解难题
- **密钥生成**：选择素数p,q → 计算n=pq → 选择e → 计算d
- **安全性**：基于n的分解难度
- **应用**：密钥交换、数字签名、小数据加密

### 4.2 ECDH
- **数学基础**：椭圆曲线离散对数难题
- **曲线**：secp256k1（比特币）、P-256（NIST）
- **优势**：相同安全性下密钥更短
- **应用**：TLS密钥交换、加密货币

### 4.3 数字签名
- **RSA签名**：使用私钥签名，公钥验证
- **ECDSA**：椭圆曲线数字签名算法
- **应用**：软件签名、文档认证、区块链

## 5. 编码研究

### 5.1 Base64
- **字符集**：A-Z, a-z, 0-9, +, /
- **编码比**：每3字节编码为4字符
- **应用**：电子邮件、URL、数据传输

### 5.2 Hex
- **字符集**：0-9, a-f
- **编码比**：每字节编码为2字符
- **应用**：哈希值显示、内存地址、颜色代码

## 6. 安全考虑

### 6.1 常见攻击
- **暴力破解**：尝试所有可能的密钥
- **字典攻击**：使用常见密码列表
- **中间人攻击**：截获和篡改通信
- **时序攻击**：通过执行时间推断信息

### 6.2 安全建议
- 使用足够长的密钥（AES-256、RSA-2048+）
- 使用随机盐值存储密码
- 使用安全的随机数生成器
- 避免使用已破解的算法（MD5、DES）
