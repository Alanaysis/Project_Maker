# 01 - 研究笔记

## 密码学基础

### 对称加密 vs 非对称加密

| 特性 | 对称加密 | 非对称加密 |
|------|----------|------------|
| 密钥 | 单一密钥 | 公钥/私钥对 |
| 速度 | 快 | 慢 |
| 安全性 | 密钥分发问题 | 数学难题保证 |
| 典型算法 | AES, DES | RSA, ECC |

### AES (Advanced Encryption Standard)

**基本参数：**
- 块大小：128位
- 密钥长度：128/192/256位
- 轮数：10/12/14轮

**AES变换：**
1. **SubBytes**: 非线性字节替换（S-box）
2. **ShiftRows**: 行移位
3. **MixColumns**: 列混合
4. **AddRoundKey**: 轮密钥加

**工作模式：**
- ECB: 电子密码本（不推荐）
- CBC: 密码块链接（本项目实现）
- CTR: 计数器模式
- GCM: 认证加密

### RSA算法

**数学基础：**
- 大数分解难题
- 欧拉定理
- 模逆运算

**密钥生成：**
```
1. 选择大素数 p, q
2. 计算 n = p × q
3. 计算 φ(n) = (p-1) × (q-1)
4. 选择 e 使得 gcd(e, φ(n)) = 1
5. 计算 d = e⁻¹ mod φ(n)
```

**加密/解密：**
```
加密: c = mᵉ mod n
解密: m = cᵈ mod n
```

### 椭圆曲线密码学 (ECC)

**椭圆曲线方程：**
```
y² = x³ + ax + b (mod p)
```

**关键特性：**
- 椭圆曲线离散对数问题 (ECDLP)
- 较小的密钥长度提供相同安全性
- 256位ECC ≈ 3072位RSA

**ECDH密钥交换：**
```
Alice: a (私钥), A = aG (公钥)
Bob: b (私钥), B = bG (公钥)
共享密钥: S = aB = bA = abG
```

**ECDSA签名：**
```
签名: (r, s) = Sign(private_key, message_hash)
验证: Verify(public_key, message_hash, r, s)
```

## 安全考虑

### 已知攻击
1. **时序攻击**: 通过测量执行时间推断密钥
2. **侧信道攻击**: 利用物理信息泄漏
3. **填充预言攻击**: 利用解密错误信息

### 最佳实践
- 使用经过验证的库
- 密钥安全存储
- 使用随机IV/nonce
- 定期轮换密钥
