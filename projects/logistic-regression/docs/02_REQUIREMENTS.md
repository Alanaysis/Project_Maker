# 02_REQUIREMENTS.md - 需求分析文档

## 1. 项目概述

### 1.1 项目目标

从零实现逻辑回归算法，深入理解分类算法的原理和实现细节。

### 1.2 目标用户

- 机器学习初学者
- 数据科学学习者
- 算法工程师

### 1.3 使用场景

- 学习逻辑回归原理
- 理解分类算法实现
- 掌握模型评估方法
- 实践特征工程技巧

## 2. 功能需求

### 2.1 基础逻辑回归

#### 2.1.1 核心功能

| 功能 | 优先级 | 描述 |
|------|--------|------|
| Sigmoid函数 | P0 | 将线性输出映射到概率 |
| 交叉熵损失 | P0 | 计算预测与真实的差异 |
| 梯度下降 | P0 | 优化损失函数 |
| 模型训练 | P0 | 训练逻辑回归模型 |
| 概率预测 | P0 | 输出属于正类的概率 |
| 类别预测 | P0 | 输出预测类别 |
| 模型评估 | P0 | 计算准确率 |

#### 2.1.2 参数配置

| 参数 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| learning_rate | float | 0.01 | 学习率 |
| n_iterations | int | 1000 | 迭代次数 |
| regularization | float | 0.0 | L2正则化强度 |
| threshold | float | 0.5 | 分类阈值 |
| verbose | bool | False | 是否打印训练过程 |

### 2.2 多分类

#### 2.2.1 One-vs-Rest (OvR)

| 功能 | 优先级 | 描述 |
|------|--------|------|
| 训练 | P0 | 为每个类别训练二分类器 |
| 预测 | P0 | 选择概率最高的类别 |
| 概率输出 | P1 | 输出每个类别的概率 |

#### 2.2.2 One-vs-One (OvO)

| 功能 | 优先级 | 描述 |
|------|--------|------|
| 训练 | P0 | 为每对类别训练分类器 |
| 预测 | P0 | 使用投票机制选择类别 |

#### 2.2.3 Softmax回归

| 功能 | 优先级 | 描述 |
|------|--------|------|
| 训练 | P0 | 直接训练多分类模型 |
| 预测 | P0 | 输出概率分布 |
| 概率输出 | P0 | 输出每个类别的概率 |

### 2.3 正则化

#### 2.3.1 L1正则化

| 功能 | 优先级 | 描述 |
|------|--------|------|
| 训练 | P0 | 带L1惩罚的训练 |
| 特征选择 | P0 | 产生稀疏权重 |

#### 2.3.2 L2正则化

| 功能 | 优先级 | 描述 |
|------|--------|------|
| 训练 | P0 | 带L2惩罚的训练 |
| 防止过拟合 | P0 | 惩罚大的权重 |

#### 2.3.3 Elastic Net

| 功能 | 优先级 | 描述 |
|------|--------|------|
| 训练 | P0 | 结合L1和L2 |
| 参数控制 | P0 | 控制L1和L2的比例 |

### 2.4 模型评估

#### 2.4.1 分类指标

| 指标 | 优先级 | 公式 | 说明 |
|------|--------|------|------|
| 准确率 | P0 | (TP+TN)/(TP+TN+FP+FN) | 正确预测的比例 |
| 精确率 | P0 | TP/(TP+FP) | 预测为正中实际为正的比例 |
| 召回率 | P0 | TP/(TP+FN) | 实际为正中被正确预测的比例 |
| F1分数 | P0 | 2*P*R/(P+R) | 精确率和召回率的调和平均 |

#### 2.4.2 曲线指标

| 指标 | 优先级 | 说明 |
|------|--------|------|
| ROC曲线 | P0 | TPR vs FPR曲线 |
| AUC | P0 | ROC曲线下面积 |
| PR曲线 | P1 | 精确率-召回率曲线 |

#### 2.4.3 混淆矩阵

| 功能 | 优先级 | 描述 |
|------|--------|------|
| 计算 | P0 | 计算TP, TN, FP, FN |
| 可视化 | P1 | 显示混淆矩阵 |

### 2.5 特征工程

| 方法 | 优先级 | 公式 | 说明 |

| 归一化 | P0 | x' = (x - min) / (max - min) | 缩放到[0,1] |

#### 2.5.2 特征选择

|------|--------|------|
| 方差阈值 | P1 | 移除低方差特征 |

#### 2.5.3 交叉验证

## 4. 非功能需求

### 4.1 性能需求

- 训练时间：对于1000个样本，1000次迭代，应在1秒内完成
- 预测时间：对于单个样本，应在1毫秒内完成
- 内存使用：对于1000个样本，7个特征，内存使用应小于100MB

### 4.2 可用性需求

- API设计简洁易用
- 提供详细的文档和注释
- 提供使用示例

### 4.3 可维护性需求

- 完整的测试覆盖

### 4.4 兼容性需求
