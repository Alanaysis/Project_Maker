**完成日期**：__________

- [ ] TCP 协议报文结构
- [ ] UDP 协议报文结构
- [ ] ICMP 协议报文结构

**重点笔记**：

1. IP 头部关键字段：
   ```
   版本、头部长度、总长度、TTL、协议、源IP、目的IP
   ```

2. TCP 头部关键字段：
   ```
   源端口、目的端口、序列号、确认号、标志位(SYN/ACK/FIN)
   ```

3. 三次握手过程：
   ```
   客户端 → SYN → 服务器
   服务器 → SYN+ACK → 客户端
   客户端 → ACK → 服务器
   ```

4. 四次挥手过程：
   ```
   主动方 → FIN → 被动方
   被动方 → ACK → 主动方
   被动方 → FIN → 主动方
   主动方 → ACK → 被动方
   ```

### 第二阶段：Netfilter 框架

**完成日期**：__________

**学习内容**：
- [ ] Netfilter 钩子机制
- [ ] 五个钩子点的作用
- [ ] libnetfilter_queue 使用

**重点笔记**：

1. Netfilter 五个钩子点：
   ```
   NF_INET_PRE_ROUTING   → 路由前
   NF_INET_LOCAL_IN      → 本地输入
   NF_INET_FORWARD       → 转发
   NF_INET_LOCAL_OUT     → 本地输出
   NF_INET_POST_ROUTING  → 路由后
   ```

2. NFQUEUE 工作流程：
   ```
   内核 → NFQUEUE → 用户空间 → 内核
   ```

3. libnetfilter_queue API：
   ```c
   nfq_open()           // 打开队列
   nfq_bind_pf()        // 绑定协议族
   nfq_create_queue()   // 创建队列
   nfq_set_mode()       // 设置模式
   nfq_handle_packet()  // 处理包
   ```

**学习内容**：

- [ ] 超时管理

**重点笔记**：

1. TCP 状态机：
   ```
   NONE → SYN_SENT → SYN_RECV → ESTABLISHED → FIN_WAIT → TIME_WAIT → CLOSED
   ```

2. 连接表设计：
   ```
   哈希表：key = (src_ip, dst_ip, src_port, dst_port, protocol)
   value = connection_state
   ```

3. 超时时间：
   - TCP ESTABLISHED: 300 秒
   - TCP NEW: 30 秒
   - UDP: 30 秒
   - ICMP: 10 秒
