# 更新日志

本项目的所有重要更改都将记录在此文件。

格式基于 [Keep a Changelog](https://keepachangelog.com/zh-CN/1.0.0/)，
并且本项目遵循 [语义化版本](https://semver.org/lang/zh-CN/)。

## [1.0.0] - 2024-01-01

### 新增

- 包解析模块
  - 支持以太网帧解析
  - 支持 IPv4 头部解析
  - 支持 TCP 头部解析
  - 支持 UDP 头部解析
  - 支持 ICMP 头部解析

- 规则引擎
  - 支持基于协议的过滤
  - 支持基于 IP 地址的过滤（包括 CIDR）
  - 支持基于端口的过滤
  - 支持规则顺序匹配
  - 支持默认动作

- 状态管理
  - 支持 TCP 连接状态跟踪
  - 支持 UDP "连接" 跟踪
  - 支持 ICMP 请求/响应匹配
  - 支持连接超时管理
  - 基于哈希表的连接表

- 入侵检测
  - SYN Flood 检测
  - 端口扫描检测
  - 异常包检测
  - ICMP Flood 检测
  - UDP Flood 检测

- 日志系统
  - 支持多级别日志（DEBUG, INFO, WARNING, ERROR, ALERT）
  - 支持文件和标准输出
  - 支持包日志记录
  - 支持告警日志记录

- 文档
  - README.md - 项目说明
  - 市场调研报告
  - 需求分析文档
  - 技术设计文档
  - 产品思维文档
  - 开发手册
  - 学习笔记模板
  - 快速开始指南

- 测试
  - 包解析测试
  - 规则引擎测试
  - 状态管理测试
  - IDS 测试

- 示例
  - 简单过滤示例
  - 示例配置文件

### 技术栈

- C 语言
- Linux Netfilter
- libnetfilter_queue
- libpcap
- POSIX 线程

### 已知限制

- 仅支持 IPv4
- 单线程实现
- 简化的规则匹配算法
- 不支持 NAT
- 不支持应用层过滤

## 版本说明

### 版本号规则

- **主版本号**：不兼容的 API 修改
- **次版本号**：向下兼容的功能性新增
- **修订号**：向下兼容的问题修正

### 变更类型

- **新增**：新功能
- **变更**：现有功能的变更
- **弃用**：即将移除的功能
- **移除**：已移除的功能
- **修复**：问题修复
- **安全**：安全相关的变更
