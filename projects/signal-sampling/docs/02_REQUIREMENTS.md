# 信号采样重建 - 需求文档

## 1. 项目概述

### 1.1 项目目标

实现信号采样与重建的核心算法，包括:
- 奈奎斯特采样
- 过采样/欠采样
- 均匀/非均匀量化
- 零阶保持/一阶保持/sinc 插值重建
- 混叠现象演示
- 抗混叠滤波
- 音频采样应用
- 图像采样应用

### 1.2 目标用户

- 信号处理学习者
- 电子工程学生
- 嵌入式开发者
- 音频/图像处理开发者

### 1.3 设计原则

1. **教学优先**: 代码清晰，注释完整
2. **从零实现**: 不依赖信号处理库
3. **模块化**: 各模块独立，可单独使用
4. **可测试**: 完整的单元测试

## 2. 功能需求

### 2.1 采样模块

#### FR-2.1.1 奈奎斯特采样

**描述**: 以奈奎斯特率或更高采样率进行采样。

**输入**:
- 信号函数
- 信号频率
- 采样频率
- 持续时间

**输出**:
- 采样时间点
- 采样值

**约束**:
- 采样频率 >= 2 * 信号频率

#### FR-2.1.2 过采样

**描述**: 以远高于奈奎斯特率的频率进行采样。

**输入**:
- 信号函数
- 信号频率
- 过采样倍数
- 持续时间

**输出**:
- 采样时间点
- 采样值
- 采样频率

#### FR-2.1.3 欠采样

**描述**: 以低于奈奎斯特率的频率进行采样。

**输入**:
- 信号函数
- 信号频率
- 采样频率
- 持续时间

**输出**:
- 采样时间点
- 采样值
- 混叠频率

### 2.2 量化模块

#### FR-2.2.1 均匀量化

**描述**: 将连续幅度等间隔量化。

**输入**:
- 信号
- 量化位数
- 最小值
- 最大值

**输出**:
- 量化值
- 量化索引

**属性**:
- 量化电平数
- 量化步长
- SQNR

#### FR-2.2.2 非均匀量化

**描述**: 使用 mu 律或 A 律压缩进行非均匀量化。

**输入**:
- 信号
- 量化位数
- mu 或 A 参数

**输出**:
- 量化值
- 量化索引

### 2.3 重建模块

#### FR-2.3.1 零阶保持

**描述**: 每个采样值保持到下一个采样点。

**输入**:
- 采样时间点
- 采样值
- 重建时间轴

**输出**:
- 重建信号

#### FR-2.3.2 一阶保持

**描述**: 采样点之间线性插值。

**输入**:
- 采样时间点
- 采样值
- 重建时间轴

**输出**:
- 重建信号

#### FR-2.3.3 sinc 插值

**描述**: 使用 sinc 函数进行理想重建。

**输入**:
- 采样时间点
- 采样值
- 重建时间轴
- 采样频率

**输出**:
- 重建信号

### 2.4 混叠模块

#### FR-2.4.1 混叠演示

**描述**: 演示欠采样产生的混叠现象。

**输入**:
- 信号频率
- 采样频率
- 持续时间

**输出**:
- 原始信号
- 采样点
- 混叠信号
- 混叠频率

#### FR-2.4.2 抗混叠滤波

**描述**: 在采样前使用低通滤波器限制带宽。

**输入**:
- 信号
- 采样频率
- 截止频率

**输出**:
- 滤波后信号

### 2.5 音频采样模块

#### FR-2.5.1 音频采样器

**描述**: 模拟音频信号的采样过程。

**功能**:
- 支持多种采样率 (8k, 22.05k, 44.1k, 48k, 96k)
- 支持多种量化位数 (8, 16, 24)
- 支持单声道/立体声
- PCM 编码/解码

#### FR-2.5.2 重采样

**描述**: 改变音频信号的采样率。

**输入**:
- 原始采样值
- 原始采样率
- 目标采样率

**输出**:
- 重采样后的值

### 2.6 图像采样模块

#### FR-2.6.1 降采样

**描述**: 减少图像像素数。

**方法**:
- 平均法
- 抽取法

**选项**:
- 抗混叠滤波

#### FR-2.6.2 上采样

**描述**: 增加图像像素数。

**方法**:
- 最近邻
- 双线性
- 双三次

| FOH 重建 (1000 点) | < 1 ms |
| sinc 重建 (1000 点) | < 100 ms |
