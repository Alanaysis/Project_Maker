# 信号采样重建 - 调研报告

## 1. 项目背景

### 1.1 信号采样的重要性

信号采样是连接模拟世界和数字世界的桥梁。从音频录制、图像处理到通信系统，采样技术无处不在。

**核心问题**: 如何将连续的模拟信号转换为离散的数字信号，并能完美重建？

### 1.2 历史发展

| 年代 | 里程碑 | 贡献者 |
|------|--------|--------|
| 1920s | 采样定理雏形 | Nyquist |
| 1940s | 严格数学证明 | Shannon |
| 1950s | 实际应用 | 电话系统 |
| 1980s | CD 音频标准 | Sony/Philips |
| 2000s | 高清音频 | SACD/DVD-Audio |

### 1.3 应用领域

1. **音频处理**: 录音、混音、编码 (MP3, AAC)
2. **图像处理**: 摄像头、显示器、压缩 (JPEG, PNG)
3. **通信系统**: 调制解调、软件无线电
4. **医学影像**: CT、MRI、超声
5. **工业控制**: 数据采集、传感器接口

## 2. 核心理论

### 2.1 奈奎斯特-香农采样定理

**定理陈述**:

如果连续信号 x(t) 的傅里叶变换 X(f) 在 |f| > fmax 时为零，
则 x(t) 可以由其采样值 x(nTs) 完全确定，其中 fs = 1/Ts >= 2*fmax。

**数学表达**:

采样:
```
x[n] = x(nTs)
```

重建:
```
x(t) = sum_{n=-∞}^{∞} x[n] * sinc((t - nTs) / Ts)
```

其中 sinc(x) = sin(πx) / (πx)

**证明思路**:
1. 采样在频域是周期延拓
2. 当 fs >= 2*fmax 时，频谱不重叠
3. 理想低通滤波可以恢复原始频谱

### 2.2 混叠 (Aliasing)

**定义**: 当 fs < 2*fmax 时，高频分量伪装成低频分量。

**混叠频率计算**:
```
f_alias = |f - k * fs|, k = round(f / fs)
```

**例子**:
- f = 100 Hz, fs = 80 Hz
- k = round(100/80) = 1
- f_alias = |100 - 80| = 20 Hz

**混叠的后果**:
- 信息不可恢复
- 产生虚假信号
- 图像中出现摩尔纹

### 2.3 量化 (Quantization)

**定义**: 将连续幅度转换为离散电平。

**均匀量化**:
```
Q(x) = round((x - Vmin) / Δ) * Δ + Vmin
Δ = (Vmax - Vmin) / (2^b - 1)
```

**SQNR (信号量化噪声比)**:
```
SQNR = 6.02 * b + 1.76 dB (正弦信号)
```

**非均匀量化**:
- mu 律: 北美、日本
- A 律: 欧洲、中国

### 2.4 信号重建

**零阶保持 (ZOH)**:
```
x(t) = x[n], nTs <= t < (n+1)Ts
```

**一阶保持 (FOH)**:
```
x(t) = x[n] + (x[n+1] - x[n]) * (t - nTs) / Ts
```

**sinc 插值**:
```
x(t) = sum x[n] * sinc((t - nTs) / Ts)
```

## 3. 技术调研

### 3.1 采样技术

#### 过采样 (Oversampling)

**定义**: 采样率远高于奈奎斯特率。

**优点**:
- 降低混叠风险
- 简化抗混叠滤波器设计
- 提高信噪比
- 允许噪声整形

**应用**:
- 音频 DAC: 通常 4x 或 8x 过采样
- Sigma-Delta ADC: 128x 或 256x 过采样

#### 欠采样 (Undersampling)

**定义**: 采样率低于奈奎斯特率。

**应用**:
- 带通采样: 利用信号带通特性
- 频谱分析: 频谱折叠
- 软件无线电: 直接采样 RF 信号

### 3.2 量化技术

#### 均匀量化

**优点**:
- 实现简单
- 误差均匀分布

**缺点**:
- 小信号 SQNR 低
- 动态范围有限

#### 非均匀量化

**mu 律压缩**:
```
y = sign(x) * ln(1 + mu*|x|) / ln(1 + mu)
```

**A 律压缩**:
```
y = sign(x) * A*|x| / (1 + ln(A)),     |x| < 1/A
y = sign(x) * (1 + ln(A*|x|)) / (1 + ln(A)),  1/A <= |x| <= 1
```

**优点**:
- 小信号精度高
- 动态范围大
- 符合人耳感知特性

### 3.3 重建技术

#### 零阶保持

**实现**: DAC 输出保持到下一个采样点。

**频域特性**:
```
H(f) = Ts * sinc(fTs) * exp(-jπfTs)
```

**补偿**: 需要后置低通滤波器。

#### 一阶保持

**实现**: 采样点之间线性插值。

**频域特性**:
```
H(f) = Ts * sinc²(fTs) * exp(-jπfTs)
```

**优点**: 比 ZOH 更平滑。

#### sinc 插值

**实现**: 使用 sinc 函数进行理想插值。

**特点**:
- 理论上完美重建
- 计算量大 (每个重建点需要所有采样点)
- 需要截断处理

### 3.4 抗混叠滤波

**目的**: 在采样前限制信号带宽。

**设计要点**:
- 截止频率: f_cutoff < fs/2
- 阻带衰减: 通常 > 60 dB
- 过渡带宽度: 影响滤波器复杂度

**常用滤波器**:
- 巴特沃斯: 最大平坦
- 切比雪夫: 等纹波
- 椭圆: 最窄过渡带

## 4. 实际应用调研

### 4.1 音频采样

| 标准 | 采样率 | 量化位数 | 应用 |
|------|--------|----------|------|
| 电话 | 8 kHz | 8 bit (mu律) | 语音通信 |
| AM 广播 | 22.05 kHz | 16 bit | 广播 |
| CD | 44.1 kHz | 16 bit | 音乐存储 |
| DVD | 48 kHz | 16/24 bit | 影视 |
| 高保真 | 96 kHz | 24 bit | 专业录音 |

**CD 采样率 44.1 kHz 的由来**:
- 人耳最高听觉: 20 kHz
- 奈奎斯特率: 40 kHz
- 过渡带: ~4 kHz
- 总计: 44.1 kHz (历史原因: 与视频帧率兼容)

### 4.2 图像采样

**采样过程**:
1. 光学系统聚焦
2. 传感器阵列采样
3. 量化编码

**混叠效果**:
- 锯齿 (jagged edges)
- 摩尔纹 (moiré patterns)
- 细节丢失

**抗混叠方法**:
- 光学低通滤波器 (OLPF)
- 软件滤波
- 过采样 + 降采样

### 4.3 通信系统

**软件无线电 (SDR)**:
- 直接采样 RF 信号
- 需要高速 ADC
- 利用带通采样定理

**调制解调**:
- 采样率决定带宽
- 量化精度影响 SNR
- 重建质量影响误码率
