# 信号采样重建 - 学习笔记

## 1. 奈奎斯特定理

### 核心概念

奈奎斯特定理 (Nyquist-Shannon 采样定理) 是数字信号处理的基石:

```
如果一个连续信号的最高频率分量为 fmax，
则以采样率 fs >= 2 * fmax 采样后，可以完美重建原始信号。
```

### 数学表达

采样过程:
```
x[n] = x(n * Ts), 其中 Ts = 1/fs
```

重建公式 (sinc 插值):
```
x(t) = sum(x[n] * sinc((t - n*Ts) / Ts))
```

其中 sinc(x) = sin(pi*x) / (pi*x)

### 关键理解

1. **奈奎斯特率**: fs_min = 2 * fmax
   - 这是完美重建的最低采样率
   - 实际应用中通常需要更高的采样率

2. **过采样**: fs >> 2 * fmax
   - 降低混叠风险
   - 简化抗混叠滤波器设计
   - 提高信噪比

3. **欠采样**: fs < 2 * fmax
   - 产生混叠
   - 高频分量伪装成低频分量
   - 混叠频率: f_alias = |f - k * fs|

### 工程意义

- **音频 CD**: 采样率 44.1 kHz (人耳最高 20 kHz)
- **电话**: 采样率 8 kHz (语音最高 4 kHz)
- **图像**: 采样率决定分辨率

## 2. 量化

### 均匀量化

将连续幅度等间隔划分:

```
量化步长: Δ = (Vmax - Vmin) / (2^b - 1)
量化值: Q(x) = round((x - Vmin) / Δ) * Δ + Vmin
```

**SQNR (信号量化噪声比)**:
```
SQNR = 6.02 * b + 1.76 dB (正弦信号)
```

每增加 1 bit，SQNR 增加约 6 dB。

### 非均匀量化

对小信号更精确，大信号更粗糙:

**mu 律压缩**:
```
y = sign(x) * ln(1 + mu*|x|) / ln(1 + mu)
```

**A 律压缩**:
```
y = sign(x) * A*|x| / (1 + ln(A)),     |x| < 1/A
y = sign(x) * (1 + ln(A*|x|)) / (1 + ln(A)),  1/A <= |x| <= 1
```

**应用**:
- mu 律: 北美、日本电话系统
- A 律: 欧洲、中国电话系统

### 量化误差

量化误差 e[n] = x[n] - Q(x[n])

- 均匀分布: |e| <= Δ/2
- 与信号不相关 (假设信号变化足够快)

## 3. 信号重建

### 零阶保持 (ZOH)

```
x(t) = x[n], nTs <= t < (n+1)Ts
```

**特点**:
- 阶梯状波形
- 实现简单，硬件友好
- 高频分量较多
- 需要后置低通滤波器

### 一阶保持 (FOH)

```
x(t) = x[n] + (x[n+1] - x[n]) * (t - nTs) / Ts
```

**特点**:
- 折线波形
- 比 ZOH 更平滑
- 计算量适中
- 仍有高频分量

### sinc 插值 (理想重建)

```
x(t) = sum(x[n] * sinc((t - nTs) / Ts))
```

**特点**:
- 理论上完美重建带限信号
- 计算量较大 (每个重建点需要所有采样点)
- 需要知道精确的采样频率
- 实际中需要截断 sinc 函数

## 4. 混叠

### 混叠现象

当采样率不足时，高频分量会伪装成低频分量:

```
f_alias = |f - k * fs|, k = round(f / fs)
```

**例子**:
- f = 100 Hz, fs = 80 Hz
- k = round(100/80) = 1
- f_alias = |100 - 80| = 20 Hz

### 混叠的后果

1. **音频**: 出现虚假的低频噪声
2. **图像**: 出现摩尔纹 (moiré pattern)
3. **数据**: 信息不可恢复

### 抗混叠滤波

在采样前使用低通滤波器:

```
x_filtered(t) = LPF{x(t)}
x[n] = x_filtered(n * Ts)
```

**设计要点**:
- 截止频率: f_cutoff < fs/2
- 需要足够的阻带衰减
- 过渡带宽度影响滤波器复杂度

## 5. 实际应用

### 音频采样

| 标准 | 采样率 | 量化位数 | 应用 |
|------|--------|----------|------|
| 电话 | 8 kHz | 8 bit (mu律) | 语音通信 |
| AM 广播 | 22.05 kHz | 16 bit | 广播 |
| CD | 44.1 kHz | 16 bit | 音乐存储 |
| DVD | 48 kHz | 16/24 bit | 影视 |
| 高保真 | 96 kHz | 24 bit | 专业录音 |

**为什么 CD 采样率是 44.1 kHz?**
- 人耳最高听觉: 20 kHz
- 奈奎斯特率: 40 kHz
- 过渡带: ~4 kHz
- 总计: 44.1 kHz (历史原因)

### 图像采样

**降采样**:
- 需要抗混叠滤波
- 方法: 平均、抽取
- 用途: 缩略图、金字塔

**上采样**:
- 需要插值
- 方法: 最近邻、双线性、双三次
- 用途: 放大、超分辨率

**混叠效果**:
- 锯齿 (jagged edges)
- 摩尔纹 (moiré patterns)
- 细节丢失

## 6. 实验心得

### 采样率选择

1. **满足奈奎斯特只是最低要求**
   - 实际需要留有余量
   - 过采样可以简化设计

2. **抗混叠滤波器设计**
   - 过渡带越窄，滤波器越复杂
   - 过采样可以放宽滤波器要求

### 量化精度

1. **8 bit 足以满足语音**
   - SQNR ≈ 50 dB
   - 人耳对语音的 SQNR 要求不高

2. **16 bit 满足音乐**
   - SQNR ≈ 98 dB
   - 动态范围足够

3. **24 bit 用于专业录音**
   - SQNR ≈ 146 dB
   - 留有余量处理

### 重建方法选择

1. **ZOH**: 硬件实现简单，用于 DAC
2. **FOH**: 信号较平滑，用于控制
3. **sinc**: 理论最优，用于数字处理

## 7. 常见误区

### 误区 1: 采样率越高越好

**事实**: 过高的采样率会:
- 增加数据量
- 增加处理负担
- 不一定提高质量

### 误区 2: 量化位数越高越好

**事实**: 过高的量化位数会:
- 增加存储空间
- 超过传感器精度无意义
- 量化噪声可能被其他噪声掩盖

### 误区 3: sinc 插值总是最好的

**事实**: sinc 插值:
- 计算量大
- 需要截断处理
- 边界效应明显
- 实际中常用近似方法

## 8. 进阶主题

### 过采样与噪声整形

- 过采样可以降低量化噪声功率谱密度
- 噪声整形将噪声推到高频
- Sigma-Delta 调制器应用

### 多速率信号处理

- 抽取 (decimation): 降低采样率
- 内插 (interpolation): 提高采样率
- 多相滤波器组

### 压缩感知

- 突破奈奎斯特限制
- 利用信号稀疏性
- 需要满足 RIP 条件
