# 研究文档

## 中文分词概述

中文分词是将连续的中文文本切分成有意义的词语序列的过程。与英文不同，中文没有空格分隔词语，因此分词是中文自然语言处理的基础任务。

## 主要分词方法

### 1. 基于词典的方法

#### 正向最大匹配 (Forward Maximum Matching, FMM)

**算法步骤**：
1. 从左到右扫描文本
2. 取词典中最长词长度的子串
3. 在词典中查找，找到则切分，否则减少一个字符继续查找
4. 重复直到文本结束

**优点**：
- 实现简单
- 速度快

**缺点**：
- 不能处理歧义
- 依赖词典质量

#### 逆向最大匹配 (Backward Maximum Matching, BMM)

**算法步骤**：
1. 从右到左扫描文本
2. 取词典中最长词长度的子串
3. 在词典中查找，找到则切分，否则减少一个字符继续查找
4. 重复直到文本开始

**优点**：
- 对某些歧义处理比 FMM 好

**缺点**：
- 同样依赖词典质量

### 2. 基于统计的方法

#### 隐马尔可夫模型 (Hidden Markov Model, HMM)

**核心思想**：
- 将分词问题转化为序列标注问题
- 每个字标注为：B(词首)、M(词中)、E(词尾)、S(单字词)

**模型要素**：
- 状态集合：{B, M, E, S}
- 观测集合：所有汉字
- 初始概率：π
- 转移概率：A
- 发射概率：B

**维特比算法**：
- 用于求解最优状态序列
- 时间复杂度：O(T * N^2)，其中 T 是序列长度，N 是状态数

### 3. 基于深度学习的方法

#### 条件随机场 (CRF)

- 考虑整个序列的全局最优
- 避免标记偏置问题

#### BiLSTM-CRF

- 结合双向 LSTM 和 CRF
- 能捕捉长距离依赖

## 评估指标

- **精确率 (Precision)**：正确切分的词数 / 切分出的词数
- **召回率 (Recall)**：正确切分的词数 / 标准答案的词数
- **F1 值**：2 * P * R / (P + R)
