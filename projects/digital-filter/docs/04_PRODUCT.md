# 产品文档：数字滤波器

## 1. 产品概述

数字滤波器库是一个 Python 工具包，用于设计、分析和应用数字滤波器。支持 FIR 和 IIR 两大类滤波器，提供完整的频率响应分析和可视化功能。

## 2. 目标用户

- **信号处理工程师**: 设计和验证滤波器
- **音频工程师**: 音频信号处理和降噪
- **研究人员**: 滤波器算法研究和教学
- **学生**: 学习数字信号处理

## 3. 核心功能

### 3.1 滤波器设计

| 功能 | 描述 |
|------|------|
| FIR 窗函数法 | 使用 Hamming/Hanning/Blackman/Kaiser 窗设计 FIR 滤波器 |
| FIR 频率采样法 | 基于频率域采样点设计 FIR 滤波器 |
| Butterworth | 最大平坦幅频响应的 IIR 滤波器 |
| Chebyshev I/II | 等纹波通带/阻带的 IIR 滤波器 |
| Elliptic | 最陡峭过渡带的 IIR 滤波器 |

### 3.2 滤波器类型

- **低通 (Lowpass)**: 通过低频，衰减高频
- **高通 (Highpass)**: 通过高频，衰减低频
- **带通 (Bandpass)**: 通过指定频段
- **带阻 (Bandstop/Notch)**: 衰减指定频段

### 3.3 频率响应分析

- 幅频响应曲线 (Magnitude Response)
- 相频响应曲线 (Phase Response)
- 群延迟 (Group Delay)
- 多滤波器对比图

### 3.4 实际应用

- **音频滤波**: 去除音频中的噪声和干扰
- **信号去噪**: 从含噪信号中恢复原始信号
- **频带分离**: 分离不同频率的信号分量

## 4. 使用场景

### 4.1 音频降噪
```
原始音频 → FIR/IIR 低通 → 去噪后音频
```

### 4.2 工频干扰去除
```
含 50Hz 干扰信号 → 带阻滤波器 → 干净信号
```

### 4.3 频带分离
```
混合信号 → 带通滤波器 → 特定频段信号
```

## 5. 性能指标

| 指标 | 目标值 |
|------|--------|
| FIR 滤波延迟 | < 100ms (10,000 点) |
| IIR 滤波延迟 | < 10ms (10,000 点) |
| 支持信号长度 | >= 100,000 点 |
| 频率分辨率 | fs / worN |

## 6. 输出格式

- **滤波后信号**: NumPy 数组
- **频率响应图**: PNG 文件
- **信号对比图**: PNG 文件
- **滤波器系数**: NumPy 数组
