# 研究文档：数字滤波器

## 1. 数字滤波器基础

数字滤波器是对离散时间信号进行运算处理的系统，通过差分方程实现信号的频率选择性处理。

### 1.1 线性时不变 (LTI) 系统

数字滤波器是 LTI 系统，满足：
- **线性**: `T[a*x1 + b*x2] = a*T[x1] + b*T[x2]`
- **时不变**: `T[x(n-k)] = y(n-k)`

### 1.2 差分方程

```
y(n) = Σ b(k)*x(n-k) - Σ a(k)*y(n-k)
       k=0..M            k=1..N
```

### 1.3 传递函数

```
H(z) = Y(z)/X(z) = Σ b(k)*z^(-k) / (1 + Σ a(k)*z^(-k))
```

## 2. FIR 滤波器

### 2.1 特点
- **有限脉冲响应**: h(n) 有限长度
- **线性相位**: 对称/反对称系数保证线性相位
- **总是稳定**: 无反馈，极点全在原点
- **实现**: 直接卷积或快速卷积 (FFT)

### 2.2 窗函数法设计

**步骤**:
1. 确定理想滤波器的脉冲响应 h_d(n)
2. 选择窗函数 w(n)
3. h(n) = h_d(n) * w(n)

**常用窗函数**:
| 窗函数 | 主瓣宽度 | 旁瓣衰减 | 过渡带宽 |
|--------|---------|---------|---------|
| 矩形窗 | 窄 | -13 dB | 窄 |
| Hanning | 中等 | -31 dB | 中等 |
| Hamming | 中等 | -41 dB | 中等 |
| Blackman | 宽 | -57 dB | 宽 |
| Kaiser | 可调 | 可调 | 可调 |

### 2.3 频率采样法设计

**原理**: 在频域均匀采样理想响应，IDFT 得到时域系数。

**步骤**:
1. 在 [0, 2π) 均匀采样 N 个频率点
2. 设置期望幅度响应 |H_d(k)|
3. h(n) = IDFT{H_d(k)}
4. 可选：加窗改善性能

**优点**: 可精确控制特定频率点的响应
**缺点**: 非采样点的响应由插值决定，可能有波动

## 3. IIR 滤波器

### 3.1 特点
- **无限脉冲响应**: 理论上无限长
- **非线性相位**: 反馈结构导致非线性相位
- **可能不稳定**: 需要检查极点位置
- **效率高**: 用较少阶数达到较好性能

### 3.2 Butterworth 滤波器

**特点**: 通带内最大平坦，无纹波。

**幅度响应**:
```
|H(jω)|² = 1 / (1 + (ω/ωc)^(2N))
```

**设计参数**: 阶数 N、截止频率 ωc

### 3.3 Chebyshev 滤波器

**Type I**: 通带等纹波，阻带单调
- 通带有纹波，过渡带更陡
- 参数: 阶数 N、纹波 ε、截止频率

**Type II**: 通带单调，阻带等纹波
- 阻带有纹波，通带平坦
- 参数: 阶数 N、阻带衰减 As、截止频率

### 3.4 椭圆滤波器 (Cauer)

**特点**: 通带和阻带均有等纹波，过渡带最陡。

**优势**: 给定阶数下过渡带最窄
**代价**: 通带和阻带都有纹波，非线性相位

## 4. 滤波器性能指标

### 4.1 频域指标
- **通带纹波** (Passband Ripple): 通带内最大增益波动
- **阻带衰减** (Stopband Attenuation): 阻带最小衰减
- **过渡带宽** (Transition Width): 通带边缘到阻带边缘的频率范围
- **3dB 截止频率**: 增益下降 3dB 的频率点

### 4.2 时域指标
- **群延迟**: 信号通过滤波器的延迟
- **建立时间**: 响应达到稳态的时间
- **过冲**: 阶跃响应的最大超调量
