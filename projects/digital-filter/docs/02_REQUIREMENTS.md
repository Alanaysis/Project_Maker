# 需求文档：数字滤波器

## 1. 功能需求

### 1.1 FIR 滤波器设计
- **FR-01**: 支持窗函数法设计 FIR 滤波器
  - 支持 Hamming、Hanning、Blackman、Kaiser、Rectangular 窗
  - 支持自定义窗函数
  - 自动处理偶数阶到奇数阶的转换
- **FR-02**: 支持频率采样法设计 FIR 滤波器
  - 接收理想频率响应采样点
  - 支持加窗平滑
- **FR-03**: 支持低通、高通、带通、带阻滤波器类型

### 1.2 IIR 滤波器设计
- **FR-04**: 支持 Butterworth 滤波器设计
  - 低通、高通、带通、带阻
  - 阶数可配置
- **FR-05**: 支持 Chebyshev Type I 滤波器设计
  - 通带纹波可配置
- **FR-06**: 支持 Chebyshev Type II 滤波器设计
  - 阻带衰减可配置
- **FR-07**: 支持椭圆滤波器设计
  - 通带纹波和阻带衰减可配置
- **FR-08**: 支持 SOS (Second-Order Sections) 表示和滤波

### 1.3 频率响应分析
- **FR-09**: 计算并绘制幅频响应
- **FR-10**: 计算并绘制相频响应
- **FR-11**: 计算群延迟
- **FR-12**: 支持多滤波器对比可视化

### 1.4 信号处理工具
- **FR-13**: 生成多频率分量测试信号
- **FR-14**: 添加指定 SNR 的高斯白噪声
- **FR-15**: 计算信噪比 (SNR)
- **FR-16**: 信号时域和频域可视化

### 1.5 滤波操作
- **FR-17**: 单向滤波 (lfilter)
- **FR-18**: 零相位滤波 (filtfilt)
- **FR-19**: SOS 滤波 (sosfilt)

## 2. 非功能需求

### 2.1 性能
- **NF-01**: 支持长度至少 100,000 点的信号
- **NF-02**: 101 阶 FIR 滤波 10,000 点信号 < 100ms
- **NF-03**: 6 阶 IIR 滤波 10,000 点信号 < 10ms

### 2.2 可靠性
- **NF-04**: 滤波器系数数值稳定性 (SOS 表示)
- **NF-05**: 输入参数验证和错误提示

### 2.3 可用性
- **NF-06**: 统一的 API 接口
- **NF-07**: 完整的文档字符串
- **NF-08**: 直观的参数命名

## 3. 技术约束

- **TC-01**: Python 3.8+
- **TC-02**: 依赖 numpy、scipy、matplotlib
- **TC-03**: 不引入额外重量级依赖

## 4. 验收标准

- 所有单元测试通过
- FIR 滤波器在截止频率处衰减 > 3dB
- IIR 滤波器满足设计规格 (纹波、衰减)
- 零相位滤波无相位失真
- 频率响应图正确显示
