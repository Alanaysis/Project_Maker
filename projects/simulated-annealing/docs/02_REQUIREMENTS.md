# 需求分析：模拟退火算法

## 1. 功能需求

### 1.1 核心算法模块

#### 温度调度
- 支持指数冷却：T(t) = T0 * α^t
- 支持线性冷却：T(t) = T0 - (T0 - Tf) * t/max_iter
- 支持对数冷却：T(t) = T0 / (1 + α * log(1 + t))
- 可配置初始温度、终止温度、冷却速率

#### Metropolis接受准则
- 新解更优时总是接受
- 新解较差时以概率 exp(-ΔE/T) 接受
- 支持自定义接受函数

#### 优化器
- 通用优化框架
- 支持自定义目标函数
- 支持自定义邻域函数
- 记录优化历史（温度、成本、接受率）

### 1.2 邻域操作模块

#### 交换操作（Swap）
- 随机选择两个位置并交换
- 适用于排列问题

#### 逆序操作（Reverse/2-opt）
- 随机选择子序列并反转
- 适用于路径优化问题

#### 插入操作（Insert）
- 随机选择一个元素并插入到另一个位置
- 适用于调度问题

#### Or-opt操作
- 移动一段连续元素到另一个位置
- 适用于TSP等路径优化

#### 混合邻域
- 支持多种邻域操作的随机选择
- 可配置各操作的选择权重

### 1.3 TSP旅行商问题

#### 问题定义
- 城市坐标管理
- 距离矩阵计算
- 路径评估

#### 邻域操作
- 2-opt逆序
- Or-opt移动
- 支持使用通用邻域操作模块

#### 工具函数
- 随机实例生成
- 随机初始解生成

### 1.4 函数优化模块

#### 测试函数
- Sphere函数（简单凸函数）
- Rastrigin函数（多模态）
- Rosenbrock函数（山谷形）
- Ackley函数（指数型）
- Griewank函数（振荡型）
- Levy函数（多模态）

#### 连续邻域生成
- 高斯扰动
- 边界处理（反射/裁剪）
- 自适应步长调整

### 1.5 调度问题模块

#### 作业车间调度（Job Shop）
- 支持n个作业、m台机器
- 支持任意加工顺序
- 目标：最小化makespan

#### 流水车间调度（Flow Shop）
- 支持n个作业、m台机器
- 所有作业加工顺序相同
- 目标：最小化makespan

#### 单机调度（Single Machine）
- 支持n个作业
- 目标：最小化加权总延迟

### 1.6 可视化模块

#### 收敛曲线
- 温度变化曲线
- 当前成本曲线
- 最优成本曲线

#### TSP路径可视化
- 路径绘制
- 城市标注
- 距离显示

#### 冷却策略对比
- 多种冷却策略的温度曲线对比
