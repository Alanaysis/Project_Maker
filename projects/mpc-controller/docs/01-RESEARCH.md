# 01 - 市场调研

## 1. MPC 技术概述

### 1.1 什么是 MPC

模型预测控制（Model Predictive Control, MPC）是一种先进的过程控制方法，它使用系统的数学模型来预测未来行为，并通过优化求解最优控制序列。

### 1.2 核心思想

```
当前状态 → 模型预测 → 优化求解 → 控制执行 → 状态更新
     ↑                                           |
     └─────────────────────────────────────────────┘
                      滚动优化
```

### 1.3 关键特点

| 特点 | 说明 |
|------|------|
| 预测能力 | 使用模型预测未来系统行为 |
| 约束处理 | 显式处理输入、状态、输出约束 |
| 滚动优化 | 每个时刻重新优化 |
| 反馈校正 | 基于实际测量更新预测 |

## 3. 主流 MPC 方法

### 3.1 线性 MPC

- 使用线性模型
- 凸优化问题
- 计算效率高
- 稳定性理论完善

### 3.2 非线性 MPC (NMPC)

- 使用非线性模型
- 非凸优化问题
- 计算复杂度高
- 适用范围更广

### 3.3 随机 MPC

- 考虑不确定性
- 概率约束
- 鲁棒性更强
- 适合不确定系统

### 3.4 显式 MPC

- 离线计算控制律
- 在线查表
- 计算速度快
- 适合嵌入式系统

## 5. 算法对比

### 5.1 MPC vs PID

| 方面 | MPC | PID |
|------|-----|-----|
| 约束处理 | 显式处理 | 难以处理 |
| 多变量 | 原生支持 | 需要解耦 |
| 预测能力 | 有 | 无 |
| 计算复杂度 | 高 | 低 |
| 调参难度 | 中等 | 简单 |

### 5.2 不同 MPC 方法对比

| 方法 | 优点 | 缺点 |
|------|------|------|
| 线性 MPC | 快速、稳定 | 精度有限 |
| NMPC | 精度高 | 计算慢 |
| 鲁棒 MPC | 抗扰动 | 保守 |
| 自适应 MPC | 适应性强 | 复杂 |
