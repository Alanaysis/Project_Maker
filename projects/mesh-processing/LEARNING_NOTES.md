# 学习笔记

## 网格处理基础

### 什么是三角网格

三角网格是 3D 表面的离散表示，由顶点、边和三角形面组成。

**优点**：
- 灵活性高，可以表示任意形状
- 渲染效率高（GPU 原生支持）
- 算法成熟

**缺点**：
- 数据量大
- 拓扑关系复杂

### 核心数据结构

**半边数据结构**：
- 每条边存储两个半边（方向相反）
- 半边指向下一个半边、对边、顶点和面
- 支持高效的拓扑查询

**本项目使用简化版本**：
- 直接存储相邻关系
- 更容易理解和实现
- 牺牲一些查询效率

## 网格简化

### 二次误差度量 (QEM)

**核心思想**：
- 每个顶点关联一个 4x4 矩阵，表示到相关平面的距离平方和
- 边折叠的代价是新顶点位置的误差
- 选择代价最小的边进行折叠

**数学原理**：
- 平面方程：`n · (p - v) = 0`
- 距离平方：`d^2 = (n · p + d)^2`
- 二次形式：`p^T * Q * p`

**算法步骤**：
1. 计算每个顶点的初始 Q 矩阵
2. 对每条边计算最优折叠位置和代价
3. 选择代价最小的边折叠
4. 更新受影响的 Q 矩阵
5. 重复直到达到目标

### 学到的经验

1. **贪心策略有效**：局部最优选择往往产生全局较好结果
2. **误差累积**：每次折叠都会引入误差，需要控制
3. **拓扑维护很重要**：错误的拓扑会导致算法失败

## 细分曲面

### Loop 细分

**基本思想**：
- 每个三角形分为 4 个子三角形
- 调整顶点位置使曲面光滑
- 递归应用可生成任意光滑曲面

**权重设计**：
- 旧顶点权重：`(1 - n*beta, beta, beta, ...)`（n 为度）
- 边上新顶点：`(3/8, 3/8, 1/8, 1/8)`（两个端点和两个对顶点）

**数学性质**：
- C2 连续（除奇异点外）
- 逼近细分（不插值原始顶点）
- 极限曲面存在且光滑

### 学到的经验

1. **权重设计是关键**：不同的权重产生不同的光滑效果
2. **边界处理**：边界边需要特殊处理
3. **递归性质**：细分可以无限递归，但要控制深度

## 网格平滑

### 拉普拉斯平滑

**基本思想**：
- 将顶点向相邻顶点平均位置移动
- 减少局部变化，使曲面更光滑

**公式**：
```
new_pos = old_pos + lambda * (avg_neighbors - old_pos)
```

**问题**：
- 导致网格收缩
- 体积减小
- 细节丢失

### Taubin 平滑

**改进思想**：
- 交替使用正向和负向平滑
- 正向收缩，负向膨胀
- 净效果是平滑但不收缩

**公式**：
```
step1: p' = p + lambda * L(p)    // 收缩
step2: p'' = p' + mu * L(p')     // 膨胀
```

**参数选择**：
- lambda > 0, mu < 0
- |mu| > lambda 确保膨胀效果
- 常用：lambda=0.5, mu=-0.53

### 学到的经验

1. **平滑和保持形状的矛盾**：过度平滑会丢失细节
2. **迭代次数的影响**：多次迭代增强效果但也增加收缩
3. **加权方式的影响**：不同权重产生不同效果
