# 电路仿真 SPICE 需求文档

## 1. 项目概述

### 1.1 项目目标

实现一个简化版的 SPICE 电路仿真器，支持基本的电路分析功能，用于教育和学习目的。

### 1.2 目标用户

- 电子工程学生
- 电路设计初学者
- 硬件工程师
- 对电路仿真感兴趣的学习者

### 1.3 项目范围

**包含**:
- 基本电路元件 (R, C, L, V, I)
- 直流分析
- 交流分析
- 瞬态分析
- 网表解析
- 结果可视化

**不包含**:
- 半导体器件 (二极管、晶体管)
- 复杂的收敛算法
- 温度分析
- 蒙特卡洛分析
- 版图后端提取

## 2. 功能需求

### 2.1 电路元件

#### 2.1.1 电阻 (Resistor)

**功能**: 实现欧姆定律 V = IR

**参数**:
- 名称 (R1, R2, ...)
- 节点 1, 节点 2
- 电阻值 (Ohm)

**约束**:
- 电阻值 > 0

**印记**:
```
G[i,i] += 1/R
G[j,j] += 1/R
G[i,j] -= 1/R
G[j,i] -= 1/R
```

#### 2.1.2 电容 (Capacitor)

**功能**: 实现 I = C dV/dt

**参数**:
- 名称 (C1, C2, ...)
- 节点 1, 节点 2
- 电容值 (F)

**约束**:
- 电容值 > 0

**特性**:
- 直流: 开路
- 交流: 阻抗 Z = 1/(jωC)
- 瞬态: 伴随模型

#### 2.1.3 电感 (Inductor)

**功能**: 实现 V = L dI/dt

**参数**:
- 名称 (L1, L2, ...)
- 节点 1, 节点 2
- 电感值 (H)

**约束**:
- 电感值 > 0

**特性**:
- 直流: 短路
- 交流: 阻抗 Z = jωL
- 瞬态: 伴随模型

#### 2.1.4 电压源 (Voltage Source)

**功能**: 提供固定或时变电压

**参数**:
- 名称 (V1, V2, ...)
- 节点 1, 节点 2
- 直流电压 (V)
- 交流幅度 (V)
- 交流相位 (度)
- 频率 (Hz)

**类型**:
- DC: 固定电压
- AC: 正弦波
- SIN: 调制正弦波

#### 2.1.5 电流源 (Current Source)

**功能**: 提供固定或时变电流

**参数**:
- 名称 (I1, I2, ...)
- 节点 1, 节点 2
- 直流电流 (A)
- 交流幅度 (A)
- 交流相位 (度)
- 频率 (Hz)

### 2.2 网表解析

#### 2.2.1 元件定义

**格式**:
```
Rname node1 node2 value
Cname node1 node2 value
Lname node1 node2 value
Vname node1 node2 DC value
Vname node1 node2 AC mag phase
Iname node1 node2 value
```

**单位后缀**:
- T = 10^12
- G = 10^9
- MEG = 10^6
- K = 10^3
- M = 10^-3
- U = 10^-6
- N = 10^-9
- P = 10^-12
- F = 10^-15

#### 2.2.2 分析命令

**DC 分析**:
```
.dc vsource start stop step
```

**AC 分析**:
```
.ac dec/lin/oct npoints fstart fstop
```

**瞬态分析**:
```
.tran tstep tstop [tstart]
```

#### 2.2.3 注释和控制

- 注释行: 以 `*` 或 `#` 开头
- 标题行: 第一行非注释内容
- 结束标记: `.end`

### 2.3 直流分析

#### 2.3.1 直流工作点

**功能**: 计算电路的直流工作点

**输入**: 电路网表

**输出**:
- 所有节点电压
- 所有支路电流
- 电压源电流

**算法**:
1. 建立 MNA 矩阵
2. 求解线性方程组
3. 提取结果

#### 2.3.2 直流扫描

**功能**: 扫描源值，计算电路响应

**输入**:
- 扫描源名称
- 起始值
- 终止值
- 步长

**输出**:
- 扫描值 vs 响应值

### 2.4 交流分析

#### 2.4.1 单频率分析

**功能**: 计算指定频率的交流响应

**输入**:
- 频率 (Hz)

**输出**:
- 节点电压 (复数)
- 幅度
- 相位

#### 2.4.2 频率扫描

**功能**: 扫描频率，计算频率响应

**输入**:
- 起始频率
- 终止频率
- 扫描类型 (dec/lin/oct)
- 点数

**输出**:
- 频率数组
- 幅度响应 (dB)
- 相位响应 (度)

#### 2.4.3 阻抗计算

**功能**: 计算元件阻抗

**公式**:
- 电阻: Z_R = R
- 电容: Z_C = 1/(jωC)
- 电感: Z_L = jωL

### 2.5 瞬态分析

#### 2.5.1 时间域仿真

**功能**: 计算电路的时间域响应

**输入**:
- 时间步长
- 终止时间
- 初始条件

**输出**:
- 时间数组
- 节点电压 vs 时间
- 支路电流 vs 时间

#### 2.5.2 数值积分

**支持方法**:
- 后向欧拉法 (一阶)
- 梯形法 (二阶)

**伴随模型**:

**电容**:
```
后向欧拉: G_eq = C/h, I_eq = (C/h) * v(t) + i(t)
梯形法: G_eq = 2C/h, I_eq = (2C/h) * v(t) + i(t)
```

**电感**:
```
后向欧拉: G_eq = h/L, I_eq = i(t)
梯形法: G_eq = h/(2L), I_eq = i(t) + (h/(2L)) * v(t)
```

## 4. 接口需求

### 4.1 Python API

```python
# 创建电路
circuit = Circuit("My Circuit")
circuit.add_resistor("R1", 1, 2, 1000)
circuit.add_voltage_source("V1", 0, 1, 10)

# 直流分析
dc = DCAnalysis(circuit)
result = dc.solve()

# 交流分析
ac = ACAnalysis(circuit)
fr = ac.frequency_response(1, 1e6, 100)

# 瞬态分析
tran = TransientAnalysis(circuit)
result = tran.solve(1e-6, 1e-3)
```

### 4.2 网表接口

```python
# 解析网表
parser = NetlistParser()
data = parser.parse(netlist_text)

# 创建电路
circuit = Circuit.from_netlist(data)
```

### 4.3 结果接口

```python
# 直流结果
result.node_voltages  # Dict[int, float]
result.branch_currents  # Dict[str, float]

# 交流结果
result.magnitude(node)  # float
result.phase(node)  # float (度)
result.db(node)  # float (dB)

# 瞬态结果
result.time  # np.ndarray
result.node_voltages  # Dict[int, np.ndarray]
```
