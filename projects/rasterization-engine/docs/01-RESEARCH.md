# 市场调研报告

## 1. 问题定义

### 要解决的问题
将矢量图形（三角形、线条、多边形）转换为像素图像的过程。

### 为什么这个问题重要
光栅化是计算机图形学的核心步骤，几乎所有渲染管线都需要。

## 2. 同类型项目概览

| 项目 | 核心特点 | 技术栈 |
|------|---------|--------|
| Vulkan | 现代 GPU 光栅化 | C++ |
| OpenGL | 标准图形 API | C/C++ |
| SoftBody | 软件渲染器 | C++ |
| Raylib | 简易图形库 | C |

## 3. 技术变体分析

**核心循环**：
```
矢量图形 → 几何变换 → 光栅化 → 像素输出
```

**变体1：三角形光栅化**
- Barycentric 坐标判断
- 扫描线算法

**变体2：Bresenham 画线**
- 整数运算
- 无浮点除法

## 4. 我们的选择

实现软件光栅化引擎：
- 三角形填充（Barycentric）
- 画线（Bresenham）
- 矩形填充
- 圆形绘制
