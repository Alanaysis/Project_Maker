# 凸优化研究文档

## 1. 凸优化基础

### 1.1 凸集

**定义**: 集合 C 是凸集，当且仅当对于任意 x, y ∈ C 和 α ∈ [0, 1]，有：

αx + (1-α)y ∈ C

**常见凸集**:
- 超平面: {x | a^T x = b}
- 半空间: {x | a^T x ≤ b}
- 球: {x | ||x - x_c|| ≤ r}
- 多面体: {x | Ax ≤ b, Cx = d}

### 1.2 凸函数

**定义**: 函数 f: R^n → R 是凸函数，当且仅当：
1. dom f 是凸集
2. 对于任意 x, y ∈ dom f 和 α ∈ [0, 1]：

f(αx + (1-α)y) ≤ αf(x) + (1-α)f(y)

**等价条件**:
- 一阶条件: f(y) ≥ f(x) + ∇f(x)^T (y - x)
- 二阶条件: ∇²f(x) ≥ 0（半正定）

### 1.3 强凸性

**定义**: 函数 f 是 μ-强凸的，当且仅当：

f(y) ≥ f(x) + ∇f(x)^T (y - x) + (μ/2) ||y - x||²

**性质**:
- 强凸函数有唯一最小值
- 梯度下降收敛速度为 O(1/k)
- 条件数 κ = L/μ 决定收敛速度

### 1.4 次梯度

**定义**: 对于非光滑凸函数 f，向量 g 是 f 在 x 处的次梯度，当且仅当：

f(y) ≥ f(x) + g^T (y - x), ∀y

**性质**:
- 次梯度总是存在（对于凸函数）
- 在可微点，次梯度等于梯度
- 次微分 ∂f(x) 是所有次梯度的集合

## 2. 优化算法

### 2.1 梯度下降

**算法**:
```
初始化 x_0
for k = 0, 1, 2, ... do
    x_{k+1} = x_k - α_k ∇f(x_k)
end for
```

**收敛性**:
- 凸函数: f(x_k) - f* ≤ O(1/k)
- 强凸函数: ||x_k - x*||² ≤ O((1 - μ/L)^k)

**变体**:
- 动量梯度下降: x_{k+1} = x_k - α∇f(x_k) + β(x_k - x_{k-1})
- Nesterov 加速: 先计算前瞻点，再计算梯度

### 2.2 牛顿法

**算法**:
```
初始化 x_0
for k = 0, 1, 2, ... do
    计算 ∇f(x_k) 和 ∇²f(x_k)
    求解 ∇²f(x_k) d_k = -∇f(x_k)
    x_{k+1} = x_k + α_k d_k
end for
```

**收敛性**:
- 局部二次收敛: ||x_{k+1} - x*|| ≤ C ||x_k - x*||²
- 需要海森矩阵正定

**改进**:
- 阻尼牛顿法: 使用线搜索选择步长
- 正则化牛顿法: 添加正则化项确保正定

### 2.3 拟牛顿法

**BFGS 更新**:
B_{k+1} = B_k + (y_k y_k^T)/(y_k^T s_k) - (B_k s_k s_k^T B_k)/(s_k^T B_k s_k)

其中 s_k = x_{k+1} - x_k, y_k = ∇f(x_{k+1}) - ∇f(x_k)

**L-BFGS**:
- 只存储最近 m 步的 s 和 y 向量
- 使用两循环递归高效计算 H_k * g

**收敛性**:
- 超线性收敛: ||x_{k+1} - x*|| ≤ C ||x_k - x*||^{1+p}
- 不需要计算海森矩阵

## 3. 约束优化

### 3.1 拉格朗日对偶

**原始问题**:
```
min f(x)
s.t. g_i(x) ≤ 0, i = 1, ..., m
     h_j(x) = 0, j = 1, ..., p
```

**拉格朗日函数**:
L(x, λ, ν) = f(x) + Σ λ_i g_i(x) + Σ ν_j h_j(x)

**对偶函数**:
g(λ, ν) = inf_x L(x, λ, ν)

**对偶问题**:
```
max g(λ, ν)
s.t. λ ≥ 0
```

**弱对偶**: g(λ, ν) ≤ f(x*)
**强对偶**: 在 Slater 条件下，g(λ*, ν*) = f(x*)

### 3.2 KKT 条件

对于凸问题，KKT 条件是最优解的充要条件：

1. **平稳性**: ∇f(x*) + Σ λ_i ∇g_i(x*) + Σ ν_j ∇h_j(x*) = 0
2. **原始可行性**: g_i(x*) ≤ 0, h_j(x*) = 0
3. **对偶可行性**: λ_i ≥ 0
4. **互补松弛性**: λ_i g_i(x*) = 0

### 3.3 内点法

**障碍函数法**:
min f(x) - (1/t) Σ log(-g_i(x))

**原始-对偶内点法**:
同时更新原始变量和对偶变量，更稳定。

**收敛性**:
- 多项式时间复杂度
- 每次迭代需要求解线性系统

## 4. 实际应用

### 4.1 最小二乘

**问题**: min ||Ax - b||²

**解法**:
- 正规方程: x = (A^T A)^{-1} A^T b
- QR 分解: A = QR, x = R^{-1} Q^T b
- SVD 分解: A = UΣV^T, x = V Σ^{-1} U^T b

**正则化**:
- 岭回归: min ||Ax - b||² + λ||x||²
- Lasso: min ||Ax - b||² + λ||x||₁

### 4.2 SVM

**问题**:
```
min 0.5 ||w||² + C Σ ξ_i
s.t. y_i (w^T x_i + b) ≥ 1 - ξ_i
     ξ_i ≥ 0
```

**对偶问题**:
```
max Σ α_i - 0.5 Σ Σ α_i α_j y_i y_j x_i^T x_j
s.t. 0 ≤ α_i ≤ C
     Σ α_i y_i = 0
```

**求解**: SMO (Sequential Minimal Optimization)

### 4.3 投资组合优化

**Markowitz 模型**:
```
min w^T Σ w
s.t. w^T μ = target_return
     w^T 1 = 1
     w ≥ 0
```

**有效前沿**:
- 最小方差组合
- 最大夏普比率组合
- 风险平价组合

## 5. 理论基础

### 5.1 凸优化的性质

1. **局部最优即全局最优**: 凸函数的任何局部最小值都是全局最小值
2. **最优性条件**: KKT 条件是凸问题最优解的充要条件
3. **对偶性**: 强对偶性在 Slater 条件下成立

### 5.2 收敛性分析

**梯度下降**:
- 凸函数: O(1/k)
- 强凸函数: O((1 - μ/L)^k)
- 条件数 κ = L/μ 影响收敛速度

**牛顿法**:
- 局部二次收敛
- 需要海森矩阵正定
- 计算复杂度 O(n³)

**拟牛顿法**:
- 超线性收敛
- 不需要计算海森矩阵
- 存储复杂度 O(n²) (BFGS) 或 O(mn) (L-BFGS)

### 5.3 复杂度分析

**梯度下降**: O(n) 每次迭代
**牛顿法**: O(n³) 每次迭代（线性系统求解）
**BFGS**: O(n²) 每次迭代
**L-BFGS**: O(mn) 每次迭代
