# 需求文档：基本电路模拟

## 1. 项目概述

### 1.1 项目目标

实现一个基本的电路模拟器，支持：
- 基本电路元件（R、C、L、V、I）的建模
- 直流电路分析（节点分析法）
- 交流电路分析（阻抗、频率响应）
- 实际应用电路（分压器、滤波器、放大器）

### 1.2 用户场景

- **学生**：学习电路分析原理
- **工程师**：快速验证电路设计
- **爱好者**：探索电路特性

## 2. 功能需求

### 2.1 电路元件模块

#### FR-2.1.1 电阻

- 支持创建电阻元件
- 计算阻抗（与频率无关）
- 计算电导
- 验证电阻值（必须为正数）

#### FR-2.1.2 电容

- 支持创建电容元件
- 计算阻抗（与频率相关）
- 计算容抗
- 直流时表现为开路

#### FR-2.1.3 电感

- 支持创建电感元件
- 计算阻抗（与频率相关）
- 计算感抗
- 直流时表现为短路

#### FR-2.1.4 电压源

- 支持直流和交流电压源
- 支持设置幅值、频率、相位
- 计算相量表示
- 计算瞬时值

#### FR-2.1.5 电流源

- 支持直流和交流电流源
- 支持设置幅值、频率、相位
- 计算相量表示
- 计算瞬时值

#### FR-2.1.6 欧姆定律

- 根据任意两个已知量计算第三个量
- 支持 V、I、R 的相互转换
- 功率计算（P=VI, P=I²R, P=V²/R）

### 2.2 电路网络模块

#### FR-2.2.1 节点管理

- 创建节点
- 设置接地点（参考节点）
- 节点命名

#### FR-2.2.2 元件添加

- 添加电阻、电容、电感
- 添加电压源、电流源
- 验证节点连接

#### FR-2.2.3 电路查询

- 获取两个节点之间的元件
- 获取连接到指定节点的元件
- 按类型筛选元件

#### FR-2.2.4 电路信息

- 输出电路摘要
- 统计节点和元件数量

### 2.3 直流分析模块

#### FR-2.3.1 节点分析法

- 构建导纳矩阵
- 处理电压源约束（MNA方法）
- 求解节点电压

#### FR-2.3.2 支路计算

- 计算支路电流
- 计算支路电压
- 计算功率消耗

#### FR-2.3.3 定律验证

- 验证KCL（基尔霍夫电流定律）
- 验证KVL（基尔霍夫电压定律）
- 报告误差

#### FR-2.3.4 辅助函数

- 分压器计算
- 分流器计算
- 串联电阻计算
- 并联电阻计算

### 2.4 交流分析模块

#### FR-2.4.1 阻抗计算

- 计算各元件阻抗
- 计算总阻抗
- 支持复数运算

#### FR-2.4.2 频率响应

- 计算幅频响应
- 计算相频响应
- 支持对数频率扫描

#### FR-2.4.3 相量分析

- 相量表示（幅值+相位）
- 相量运算
- 与复数转换

#### FR-2.4.4 辅助函数

- 串联阻抗计算
- 并联阻抗计算
- 谐振频率计算
- 品质因数计算

### 2.5 实际应用模块

#### FR-2.5.1 分压器

- 基本分压器计算
- 考虑负载效应
- 计算输出阻抗
- 计算功率消耗

#### FR-2.5.2 RC滤波器

- 低通滤波器
- 高通滤波器
- 截止频率计算
- 频率响应计算

#### FR-2.5.3 RLC滤波器

- 带通滤波器
- 带阻滤波器
- 谐振频率计算
- 品质因数计算

#### FR-2.5.4 放大器

- 反相放大器
- 同相放大器
- 差分放大器
- 增益计算
- 带宽计算

#### FR-2.5.5 积分器/微分器

- 积分器电路
- 微分器电路
- 频率响应

## 3. 非功能需求

### 3.1 性能需求

- 单次DC分析：< 100ms（100个节点以内）
- 单次AC分析：< 50ms（单频率点）
- 频率响应计算：< 1s（1000个频率点）

### 3.2 可用性需求

- 清晰的API设计
- 完整的类型提示
- 详细的文档和示例
- 友好的错误信息

### 3.3 可靠性需求

- 输入验证
- 异常处理
- 数值稳定性
- 单元测试覆盖 > 90%

### 3.4 可维护性需求

- 模块化设计
- 单一职责原则
- 代码注释
- 文档完整

## 4. 数据需求

### 4.1 输入数据

- 元件参数（电阻值、电容值等）
- 频率范围
- 分析配置

### 4.2 输出数据

- 节点电压
- 支路电流
- 功率消耗
- 频率响应数据

### 4.3 数据格式

- 使用Python内置类型
- 复数使用complex类型
- 数组使用NumPy数组

## 5. 接口需求

### 5.1 用户接口

- Python API
- 命令行示例
- 可视化输出

### 5.2 软件接口

- NumPy：数值计算
- Matplotlib：可视化
- SciPy：科学计算

## 6. 约束条件

### 6.1 技术约束

- 仅支持线性电路
- 不支持非线性元件
- 不支持瞬态分析
- 不支持分布式参数

### 6.2 环境约束

- Python 3.8+
- 操作系统：Windows/Linux/MacOS
