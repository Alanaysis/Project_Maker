# 调研文档：基本电路模拟

## 1. 电路分析基础

### 1.1 基本电路元件

| 元件 | 符号 | 单位 | 阻抗 | 特性 |
|------|------|------|------|------|
| 电阻 | R | 欧姆 (Ω) | Z = R | 消耗能量 |
| 电容 | C | 法拉 (F) | Z = 1/(jωC) | 储存电场能 |
| 电感 | L | 亨利 (H) | Z = jωL | 储存磁场能 |
| 电压源 | V | 伏特 (V) | - | 提供恒定电压 |
| 电流源 | I | 安培 (A) | - | 提供恒定电流 |

### 1.2 欧姆定律

欧姆定律描述电压、电流和电阻之间的关系：

```
V = IR
```

功率计算：
```
P = VI = I²R = V²/R
```

### 1.3 基尔霍夫定律

**基尔霍夫电流定律 (KCL)**：
- 流入节点的电流之和等于流出节点的电流之和
- ΣI_in = ΣI_out

**基尔霍夫电压定律 (KVL)**：
- 闭合回路中电压降之和等于电压升之和
- ΣV = 0

## 2. 电路分析方法

### 2.1 节点分析法 (Nodal Analysis)

节点分析法基于KCL，以节点电压为未知量：

1. 选择参考节点（接地点）
2. 对每个非参考节点列KCL方程
3. 求解线性方程组

**改进节点分析法 (MNA)**：
- 处理电压源的特殊约束
- 扩展导纳矩阵

### 2.2 网孔分析法 (Mesh Analysis)

网孔分析法基于KVL，以网孔电流为未知量：

1. 识别网孔（独立回路）
2. 对每个网孔列KVL方程
3. 求解线性方程组

### 2.3 叠加定理

线性电路中，多个电源的响应等于各电源单独响应的叠加。

### 2.4 戴维南/诺顿等效

- **戴维南等效**：V_th + R_th 串联
- **诺顿等效**：I_n + R_n 并联

## 3. 交流电路分析

### 3.1 阻抗

阻抗是电阻的推广，包含实部（电阻）和虚部（电抗）：

```
Z = R + jX
```

其中：
- R = 电阻
- X = 电抗

### 3.2 相量分析

相量表示正弦信号的幅值和相位：

```
V(t) = V_m cos(ωt + φ) ↔ V = V_m ∠φ
```

### 3.3 频率响应

频率响应描述系统对不同频率信号的响应：

- **幅频响应**：|H(f)| vs f
- **相频响应**：∠H(f) vs f

### 3.4 谐振

RLC电路在特定频率下发生谐振：

```
f_r = 1 / (2π√(LC))
```

品质因数：
```
Q = ωL / R = 1 / (ωCR)
```

## 4. 滤波器

### 4.1 滤波器类型

| 类型 | 功能 | 应用 |
|------|------|------|
| 低通 (LPF) | 通过低频，阻隔高频 | 去噪、平滑 |
| 高通 (HPF) | 通过高频，阻隔低频 | 去除直流、提取边缘 |
| 带通 (BPF) | 通过特定频段 | 通信、音频 |
| 带阻 (BSF) | 阻隔特定频段 | 陷波、干扰消除 |

### 4.2 RC滤波器

**一阶RC低通**：
```
H(f) = 1 / (1 + j2πfRC)
f_c = 1 / (2πRC)
```

**一阶RC高通**：
```
H(f) = j2πfRC / (1 + j2πfRC)
f_c = 1 / (2πRC)
```

### 4.3 RLC滤波器

**带通滤波器**：
```
H(f) = (jωL) / (R + jωL + 1/(jωC))
```

**带阻滤波器**：
```
H(f) = R / (R + jωL + 1/(jωC))
```

## 5. 放大器

### 5.1 运算放大器

理想运放特性：
- 开环增益无穷大
- 输入阻抗无穷大
- 输出阻抗为零
- 虚短、虚断

### 5.2 反相放大器

```
V_out = -(R_f / R_in) × V_in
增益 = -R_f / R_in
```

### 5.3 同相放大器

```
V_out = (1 + R_f / R_in) × V_in
增益 = 1 + R_f / R_in
```

### 5.4 差分放大器

```
V_out = (R_f / R_in) × (V+ - V-)
```

## 7. 技术选型

### 7.1 Python库选择

| 库 | 用途 | 选择理由 |
|----|------|----------|
| NumPy | 数值计算 | 矩阵运算、线性代数 |
| Matplotlib | 可视化 | 绘制频率响应曲线 |
| SciPy | 科学计算 | 信号处理、优化算法 |

### 7.2 数据结构设计

- **元件类**：继承关系，统一接口
- **电路类**：图结构，节点+支路
- **分析器类**：策略模式，可扩展

### 7.3 算法选择

- **节点分析法**：通用性强，适合计算机实现
- **MNA方法**：处理电压源的标准方法
- **频率扫描**：对数间隔，覆盖宽频率范围

## 8. 风险与限制

### 8.1 当前限制

- 仅支持线性电路
- 不支持非线性元件（二极管、晶体管）
- 不支持瞬态分析
- 不支持分布式参数电路

### 8.2 未来扩展

- 非线性元件支持
- 瞬态分析（时域）
- 噪声分析
- 蒙特卡洛分析
- PCB布线分析
