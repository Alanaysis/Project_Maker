# 因子挖掘框架 - 需求分析

## 1. 功能需求

### 1.1 因子计算模块

#### 技术因子
| 因子 | 输入 | 输出 | 优先级 |
|------|------|------|--------|
| 动量 | 收盘价、窗口 | 因子值序列 | P0 |
| 波动率 | 收盘价、窗口 | 因子值序列 | P0 |
| 换手率 | 成交量、流通股本 | 因子值序列 | P0 |
| RSI | 收盘价、窗口 | 因子值序列 | P1 |
| MACD | 收盘价、参数 | DataFrame | P1 |
| 布林带 | 收盘价、窗口 | 因子值序列 | P2 |

#### 基本面因子
| 因子 | 输入 | 输出 | 优先级 |
|------|------|------|--------|
| EP | PE_TTM | 因子值序列 | P0 |
| BP | PB | 因子值序列 | P0 |
| ROE | 财务数据 | 因子值序列 | P0 |
| 营收增长 | 财务数据 | 因子值序列 | P1 |
| 综合估值 | 多个估值指标 | 因子值序列 | P1 |

#### 另类因子
| 因子 | 输入 | 输出 | 优先级 |
|------|------|------|--------|
| 资金流向 | 成交量、价格 | 因子值序列 | P1 |
| 短期反转 | 收盘价 | 因子值序列 | P1 |
| 偏度 | 收盘价 | 因子值序列 | P2 |

### 1.2 因子评估模块

| 功能 | 输入 | 输出 | 优先级 |
|------|------|------|--------|
| Rank IC 计算 | 因子序列、收益序列 | IC 值 | P0 |
| IC 时间序列 | 因子面板、收益面板 | IC 序列 | P0 |
| IC 统计摘要 | IC 序列 | 统计指标 | P0 |
| IR 计算 | IC 序列 | IR 值 | P0 |
| 分组回测 | 因子面板、收益面板 | 回测结果 | P0 |
| IC 衰减分析 | 因子面板、收益面板 | 衰减曲线 | P1 |
| 单调性检验 | 分组收益 | 单调性指标 | P1 |

### 1.3 因子优化模块

| 功能 | 输入 | 输出 | 优先级 |
|------|------|------|--------|
| Z-score 标准化 | 因子序列 | 标准化序列 | P0 |
| 排名标准化 | 因子序列 | 排名序列 | P0 |
| 行业中性化 | 因子、行业 | 中性化序列 | P0 |
| 市值中性化 | 因子、市值 | 中性化序列 | P0 |
| MAD 去极值 | 因子序列 | 去极值序列 | P0 |
| 前值填充 | 因子序列 | 填充序列 | P1 |

### 1.4 因子组合模块

| 功能 | 输入 | 输出 | 优先级 |
|------|------|------|--------|
| 等权组合 | 多个因子 | 组合因子 | P0 |
| IC 加权 | 多个因子、IC | 组合因子 | P1 |
| 最大化 IR | 多个因子、收益 | 组合因子 | P1 |
| ML 组合 | 多个因子、收益 | 组合因子 | P2 |

### 1.5 回测系统

| 功能 | 输入 | 输出 | 优先级 |
|------|------|------|--------|
| 数据回放 | 历史数据 | 迭代器 | P0 |
| 因子回测 | 因子面板 | 回测报告 | P0 |
| 组合回测 | 因子面板 | 回测结果 | P0 |
| 性能分析 | 收益序列 | 性能指标 | P0 |

### 1.6 实际应用

| 功能 | 输入 | 输出 | 优先级 |
|------|------|------|--------|
| 因子注册 | 因子元数据 | 注册信息 | P1 |
