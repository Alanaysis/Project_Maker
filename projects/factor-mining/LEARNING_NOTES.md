# 学习笔记 - 因子挖掘框架

## 核心概念

### 什么是量化因子?
量化因子是可以解释股票收益差异的系统性特征。例如:
- **动量因子**: 过去涨得多的股票, 未来可能继续涨
- **波动率因子**: 低波动的股票可能有更高的风险调整收益
- **流动性因子**: 流动性差的股票可能有流动性溢价

### 因子 vs 指标
- **指标 (Indicator)**: RSI、MACD 等技术指标, 是原始信号
- **因子 (Factor)**: 经过截面标准化、可直接用于选股的信号

## 学到的关键知识

### 1. Rank IC 优于 Pearson IC
```
Rank IC (Spearman) 更稳健:
- 不受异常值影响
- 不要求线性关系
- 更适合非正态分布的因子值
```

### 2. ICIR 比 IC 均值更重要
```
IC均值 = 0.05, IC标准差 = 0.10 → ICIR = 0.5 (一般)
IC均值 = 0.03, IC标准差 = 0.03 → ICIR = 1.0 (优秀)

ICIR 高意味着因子更稳定, 可预测性更强
```

### 3. 分组收益的单调性
理想的因子应该让分组收益单调递增:
```
G1 (低): -2%
G2:      -1%
G3:       0%
G4:      +1%
G5 (高): +2%
```
如果不单调, 因子可能只在极端值有效。

### 4. 多空 vs 纯多头
- **多空策略**: 做多最高组 + 做空最低组, 收益更高但需要融券
- **纯多头策略**: 仅做多最高组, 更实际但收益较低

### 5. 交易成本的影响
```
因子预测能力: IC = 0.05
换手率: 50%/期
单边成本: 0.1%
每期成本: 0.5% * 2 * 0.1% = 0.1%

如果每期多空收益 < 0.1%, 扣除成本后为负!
```

### 6. 因子衰减
```
IC 随持有期变化:
lag=1:  IC = 0.05
lag=5:  IC = 0.03
lag=10: IC = 0.01
lag=20: IC = 0.00

因子预测力随时间衰减, 需要定期换仓
```

## 实践心得

### 数据质量最重要
- 模拟数据容易, 真实数据有各种问题 (停牌、复权、缺值)
- 数据预处理可能占 70% 的工作量

### 过拟合风险
- 在训练集上 IC 很高, 测试集上可能为负
- 需要样本外验证
- 因子越复杂, 过拟合风险越大

### 因子组合的威力
单个因子 IC = 0.05, 但如果因子之间相关性低:
```
组合后 IC = 0.05 * sqrt(n) / sqrt(n) ≈ 0.05 * sqrt(相关性修正)
```
多因子组合可以显著提升稳定性。
