## 技术难点

### 1. 并发控制

**问题**：多个协程同时访问共享状态导致数据竞争

**解决方案**：
- 使用 sync.RWMutex 保护共享状态
- 提供线程安全的访问方法
- 使用 channel 进行协程间通信

### 2. 选举超时

**问题**：固定超时时间导致选票分割

**解决方案**：
- 随机化选举超时时间
- 设置合理的超时范围
- 使用指数退避策略

### 3. 日志一致性

**问题**：日志不一致导致复制失败

**解决方案**：
- 实现日志回退机制
- 使用 conflictIndex 和 conflictTerm
- 逐步回退直到找到匹配点

### 4. 网络分区

**问题**：网络分区导致节点无法通信

**解决方案**：
- 实现超时重试机制
- 使用心跳检测节点状态
- 支持节点重新加入
