# 梯度下降家族 - 市场调研

## 1. 梯度下降算法概述

梯度下降是机器学习中最基础且最重要的优化算法，用于通过迭代更新参数来最小化损失函数。

### 核心原理

梯度下降基于以下数学原理：

```
θ_{t+1} = θ_t - η * ∇J(θ_t)
```

其中：
- `θ` 是模型参数
- `η` 是学习率（步长）
- `∇J(θ)` 是损失函数的梯度

## 2. 主要梯度下降变体

### 2.1 批量梯度下降 (Batch Gradient Descent)

**特点**：
- 每次迭代使用全部训练样本计算梯度
- 收敛稳定，但计算开销大
- 适合小规模数据集

**数学表达**：
```
∇J(θ) = (1/N) * Σ_{i=1}^{N} ∇J_i(θ)
```

### 2.2 随机梯度下降 (Stochastic Gradient Descent, SGD)

**特点**：
- 每次迭代只使用一个样本计算梯度
- 计算速度快，但收敛不稳定
- 有逃离局部最优的能力

**数学表达**：
```
∇J(θ) ≈ ∇J_i(θ)  (i 为随机选择的样本)
```

### 2.3 小批量梯度下降 (Mini-batch Gradient Descent)

**特点**：
- 折中方案，每次使用一个小批量样本
- 结合了批量和随机梯度下降的优点
- 是深度学习中最常用的方法

**数学表达**：
```
∇J(θ) = (1/B) * Σ_{i=1}^{B} ∇J_i(θ)  (B 为批量大小)
```

## 3. 高级优化算法

### 3.1 动量法 (Momentum)

**提出背景**：解决 SGD 在峡谷型损失函数中震荡的问题

**数学表达**：
```
v_t = γ * v_{t-1} + η * ∇J(θ_t)
θ_{t+1} = θ_t - v_t
```

**优势**：
- 加速收敛
- 减少震荡
- 常用动量系数 γ = 0.9

### 3.2 Nesterov 加速梯度 (NAG)

**改进点**：在计算梯度前先"向前看一步"

**数学表达**：
```
v_t = γ * v_{t-1} + η * ∇J(θ_t - γ * v_{t-1})
θ_{t+1} = θ_t - v_t
```

**优势**：
- 更准确的梯度估计
- 更快的收敛速度

### 3.3 AdaGrad (Adaptive Gradient)

**提出背景**：为不同参数自适应调整学习率

**数学表达**：
```
G_t = G_{t-1} + (∇J(θ_t))^2
θ_{t+1} = θ_t - (η / √(G_t + ε)) * ∇J(θ_t)
```

**特点**：
- 对稀疏特征友好
- 学习率单调递减，可能过早停止学习

### 3.4 RMSProp

**改进点**：使用移动平均代替累积平方和

**数学表达**：
```
G_t = β * G_{t-1} + (1-β) * (∇J(θ_t))^2
θ_{t+1} = θ_t - (η / √(G_t + ε)) * ∇J(θ_t)
```

**优势**：
- 解决 AdaGrad 学习率过早衰减的问题
- 常用 β = 0.999

### 3.5 Adam (Adaptive Moment Estimation)

**核心思想**：结合动量法和 RMSProp 的优点

**数学表达**：
```
m_t = β1 * m_{t-1} + (1-β1) * ∇J(θ_t)      # 一阶矩估计
v_t = β2 * v_{t-1} + (1-β2) * (∇J(θ_t))^2    # 二阶矩估计
m̂_t = m_t / (1 - β1^t)                        # 偏差修正
v̂_t = v_t / (1 - β2^t)                        # 偏差修正
θ_{t+1} = θ_t - (η / (√v̂_t + ε)) * m̂_t
```

**默认超参数**：
- β1 = 0.9
- β2 = 0.999
- ε = 1e-8

**优势**：
- 自适应学习率
- 偏差修正机制
- 适用于大多数问题

### 3.6 AdamW (Adam with Weight Decay)

**改进点**：解耦权重衰减与梯度更新

**数学表达**：
```
m_t = β1 * m_{t-1} + (1-β1) * ∇J(θ_t)
v_t = β2 * v_{t-1} + (1-β2) * (∇J(θ_t))^2
m̂_t = m_t / (1 - β1^t)
v̂_t = v_t / (1 - β2^t)
θ_{t+1} = θ_t - η * (m̂_t / (√v̂_t + ε) + λ * θ_t)
```

**优势**：
- 更好的泛化性能
- 现代深度学习的默认选择

### 3.7 Nadam (Nesterov-accelerated Adaptive Moment Estimation)

**核心思想**：结合 Nesterov 动量和 Adam 的优点

**数学表达**：
```
m_t = β1 * m_{t-1} + (1-β1) * ∇J(θ_t)
v_t = β2 * v_{t-1} + (1-β2) * (∇J(θ_t))^2
m̂_t = m_t / (1 - β1^t)
v̂_t = v_t / (1 - β2^t)
θ_{t+1} = θ_t - η * ((1-β1) * ∇J(θ_t) + β1 * m̂_t) / (√v̂_t + ε)
```

**优势**：
- 结合 Nesterov 动量的前瞻性和 Adam 的自适应学习率
- 通常比 Adam 收敛更快
- 对超参数不敏感

## 4. 学习率调度策略

### 4.1 学习率衰减 (Learning Rate Decay)

**类型**：
- **指数衰减**: `lr = lr_0 * decay_rate^(epoch/decay_steps)`
- **余弦退火**: `lr = lr_0 * 0.5 * (1 + cos(π * epoch/total_epochs))`
- **阶梯衰减**: 每隔固定 epoch 降低学习率

### 4.2 Warmup

**原理**：训练初期使用较小学习率，逐渐增加到目标学习率

**优势**：
- 稳定训练初期
- 防止梯度爆炸

### 4.3 余弦退火 (Cosine Annealing)

**数学表达**：
```
lr = lr_min + 0.5 * (lr_max - lr_min) * (1 + cos(π * t / T))
```

**特点**：
- 平滑的学习率变化
- 有助于跳出局部最优

## 6. 实际应用建议

### 6.1 选择算法的考虑因素

1. **数据规模**：
   - 小数据集：SGD、Momentum
   - 大数据集：Adam、AdamW

2. **问题类型**：
   - 稀疏特征：AdaGrad、Adam
   - 稠密特征：SGD + Momentum

3. **收敛速度要求**：
   - 快速收敛：Adam
   - 精细调优：SGD + 学习率调度

4. **泛化性能**：
   - 追求泛化：SGD + Momentum + 学习率调度
   - 快速原型：Adam

### 6.2 超参数调优建议

**学习率**：
- Adam: 1e-3 到 1e-4
- SGD: 1e-1 到 1e-3

**批量大小**：
- 常用：32, 64, 128, 256
- 更大批量 → 更稳定但可能泛化差

**动量系数**：
- Momentum: 0.9
- Adam β1: 0.9
- Adam β2: 0.999

## 7. 最新研究进展

### 7.1 LAMB (Layer-wise Adaptive Moments optimizer for Batch training)

- 专为大批量训练设计
- 支持更大的批量大小而不损失性能

### 7.2 LARS (Layer-wise Adaptive Rate Scaling)

- 逐层自适应学习率
- 解决深层网络训练不稳定问题

### 7.3 Lookahead

- "向前看"优化算法
- 维护两组权重，提高稳定性和收敛性

### 7.4 RAdam (Rectified Adam)

- 修复 Adam 训练初期的方差问题
- 无需 warmup

## 8. 总结

梯度下降家族的发展历程体现了优化算法从简单到复杂、从固定到自适应的演进：

1. **第一代**：SGD 及其变体（简单、可解释）
2. **第二代**：自适应学习率算法（AdaGrad, RMSProp）
3. **第三代**：结合动量和自适应（Adam, AdamW）
4. **第四代**：针对特定场景优化（LAMB, LARS）

理解这些算法的原理和适用场景，对于机器学习实践至关重要。本项目将实现这些核心算法，并通过可视化帮助理解它们的行为差异。
