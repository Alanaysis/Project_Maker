# 01 - 物理引擎研究

## 物理引擎概述

物理引擎是用于模拟物理系统的软件组件，广泛应用于游戏开发、动画制作和科学仿真。

### 核心概念

#### 1. 刚体动力学

刚体是理想化的物体，其形状和大小在运动过程中保持不变。

**关键属性：**
- **质量 (Mass)**：物体的惯性量度
- **位置 (Position)**：物体在空间中的坐标
- **速度 (Velocity)**：位置的变化率
- **加速度 (Acceleration)**：速度的变化率
- **力 (Force)**：改变物体运动状态的原因

**牛顿运动定律：**
1. **第一定律**：物体保持静止或匀速直线运动，除非受外力作用
2. **第二定律**：F = ma（力等于质量乘以加速度）
3. **第三定律**：作用力与反作用力大小相等、方向相反

#### 2. 碰撞检测

碰撞检测是判断两个物体是否接触的过程。

**常见算法：**

1. **AABB 碰撞检测**
   - 轴对齐包围盒
   - 快速但不够精确
   - 适用于粗略检测

2. **圆形碰撞检测**
   - 计算圆心距离
   - 简单高效
   - 适用于圆形物体

3. **分离轴定理 (SAT)**
   - 适用于凸多边形
   - 投影到所有可能的分离轴
   - 如果存在分离轴，则不相交

4. **GJK 算法**
   - 更复杂的凸形状碰撞检测
   - 迭代算法
   - 高效且通用

#### 3. 碰撞响应

碰撞响应是处理物体碰撞后的运动变化。

**弹性碰撞：**
- 动量守恒：m₁v₁ + m₂v₂ = m₁v₁' + m₂v₂'
- 能量守恒：½m₁v₁² + ½m₂v₂² = ½m₁v₁'² + ½m₂v₂'²

**非弹性碰撞：**
- 动量守恒，但能量不守恒
- 恢复系数 e：e = (v₂' - v₁') / (v₁ - v₂)
  - e = 1：完全弹性碰撞
  - e = 0：完全非弹性碰撞

**摩擦力：**
- 静摩擦力：阻止物体开始运动
- 动摩擦力：阻止物体相对运动
- 库仑摩擦模型：f = μN

#### 4. 约束系统

约束是限制物体运动的规则。

**常见约束类型：**

1. **距离约束**
   - 保持两点之间的距离不变
   - 用于绳索、杆件

2. **弹簧约束**
   - 弹性连接两点
   - 胡克定律：F = -kx

3. **钉子约束**
   - 固定物体在世界坐标系中的位置
   - 用于锚点

4. **铰链约束**
   - 允许旋转但限制位置
   - 用于门、摆锤

5. **滑轮约束**
   - 改变力的方向
   - 用于简单机械

#### 5. 积分方法

积分是将加速度转换为速度，再将速度转换为位置的过程。

**常见积分方法：**

1. **欧拉方法**
   - 最简单的积分方法
   - 显式欧拉：v += a*dt; x += v*dt
   - 半隐式欧拉：v += a*dt; x += v*dt（先更新速度）
   - 稳定性较差

2. **Verlet 积分**
   - 基于位置的积分
   - x(t+dt) = 2x(t) - x(t-dt) + a*dt²
   - 更稳定，适合约束求解
   - 无显式速度存储

3. **龙格-库塔方法 (RK4)**
   - 四阶精度
   - 计算四个中间状态
   - 更精确但计算量大

4. **辛积分器**
   - 保持能量守恒
   - 适合长时间仿真
   - 如蛙跳积分

### 物理引擎架构

#### 典型物理引擎流程

```
1. 应用外力（重力、用户输入等）
2. 积分（更新速度和位置）
3. 碰撞检测
4. 碰撞响应
5. 约束求解
6. 更新物理状态
```

#### 常见优化技术

1. **空间分区**
   - 四叉树（2D）
   - 八叉树（3D）
   - BVH（层次包围盒）

2. **宽相检测**
   - 快速排除不可能碰撞的物体对
   - 使用 AABB 进行初步筛选

3. **窄相检测**
   - 精确碰撞检测
   - 仅对宽相筛选出的物体对进行

4. **休眠机制**
   - 静止物体进入休眠状态
   - 减少不必要的计算

### 本项目实现目标

本项目旨在实现一个基础的 2D 物理引擎，涵盖以下核心功能：

1. **向量数学**
   - 2D 向量基本运算
   - 点积、叉积
   - 归一化、旋转

2. **刚体系统**
   - 质量、速度、加速度
   - 力和冲量
   - 静态和动态物体

3. **碰撞检测**
   - AABB 碰撞
   - 圆形碰撞
   - SAT 多边形碰撞

4. **碰撞响应**
   - 弹性碰撞
   - 非弹性碰撞
   - 摩擦力

5. **约束系统**
   - 距离约束
   - 弹簧约束

6. **积分方法**
   - 欧拉方法
   - Verlet 积分
   - RK4 方法

7. **物理世界**
   - 重力模拟
   - 物体管理
   - 碰撞回调

通过实现这些功能，深入理解物理引擎的工作原理，为更复杂的物理模拟打下基础。
