# 实现细节

## 1. 基础图元

### 立方体
8 顶点，12 三角面，每面 2 个三角形。

### 球体
经纬线细分，顶点数 = (segments+1) * (rings+1)。

### 圆柱
顶面中心 + 顶面环 + 底面环 + 底面中心。

## 2. 变换矩阵

### 平移
```
| 1  0  0  tx |
| 0  1  0  ty |
| 0  0  1  tz |
| 0  0  0  1  |
```

### 旋转 X
```
| 1   0    0   0 |
| 0  cos  -sin  0 |
| 0  sin   cos  0 |
| 0   0    0   1 |
```

## 3. 法线计算

三角面法线 = (v2-v1) × (v3-v1)，归一化。
