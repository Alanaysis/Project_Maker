# 模拟滤波器学习笔记

## 1. 基础概念

### 1.1 什么是滤波器

滤波器是一种能够选择性地通过或衰减特定频率信号的电路或系统。模拟滤波器使用电阻(R)、电感(L)、电容(C)等无源元件实现。

### 1.2 频率域 vs 时域

- **时域**: 信号随时间变化的表示
- **频域**: 信号在不同频率分量上的表示
- **傅里叶变换**: 连接时域和频域的桥梁

### 1.3 传递函数

传递函数 H(s) 描述了系统输出与输入的比值：

```
H(s) = Y(s) / X(s)
```

其中 s = jω = j2πf 是复频率变量。

## 2. 基本电路元件

### 2.1 电阻 (R)

- 阻抗: Z_R = R (纯实数)
- 电压-电流关系: V = IR
- 不产生相移

### 2.2 电容 (C)

- 阻抗: Z_C = 1/(jωC) = 1/(sC)
- 电压-电流关系: I = C(dV/dt)
- 电流超前电压 90°

### 2.3 电感 (L)

- 阻抗: Z_L = jωL = sL
- 电压-电流关系: V = L(dI/dt)
- 电压超前电流 90°

## 3. RC 低通滤波器

### 3.1 电路结构

```
输入 ─── R ───┬─── 输出
              │
              C
              │
             GND
```

### 3.2 传递函数推导

使用分压公式：
```
H(s) = Z_C / (R + Z_C) = (1/sC) / (R + 1/sC)
     = 1 / (1 + sRC)
```

### 3.3 频率响应

幅度响应:
```
|H(jω)| = 1 / √(1 + (ωRC)²)
```

相位响应:
```
φ(ω) = -arctan(ωRC)
```

### 3.4 截止频率

当 |H| = 1/√2 ≈ 0.707 (-3dB) 时：
```
ω_c = 1/RC
f_c = 1/(2πRC)
```

### 3.5 关键特性

- 直流增益: 1 (0dB)
- 截止频率处增益: -3dB
- 高频衰减: -20dB/decade
- 截止频率处相位: -45°
- 相位范围: 0° ~ -90°

## 4. RL 低通滤波器

### 4.1 电路结构

```
输入 ─── R ───┬─── 输出
              │
              L
              │
             GND
```

### 4.2 传递函数

```
H(s) = (R/L) / (s + R/L)
```

### 4.3 截止频率

```
f_c = R / (2πL)
```

### 4.4 与 RC 低通的关系

RC 和 RL 低通滤波器具有相同形式的传递函数，只是时间常数不同：
- RC: τ = RC
- RL: τ = L/R

## 5. RC 高通滤波器

### 5.1 电路结构

```
输入 ─── C ───┬─── 输出
              │
              R
              │
             GND
```

### 5.2 传递函数

```
H(s) = sRC / (1 + sRC)
```

### 5.3 频率响应特性

- 直流增益: 0 (-∞ dB)
- 截止频率处增益: -3dB
- 高频增益: 1 (0dB)
- 低频衰减: +20dB/decade
- 截止频率处相位: +45°
- 相位范围: +90° ~ 0°

## 6. RL 高通滤波器

### 6.1 传递函数

```
H(s) = s / (s + R/L)
```

### 6.2 截止频率

```
f_c = R / (2πL)
```

## 7. RLC 带通滤波器

### 7.1 电路结构

串联 RLC 电路，输出取自电阻 R 两端。

### 7.2 传递函数

```
H(s) = (s/RC) / (s² + s/RC + 1/LC)
```

### 7.3 关键参数

中心频率:
```
f_0 = 1 / (2π√(LC))
```

带宽 (Hz):
```
BW = 1 / (2πRC)
```

品质因数:
```
Q = f_0 / BW = (1/R)√(L/C)
```

### 7.4 频率响应特性

- 中心频率处增益: 1 (0dB)
- 直流和无穷大频率处增益: 0
- 相位范围: +90° ~ 0° ~ -90°
- 中心频率处相位: 0°

### 7.5 Q 值的意义

- Q 值越高，带宽越窄，选择性越好
- Q 值越低，带宽越宽，选择性越差
- Q > 0.5 为欠阻尼，会产生振荡

## 8. RLC 带阻滤波器 (陷波滤波器)

### 8.1 电路结构

LC 串联谐振电路与负载并联。

### 8.2 传递函数

```
H(s) = (s² + 1/LC) / (s² + s/RC + 1/LC)
```

### 8.3 频率响应特性

- 中心频率处增益: 0 (陷波)
- 直流和无穷大频率处增益: 1
- 相位范围: 0° ~ ±180° ~ 0°

### 8.4 与带通的互补性

对于相同的 RLC 参数：
```
|H_bp(f)|² + |H_bs(f)|² ≈ 1
```

## 9. 波特图

### 9.1 什么是波特图

波特图由两部分组成：
1. **幅频响应**: 频率 vs 增益 (dB)
2. **相频响应**: 频率 vs 相位 (度)

频率轴使用对数刻度。

### 9.2 绘制波特图

```python
import numpy as np
import matplotlib.pyplot as plt

# 创建滤波器
from src.lowpass import RCLowPass
filt = RCLowPass(R=1000, C=1e-6)

# 生成频率数组
f = np.logspace(1, 5, 1000)

# 计算响应
mag_db = filt.magnitude_db(f)
phase = filt.phase(f)

# 绘制
fig, (ax1, ax2) = plt.subplots(2, 1)
ax1.semilogx(f, mag_db)
ax2.semilogx(f, phase)
```

### 9.3 波特图解读

- **-3dB 点**: 截止频率
- **斜率**: 每十倍频衰减量 (dB/decade)
- **相位裕度**: 系统稳定性指标

## 10. 实际应用

### 10.1 音频处理

- **低音增强**: 低通滤波器
- **高音增强**: 高通滤波器
- **均衡器**: 多个带通滤波器组合
- **分频器**: 将音频分为低频和高频通道

### 10.2 信号调理

- **抗混叠**: 采样前低通滤波
- **直流去除**: 高通滤波器
- **工频消除**: 50/60Hz 陷波滤波器
- **带宽限制**: 带通滤波器

### 10.3 通信系统

- **信道选择**: 带通滤波器
- **噪声抑制**: 陷波滤波器
- **信号恢复**: 匹配滤波器

## 11. 常见问题

### Q1: 为什么使用 dB 表示增益？

dB 是对数刻度，适合表示大范围的数值：
- 便于计算级联系统的总增益
- 符合人耳对声音强度的感知
- 简化波特图的绘制

### Q2: 截止频率的定义是什么？

截止频率是增益下降到最大增益的 1/√2 (约 0.707) 时的频率，对应 -3dB。

### Q3: Q 值和带宽的关系？

Q = f_0 / BW

Q 值越高，带宽越窄，滤波器的选择性越好。

### Q4: 为什么实际滤波器与理想滤波器不同？

理想滤波器在截止频率处有无限陡峭的过渡，这在物理上无法实现。实际滤波器存在过渡带，其陡峭程度取决于滤波器阶数。
