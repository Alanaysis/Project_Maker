### 阶段 2: 核心滤波器实现

**目标**: 实现基本滤波器类

**完成内容**:

#### 2.1 低通滤波器

- `RCLowPass`: RC 低通滤波器
  - 传递函数: H(s) = 1 / (1 + sRC)
  - 截止频率: fc = 1 / (2πRC)
  - 实现频率响应计算

- `RLLowPass`: RL 低通滤波器
  - 传递函数: H(s) = (R/L) / (s + R/L)
  - 截止频率: fc = R / (2πL)

#### 2.2 高通滤波器

- `RCHighPass`: RC 高通滤波器
  - 传递函数: H(s) = sRC / (1 + sRC)

- `RLHighPass`: RL 高通滤波器
  - 传递函数: H(s) = s / (s + R/L)

#### 2.3 带通滤波器

- `RLCBandPass`: RLC 带通滤波器
  - 传递函数: H(s) = (s/RC) / (s² + s/RC + 1/LC)
  - 实现中心频率、带宽、Q 值计算

#### 2.4 带阻滤波器

- `RLCBandStop`: RLC 带阻滤波器
  - 传递函数: H(s) = (s² + 1/LC) / (s² + s/RC + 1/LC)

**技术要点**:
- 使用复数运算计算传递函数
- 处理数值稳定性问题 (避免 log(0))
- 统一的接口设计

## 技术难点与解决方案

### 难点 1: 数值稳定性

**问题**: 计算 dB 时 log(0) 导致错误

**解决**:
```python
mag_db = 20 * np.log10(np.maximum(mag, 1e-30))
```

### 难点 2: 截止频率检测

**问题**: 离散频率点可能不精确

**解决**: 使用线性插值提高精度

### 难点 3: 频域滤波

**问题**: 时域卷积计算量大

**解决**: 使用 FFT 进行频域滤波

### 难点 4: 相位计算

**问题**: 相位包裹问题

**解决**: 使用 np.angle() 自动处理
