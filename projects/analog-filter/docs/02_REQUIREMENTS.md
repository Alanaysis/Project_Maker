## 4. 接口设计

### 4.1 统一接口

所有滤波器类应实现以下方法：

```python
class FilterBase:
    def transfer_function(self, f: np.ndarray) -> np.ndarray:
        """计算传递函数 H(jω)"""

    def magnitude(self, f: np.ndarray) -> np.ndarray:
        """计算幅频响应 (线性值)"""

    def magnitude_db(self, f: np.ndarray) -> np.ndarray:
        """计算幅频响应 (dB)"""

    def phase(self, f: np.ndarray) -> np.ndarray:
        """计算相频响应 (度)"""
```

### 4.2 输入输出规范

**输入**:
- 频率: Hz (标量或数组)
- 元件参数: R (Ω), L (H), C (F)

**输出**:
- 幅度: 线性值或 dB
- 相位: 度
- 传递函数: 复数
