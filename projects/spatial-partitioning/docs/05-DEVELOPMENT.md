# 空间划分算法开发日志

## 2024-01-15

### 项目初始化

- 创建项目结构
- 实现基础数据结构（Vec3, AABB）
- 编写 CMakeLists.txt

### 遇到的问题

1. **浮点精度问题**：AABB 碰撞检测需要处理浮点误差
   - 解决方案：使用 epsilon 比较

2. **内存管理**：Octree 节点分配效率低
   - 解决方案：使用内存池预分配

## 2024-01-16

### BVH 实现

- 实现 BVH 构建算法
- 实现 AABB 碰撞检测
- 实现射线查询

### 性能优化

1. **构建优化**：使用迭代而非递归
2. **查询优化**：使用栈而非队列
3. **内存优化**：使用紧凑节点结构

### 测试结果

- 100,000 个物体构建时间：< 100ms
- 10,000 次查询时间：< 10ms

## 2024-01-17

### Octree 实现

- 实现八叉树构建
- 实现插入和查询
- 实现动态更新

### 遇到的问题

1. **边界物体**：物体跨越多个子节点
   - 解决方案：插入到所有相交的子节点

2. **深度控制**：需要限制最大深度
   - 解决方案：添加 maxDepth 参数
