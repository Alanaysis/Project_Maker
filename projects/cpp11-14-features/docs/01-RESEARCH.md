# C++11/14 新特性调研报告

## 1. 市场调研

### 1.1 C++11/14 的重要性

C++11 是 C++ 语言自 1998 年以来最重要的标准更新，引入了大量现代化特性：
- **移动语义**：显著提升性能，减少不必要的拷贝
- **Lambda 表达式**：简化函数式编程
- **智能指针**：解决内存管理问题
- **并发支持**：标准化多线程编程

C++14 作为 C++11 的小幅改进，主要增加了：
- 泛型 Lambda
- 返回类型推导
- 变量模板
- 二进制字面量

## 2. 核心特性分析

### 2.1 移动语义和右值引用

**问题**：传统 C++ 中，临时对象的拷贝造成性能浪费。

**解决方案**：
- 右值引用（`T&&`）绑定临时对象
- 移动构造函数和移动赋值运算符
- `std::move` 将左值转换为右值引用

**示例场景**：
```cpp
std::vector<int> createLargeVector() {
    std::vector<int> v(1000000);
    return v;  // 移动而非拷贝
}
```

### 2.2 Lambda 表达式

**优势**：
- 简化回调函数
- 支持闭包（捕获外部变量）
- 与 STL 算法完美配合

**语法**：
```cpp
[capture](params) -> return_type { body }
```

### 2.3 智能指针

| 类型 | 所有权 | 使用场景 |
|-----|-------|---------|
| `unique_ptr` | 独占 | 独占资源所有权 |
| `shared_ptr` | 共享 | 共享资源所有权 |
| `weak_ptr` | 无 | 打破循环引用 |

### 2.4 并发编程

C++11 标准化了多线程支持：
- `std::thread`：线程管理
- `std::mutex`：互斥锁
- `std::condition_variable`：条件变量
- `std::future`/`std::promise`：异步编程

### 2.5 可变参数模板

允许模板接受任意数量的参数：
```cpp
template<typename... Args>
void print(Args... args) {
    // 使用折叠表达式(C++17)或递归展开
}
```

### 2.6 constexpr

编译期计算，提升运行时性能：
```cpp
constexpr int factorial(int n) {
    return n <= 1 ? 1 : n * factorial(n - 1);
}
```

### 2.7 auto 和 decltype

类型推导，简化代码：
```cpp
auto x = 42;                    // int
decltype(x) y = 10;            // int
```

### 2.8 范围 for 循环

简化容器遍历：
```cpp
std::vector<int> v = {1, 2, 3};
for (auto& elem : v) {
    // 处理 elem
}
```

### 2.9 初始化列表

统一初始化语法：
```cpp
std::vector<int> v{1, 2, 3, 4, 5};
int arr[]{1, 2, 3};
```
