## 🎯 学习路径

```
iOS 开发 → Android 开发 → 鸿蒙开发 → 跨平台框架
     ↓           ↓           ↓           ↓
  UIKit/SwiftUI  Activity/Compose ArkUI     Dart VM
  Auto Layout    Lifecycle     状态管理   Skia 渲染
  URLSession     Intent       分布式    Widget 树
```

### 推荐学习顺序

1. **ios-app** (iOS 基础应用)
   - 学习 UIKit/SwiftUI
   - 理解 Auto Layout
   - 掌握网络请求

2. **android-app** (Android 基础应用)
   - 学习 Jetpack Compose
   - 理解生命周期
   - 掌握 Intent 导航

3. **harmonyos-app** (鸿蒙基础应用)
   - 学习 ArkUI 声明式 UI
   - 理解状态管理
   - 掌握分布式能力

4. **cross-platform** (跨平台框架原理)
   - 学习 Dart VM 原理
   - 理解 Skia 渲染
   - 掌握 Widget 树和平台通道
