## 1. 核心概念

### SVG
可缩放矢量图形标准，基于 XML 的 2D 图形格式。

### 矢量图形
用数学公式描述图形，可无限缩放不失真。

### 渲染管线
解析 → 构建场景图 → 应用样式 → 输出 SVG

## 2. 重点难点

### 路径解析
SVG path 命令 (M, L, C, Z) 的解析和追踪。
