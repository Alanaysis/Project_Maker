## 学习路径

```
PID 基础 → 状态空间 → 自适应控制 → 模糊控制 → 模型预测控制
   ↓           ↓           ↓           ↓           ↓
 比例控制  状态估计    MRAC算法    模糊推理    滚动优化
 积分控制  可控可观    参数估计    隶属函数    约束处理
 微分控制  LQR控制    Lyapunov   去模糊化    预测模型
```

### 推荐学习顺序

1. **pid-controller** (PID 控制器)
   - 学习比例、积分、微分控制原理
   - 理解 PID 参数整定方法
   - 掌握基本控制回路设计

2. **state-space** (状态空间模型)
   - 学习连续/离散时间状态空间表示和离散化方法
   - 掌握可控性、可观性和稳定性分析
   - 理解卡尔曼滤波和状态估计
   - 掌握极点配置和 LQR 最优控制
   - 设计全阶/降阶观测器
   - 应用案例: 倒立摆控制、电机控制

3. **adaptive-controller** (自适应控制器)
   - 学习模型参考自适应控制 (MRAC)
   - 理解参数估计方法 (RLS, 梯度下降)
   - 掌握 Lyapunov 稳定性设计

4. **fuzzy-controller** (模糊控制器)
   - 学习模糊集合理论
   - 理解模糊推理机制
   - 掌握模糊控制器设计

5. **mpc-controller** (模型预测控制)
   - 学习 MPC 基本原理和滚动优化
   - 理解约束优化和 QP 问题
   - 掌握模型预测和反馈校正

6. **system-response** (系统响应分析)
   - 学习时域响应（阶跃/脉冲/斜坡）
   - 掌握频域分析（波特图/奈奎斯特图）
   - 理解稳定裕度和性能指标
   - 掌握劳斯判据和根轨迹
   - 学习系统辨识和控制器设计

## 核心概念

### 控制系统基本结构

```
参考输入 r(t) ──▶ [控制器] ──▶ [被控系统] ──▶ 输出 y(t)
                       ▲               │
                       │               │
                       └─── 误差 e(t) ◀┘
```

### 自适应控制核心循环

```
系统输出 → 参数估计 → 控制器更新 → 控制输出
```

### 主要控制方法

| 方法 | 特点 | 适用场景 |
|------|------|---------|
| PID | 简单、成熟、应用广泛 | 线性时不变系统 |
| 状态空间 | 状态估计、可控可观分析 | 多变量线性系统 |
| MRAC | 参数自动调整、跟踪性能好 | 参数不确定系统 |
| 模糊控制 | 无需精确模型、鲁棒性强 | 非线性、复杂系统 |
| MPC | 预测控制、约束处理、多变量 | 有约束的多变量系统 |
| 系统响应分析 | 时域/频域全面分析、系统辨识 | 系统建模与分析 |

## 技术要点

### PID 控制器
- **比例 (P)**：响应当前误差
- **积分 (I)**：消除稳态误差
- **微分 (D)**：预测误差趋势

### 状态空间模型
- **状态方程**：x(k+1) = A*x(k) + B*u(k)
- **卡尔曼滤波**：最优状态估计
- **可控性/可观性**：系统分析基础
- **LQR 控制**：线性二次型最优控制

### 自适应控制器
- **参考模型**：定义期望闭环行为
- **自适应律**：MIT 规则、Lyapunov 方法
- **参数估计**：RLS、梯度下降

### 模糊控制器
- **模糊化**：将精确值转换为模糊集合
- **模糊推理**：应用模糊规则
- **去模糊化**：将模糊输出转换为精确值

### 模型预测控制
- **预测模型**：使用系统模型预测未来行为
- **滚动优化**：在每个时刻求解有限时域优化问题
- **反馈校正**：基于实际测量更新预测
- **约束处理**：显式处理输入、状态、输出约束

### 系统响应分析
- **传递函数**：G(s) = N(s)/D(s)，零极点增益表示
- **时域指标**：上升时间、超调量、调节时间、稳态误差
- **频域分析**：波特图、奈奎斯特图、增益/相位裕度
- **稳定性**：劳斯判据、根轨迹、临界增益
- **系统辨识**：从阶跃/频率/脉冲响应辨识模型
- **控制器设计**：PID 整定、超前/滞后补偿器
