# 01 - 决策树调研文档

## 1. 决策树概述

决策树是一种基本的分类与回归方法。它是一种树形结构，其中每个内部节点表示对某个属性的判断，每个分支表示一个判断结果的输出，最后每个叶节点表示一种分类结果。

## 2. 决策树原理

### 2.1 信息增益 (Information Gain)

信息增益是 ID3 算法中用于特征选择的核心概念。它基于信息论中的熵(Entropy)概念。

**熵的定义**：
$$H(D) = -\sum_{i=1}^{n} p_i \log_2(p_i)$$

其中：
- $D$ 是数据集
- $p_i$ 是第 $i$ 类样本在数据集中的比例
- $n$ 是类别的数量

**信息增益定义**：
$$Gain(D, A) = H(D) - \sum_{v} \frac{|D_v|}{|D|} H(D_v)$$

其中：
- $H(D)$ 是数据集 $D$ 的熵
- $A$ 是特征
- $D_v$ 是特征 $A$ 取值为 $v$ 的子集

### 2.2 信息增益率 (Gain Ratio)

信息增益率是 C4.5 算法使用的特征选择标准，解决了 ID3 偏向于选择取值较多特征的问题。

**信息增益率定义**：
$$GainRatio(D, A) = \frac{Gain(D, A)}{IV(A)}$$

其中 IV(A) 是特征 A 的固有值：
$$IV(A) = -\sum_{v} \frac{|D_v|}{|D|} \log_2 \frac{|D_v|}{|D|}$$

### 2.3 基尼系数 (Gini Index)

基尼系数是 CART 分类算法使用的特征选择标准。

**基尼系数定义**：
$$Gini(D) = 1 - \sum_{i=1}^{n} p_i^2$$

其中 $p_i$ 是类别 $i$ 的比例。

**基尼系数的含义**：
- Gini(D) = 0 时，数据集 D 纯度最高
- Gini(D) = 1 - 1/n 时，数据集 D 纯度最低（n 为类别数）

### 2.4 方差减少 (Variance Reduction)

方差减少是 CART 回归算法使用的特征选择标准。

**方差定义**：
$$Var(D) = \frac{1}{|D|} \sum_{i=1}^{|D|} (y_i - \bar{y})^2$$

其中 $\bar{y}$ 是目标变量的均值。

## 3. 决策树构建算法

### 3.1 ID3 算法

ID3 (Iterative Dichotomiser 3) 是由 Ross Quinlan 于 1986 年提出的。

**特点：**
- 使用信息增益作为特征选择标准
- 只能处理离散特征
- 容易过拟合
- 没有剪枝机制

**算法流程：**
1. 计算数据集的熵
2. 对每个特征，计算信息增益
3. 选择信息增益最大的特征作为分裂特征
4. 根据特征的取值分裂数据集
5. 递归构建子树

### 3.2 C4.5 算法

C4.5 是 ID3 的改进版本，由 Ross Quinlan 于 1993 年提出。

**改进：**
- 使用信息增益率解决偏向问题
- 可以处理连续特征
- 可以处理缺失值
- 支持后剪枝

**连续特征处理：**
- 将连续特征的值排序
- 尝试所有可能的分裂点
- 选择信息增益率最大的分裂点

### 3.3 CART 算法

CART (Classification and Regression Trees) 是由 Breiman 等人于 1984 年提出的。

**特点：**
- 二叉树结构
- 分类任务使用基尼系数
- 回归任务使用方差减少
- 支持后剪枝（代价复杂度剪枝）

**算法流程：**
1. 对每个特征，计算基尼系数（分类）或方差（回归）
2. 选择基尼系数最小（分类）或方差减少最大（回归）的特征
3. 寻找最佳分裂阈值
4. 根据阈值分裂数据集
5. 递归构建左右子树

## 4. 剪枝技术

### 4.1 预剪枝 (Pre-pruning)

在构建树的过程中提前停止分裂。

**常用策略：**
- 设置最大深度
- 设置最小样本数
- 设置最小信息增益
- 使用验证集评估

**优点：**
- 计算效率高
- 防止过拟合

**缺点：**
- 可能导致欠拟合
- 需要仔细调整参数

### 4.2 后剪枝 (Post-pruning)

先构建完整的树，然后自底向上剪枝。

**代价复杂度剪枝 (Cost Complexity Pruning, CCP)：**

```
Cost(T) = R(T) + alpha * |T_leaf|
```

其中：
- R(T) 是树的训练误差
- alpha 是复杂度参数
- |T_leaf| 是叶节点数量

**减少错误剪枝 (Reduced Error Pruning)：**

使用验证集评估剪枝是否有效。

**优点：**
- 更不容易欠拟合
- 通常比预剪枝效果更好

**缺点：**
- 计算成本高
- 需要额外的验证数据

## 5. 特征重要性

### 5.1 基于不纯度的特征重要性

特征重要性可以通过特征在树中被选为分裂特征的次数和效果来衡量。

```
importance(feature) = Σ (节点不纯度减少 * 节点样本数)
```

### 5.2 排列重要性

通过打乱特征值并观察模型性能下降来衡量特征重要性。

## 6. 评估指标

### 6.1 分类指标

- **准确率 (Accuracy)**: 正确预测的比例
- **精确率 (Precision)**: 预测为正的样本中实际为正的比例
- **召回率 (Recall)**: 实际为正的样本中被正确预测的比例
- **F1 分数**: 精确率和召回率的调和平均

### 6.2 回归指标

- **均方误差 (MSE)**: 预测值与真实值差的平方的平均
- **均方根误差 (RMSE)**: MSE 的平方根
- **平均绝对误差 (MAE)**: 预测值与真实值差的绝对值的平均
- **R² 分数**: 模型解释的方差比例

## 10. 总结

决策树是机器学习中最基础、最直观的算法之一。通过信息增益、信息增益率或基尼系数进行特征选择，递归构建树结构，最终实现分类或回归任务。理解决策树的原理对于掌握更复杂的集成学习方法（如随机森林、梯度提升树）至关重要。
