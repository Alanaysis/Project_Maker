# Research: System Response Analysis

## 领域概述

系统响应分析是控制理论的核心，研究 LTI（线性时不变）系统在各种输入下的动态行为。

## 核心理论

### 1. 传递函数

传递函数是 LTI 系统在拉普拉斯域的数学表示：

```
G(s) = Y(s)/U(s) = N(s)/D(s) = (b_m s^m + ... + b_0) / (a_n s^n + ... + a_0)
```

关键属性：
- **极点**：D(s) = 0 的根，决定系统稳定性
- **零点**：N(s) = 0 的根，影响响应形状
- **DC 增益**：G(0)，稳态增益

### 2. 时域响应

#### 阶跃响应
单位阶跃输入 u(t) = 1(t) 下的输出：
- 一阶系统：y(t) = K(1 - e^{-t/tau})
- 二阶系统：取决于阻尼比 zeta

#### 脉冲响应
单位脉冲输入 delta(t) 下的输出，等于传递函数的拉普拉斯逆变换。

#### 斜坡响应
斜坡输入 r(t) = t 下的输出，用于评估跟踪能力。

### 3. 频域响应

#### 波特图
- 幅频特性：|G(jw)| vs log(w)
- 相频特性：angle(G(jw)) vs log(w)

#### 奈奎斯特图
G(jw) 在复平面上的极坐标图，用于判断闭环稳定性。

#### 稳定裕度
- **增益裕度**：相位穿越频率处的增益余量
- **相位裕度**：增益穿越频率处的相位余量

### 4. 性能指标

| 指标 | 定义 | 典型要求 |
|------|------|----------|
| 上升时间 | 10%-90% | 尽量短 |
| 超调量 | (峰值-终值)/终值 | < 20% |
| 调节时间 | 进入2%误差带 | 尽量短 |
| 稳态误差 | 1 - 终值 | 接近 0 |

### 5. 稳定性分析

#### 劳斯判据
通过特征多项式系数构建劳斯表，判断 RHP 极点数量。

#### 根轨迹
随增益 K 变化，闭环极点在 s 平面上的轨迹。

## 算法选择

### 传递函数运算
- 使用 NumPy 多项式运算实现加减乘除
- 反馈连接通过特征多项式构造

### 时域响应
- 基于 scipy.signal.lti/step/impulse 实现
- 斜坡响应通过 lsim 实现

### 频域响应
- 波特图：scipy.signal.bode
- 奈奎斯特：直接计算 G(jw)
- 裕度：scipy.signal.margin

### 稳定性分析
- 劳斯表：手动实现数值计算
- 根轨迹：对每个增益值求特征根

### 系统辨识
- 阶跃响应法：提取特征参数
- 频域法：最小二乘拟合
- 脉冲响应法：Prony 方法

### 控制器设计
- PID：Ziegler-Nichols 整定规则
- 超前/滞后：基于相位裕度的设计方法
