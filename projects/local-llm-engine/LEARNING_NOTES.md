### 1.2 KV Cache 原理

**学习日期**: ___________

#### 核心概念
- [ ] 为什么需要 KV Cache？
- [ ] KV Cache 如何减少计算量？
- [ ] 内存占用如何计算？
- [ ] 有哪些优化策略？

#### 工作原理
```
没有 KV Cache:
- 每生成一个 token，需要重新计算所有位置的 K, V
- 时间复杂度: O(n²)

有 KV Cache:
- 缓存已计算的 K, V
- 只计算新 token 的 K, V
- 时间复杂度: O(n)
```

### 1.3 位置编码 (RoPE)

**学习日期**: ___________

#### 核心概念
- [ ] 什么是 RoPE？
- [ ] 为什么选择 RoPE 而不是绝对位置编码？
- [ ] RoPE 的数学原理是什么？
- [ ] 如何实现 RoPE？

#### 工作原理
```
RoPE 将位置信息编码为旋转：
q'[2i] = q[2i] * cos(θ) - q[2i+1] * sin(θ)
q'[2i+1] = q[2i] * sin(θ) + q[2i+1] * cos(θ)

其中 θ = position * freq
```
