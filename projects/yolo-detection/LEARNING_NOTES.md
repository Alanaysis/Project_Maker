# YOLO 目标检测 - 学习笔记

## 核心概念理解

### 1. 目标检测 vs 图像分类

**图像分类**：一张图一个标签
**目标检测**：一张图多个物体，每个有位置和类别

关键区别：
- 检测需要处理**可变数量**的输出
- 需要**定位**和**分类**同时进行
- 评估指标更复杂（IoU, mAP）

### 2. YOLO 的核心思想

**传统方法**：滑动窗口 → 每个位置独立判断
**YOLO 方法**：全局推理 → 一次前向传播完成所有检测

```
输入图像 → [神经网络] → 所有检测结果
```

关键创新：
- 将检测视为**回归问题**
- 端到端训练
- 实时性能

### 3. 网格划分机制

```
图像划分为 S×S 网格 (默认 7×7)
每个单元格预测:
  - B 个边界框 (默认 2)
  - 每个框: (x, y, w, h, confidence)
  - C 个类别概率 (默认 20)
```

输出张量形状：S × S × (B×5 + C) = 7 × 7 × 30

### 4. 边界框表示

**绝对坐标**：像素值 (100, 200, 50, 80)
**归一化坐标**：相对于图像尺寸 (0.22, 0.45, 0.11, 0.18)
**网格相对坐标**：相对于单元格位置 (0.5, 0.3)

```python
# 绝对坐标 → 网格相对坐标
cell_x = int(x / cell_size)
x_rel = (x / cell_size) - cell_x

# 网格相对坐标 → 绝对坐标
x_abs = (x_rel + cell_x) * cell_size
```

### 5. 置信度含义

```
confidence = P(Object) × IoU
```

- P(Object)：单元格中是否有物体
- IoU：预测框与真实框的交并比
- 训练时：有物体的单元格目标置信度 = 1，无物体 = 0

### 6. NMS 算法

**问题**：检测器产生多个重叠框
**解决**：保留最好的，删除重复的

```python
# 算法步骤
1. 按置信度排序
2. 选择最高的框 A
3. 删除与 A 的 IoU > 阈值的框
4. 重复
```

关键参数：
- IoU 阈值：通常 0.5
- 置信度阈值：过滤低置信度检测

## 损失函数设计

### 五部分损失

```python
Loss = λ_coord × (loc_xy + loc_wh)  # 定位损失
     + conf_obj                       # 有目标置信度损失
     + λ_noobj × conf_noobj           # 无目标置信度损失
     + class_loss                     # 分类损失
```

### 为什么用平方根？

```python
# wh 损失使用平方根
(√w - √ŵ)²  而不是  (w - ŵ)²
```

原因：小物体的尺寸误差应该比大物体更敏感
- 大物体：100 → 105 (5% 误差)
- 小物体：10 → 15 (50% 误差)
- 平方根可以平衡这种差异

### λ 参数的意义

- λ_coord = 5.0：强调定位准确性
- λ_noobj = 0.5：大部分单元格没有物体，降低权重防止梯度消失

## 实现技巧

### 1. 批量 IoU 计算

```python
# 使用广播避免循环
boxes1: (N, 4) → (N, 1, 4)
boxes2: (M, 4) → (1, M, 4)
# 广播后：(N, M, 4) → 计算 (N, M) 的 IoU 矩阵
```

### 2. 预测解码

```python
# 创建网格偏移
grid_y, grid_x = torch.meshgrid(
    torch.arange(S), torch.arange(S), indexing='ij'
)

# 转换坐标
x_abs = (pred_x + grid_x) * cell_size
y_abs = (pred_y + grid_y) * cell_size
```

### 3. 目标张量创建

```python
# 对每个标注框
1. 确定所属单元格 (cell_x, cell_y)
2. 计算单元格内相对坐标
3. 填充目标张量
```

### 4. Soft NMS

```python
# 不删除框，而是衰减分数
score = score × exp(-iou² / sigma)
```

优点：更适合密集场景，不会误删相邻物体

## 常见问题

### Q1: 为什么检测效果不好？

可能原因：
1. 训练数据不足
2. 模型太小
3. 学习率不合适
4. 没有数据增强
5. 损失权重不平衡

### Q2: 如何处理不同大小的物体？

YOLOv1 的局限：
- 每个单元格只预测固定数量的框
- 难以检测小物体

改进方法（YOLOv2/v3）：
- 引入锚框
- 多尺度预测
- 特征金字塔网络

### Q3: NMS 阈值如何选择？

- IoU 阈值：0.5（通用），0.75（严格）
- 置信度阈值：0.1-0.5（取决于应用）
- 可以通过验证集调优

### Q4: 如何加速训练？

1. 使用预训练骨干网络
2. 混合精度训练
3. 增大批量大小
4. 使用学习率预热
5. 梯度累积

## 扩展学习

### YOLO 系列演进

| 版本 | 关键改进 | 年份 |
|------|---------|------|
| v1 | 网格划分，全局推理 | 2016 |
| v2 | 锚框，批归一化，多尺度 | 2017 |
| v3 | 多尺度预测，特征金字塔 | 2018 |
| v4 | 更多优化技巧 | 2020 |
| v5 | 工程优化，易于使用 | 2020 |

### 相关技术

1. **锚框 (Anchor Boxes)**
   - 预定义参考框
   - 使用 k-means 聚类确定尺寸
   - 预测相对偏移

2. **特征金字塔网络 (FPN)**
   - 多尺度特征融合
   - 提高小物体检测

3. **损失函数改进**
   - Focal Loss：处理类别不平衡
   - CIoU Loss：更好的回归损失

4. **数据增强**
   - Mosaic：四张图拼接
   - Mixup：图像混合
   - CutOut：随机遮挡
