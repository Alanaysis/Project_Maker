# 开发日志

## 开发环境

- Python 3.10+
- 无第三方依赖
- pytest (测试)

## 开发过程

### 阶段1: 需求分析

**目标**: 实现从零开始的朴素贝叶斯分类器

**需求**:
1. 实现贝叶斯定理
2. 支持三种变体 (高斯、多项式、伯努利)
3. 完整的训练和预测功能
4. 概率估计
5. 充分的测试

### 阶段2: 架构设计

**决策**:
1. 使用抽象基类定义统一接口
2. 每个变体独立实现
3. 使用对数概率避免数值问题
4. 应用Laplace平滑处理零概率

### 阶段3: 核心实现

**实现顺序**:
1. 基类 `NaiveBayesClassifier`
2. 高斯朴素贝叶斯
3. 多项式朴素贝叶斯
4. 伯努利朴素贝叶斯

**关键代码**:
- 贝叶斯计算: `_calculate_likelihood()`
- 概率归一化: `predict_proba()`
- 参数估计: `fit()`

### 阶段4: 测试验证

**测试内容**:
- 单元测试每个方法
- 集成测试完整流程
- 边界测试异常情况

**测试结果**: 全部通过

## 遇到的问题

### 问题1: 数值下溢

**现象**: 连乘小概率导致结果为0
**解决**: 使用对数空间计算

### 问题2: 零概率

**现象**: 某特征在某类别中未出现，概率为0
**解决**: Laplace平滑

### 问题3: 方差为零

**现象**: 某类别下某特征值全部相同
**解决**: 方差平滑参数
