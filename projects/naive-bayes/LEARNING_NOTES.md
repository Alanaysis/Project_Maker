# 学习笔记

## 贝叶斯定理

### 公式

```
P(C|X) = P(X|C) * P(C) / P(X)
```

### 含义

- **P(C|X)**: 后验概率 - 给定数据X，类别为C的概率
- **P(X|C)**: 似然 - 类别C下观测到数据X的概率
- **P(C)**: 先验概率 - 类别C的概率
- **P(X)**: 证据 - 观测到数据X的概率

### 直观理解

贝叶斯定理告诉我们如何根据观测数据更新我们的信念:

1. **先验**: 在观测之前的信念 (P(C))
2. **似然**: 数据对假设的支持程度 (P(X|C))
3. **后验**: 观测之后的信念 (P(C|X))

### 例子

**垃圾邮件检测**:
- P(垃圾邮件) = 0.3 (先验)
- P("免费"|垃圾邮件) = 0.8 (似然)
- P("免费"|正常邮件) = 0.1 (似然)

收到包含"免费"的邮件:
```
P(垃圾邮件|"免费") ∝ P("免费"|垃圾邮件) * P(垃圾邮件)
                   = 0.8 * 0.3 = 0.24

P(正常邮件|"免费") ∝ P("免费"|正常邮件) * P(正常邮件)
                   = 0.1 * 0.7 = 0.07

归一化后:
P(垃圾邮件|"免费") = 0.24 / (0.24 + 0.07) ≈ 0.77
```

## 条件独立假设

### 定义

朴素贝叶斯假设所有特征在给定类别条件下相互独立:

```
P(X1, X2, ..., Xn | C) = P(X1|C) * P(X2|C) * ... * P(Xn|C)
```

### 为什么"朴素"

这个假设在现实中几乎不成立，因为特征之间往往存在相关性。

**例子**: 文本分类中，"machine" 和 "learning" 经常一起出现，不独立。

### 为什么有效

尽管假设不成立，朴素贝叶斯在实践中表现良好:

1. **分类只需要排序**: 不需要精确概率
2. **特征相关性影响有限**: 相关性对所有类别影响相似
3. **正则化效果**: 独立假设相当于正则化

## 概率估计

### 最大似然估计 (MLE)

直接使用频率估计概率:

```
P(xi|C) = count(xi in C) / count(C)
```

**问题**: 可能为0

### 最大后验估计 (MAP)

加入先验:

```
P(xi|C) = (count(xi in C) + α) / (count(C) + α * n)
```

**效果**: Laplace平滑

## 三种变体

### 高斯朴素贝叶斯

**适用**: 连续特征
**假设**: 特征服从正态分布
**参数**: 均值μ、方差σ²

**公式**:
```
P(xi|C) = 1/√(2πσ²) * exp(-(xi-μ)²/(2σ²))
```

### 多项式朴素贝叶斯

**适用**: 离散特征 (计数)
**典型**: 文本分类 (词频)
**参数**: 特征概率

**公式**:
```
P(xi|C) = (Nxi + α) / (Nc + α*n)
```

### 伯努利朴素贝叶斯

**适用**: 二值特征 (0/1)
**典型**: 文本分类 (词是否出现)
**参数**: 特征为1的概率

**公式**:
```
P(xi=1|C) = (count + α) / (n + 2α)
P(xi=0|C) = 1 - P(xi=1|C)
```

## 关键概念

### 1. 对数空间计算

**为什么**: 避免连乘导致的数值下溢
**怎么做**: log(a*b) = log(a) + log(b)

### 2. Laplace平滑

**为什么**: 避免零概率
**怎么做**: 给每个计数加上α

### 3. 概率归一化

**为什么**: 后验概率需要归一化
**怎么做**: 使用log-sum-exp技巧

## 实践经验

### 1. 数据预处理

- 高斯: 标准化可能有帮助
- 多项式: 词频或TF-IDF
- 伯努利: 二值化

### 2. 参数选择

- α: 通常1.0，可通过交叉验证调整
- 特征选择: 移除低频词

### 3. 评估指标

- 准确率: 整体正确率
- 精确率: 预测为正中真正为正的比例
- 召回率: 真正为正中被预测为正的比例
- F1: 精确率和召回率的调和平均

## 总结

朴素贝叶斯是一个简单但强大的分类算法:

1. **理论基础**: 贝叶斯定理
2. **核心假设**: 条件独立
3. **优点**: 快速、简单、有效
4. **缺点**: 假设过强
5. **应用**: 文本分类等

通过实现这个算法，深入理解了:
- 贝叶斯推理的原理
- 概率估计的方法
- 数值计算的技巧
