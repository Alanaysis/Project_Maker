## 3. 核函数测试

### 3.1 线性核测试

- 基本点积计算
- 正交向量核值为 0
- 向量自身核值等于模的平方
- 对称性 K(x,y) = K(y,x)
- 零向量核值为 0

### 3.2 RBF 核测试

- 向量自身核值为 1
- 对称性
- 核值范围 (0, 1]
- gamma 参数影响
- 无效 gamma 值抛出异常
- 远距离点核值接近 0

### 3.3 多项式核测试

- degree=1 等价于线性核加偏置
- 对称性
- degree 参数影响
- 无效 degree 值抛出异常

### 3.4 Sigmoid 核测试

- 对称性
- 核值范围 (-1, 1)
- 零向量核值为 0
- coef0 参数影响
- 无效 gamma 值抛出异常
