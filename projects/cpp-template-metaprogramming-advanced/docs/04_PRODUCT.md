# 04_PRODUCT.md - 产品思考

## 学习目标

### 入门目标
1. 理解模板元编程的基本概念
2. 掌握类型列表的操作
3. 理解编译期计算的原理
4. 能够使用 `static_assert` 进行编译期验证

### 中级目标
1. 掌握表达式模板技术
2. 理解 CRTP 及其应用
3. 能够使用策略类配置行为
4. 掌握 SFINAE 成员检测

### 高级目标
1. 实现编译期状态机
2. 使用 C++20 Concepts
3. 设计类型安全的库
4. 优化编译时间和运行时性能

## 进阶要点

### 1. 模板实例化机制
- 两阶段查找
- 依赖名称查找
- ADL (Argument Dependent Lookup)

### 2. SFINAE 原理
- 替换失败不是错误
- `void_t` 技巧
- `decltype` 和 `declval`

### 3. 表达式模板
- 延迟求值
- 表达式树构建
- 避免临时对象

### 4. CRTP 详解
- 静态多态
- 运算符重载辅助
- Mixin 模式

### 5. Concepts (C++20)
- 约束定义
- 约束检查
- 约束子句

## 设计决策

### 1. 头文件库
- 模板必须在头文件中定义
- 避免链接错误
- 方便使用

### 2. 命名空间
- 使用 `tmp` 命名空间
- 避免命名冲突
- 清晰的作用域

### 3. C++17/20 特性
- `constexpr` 扩展
- `if constexpr`
- 折叠表达式
- Concepts (C++20)

### 4. 注释风格
- 详细中文注释
- Doxygen 格式
- 示例代码
