# 状态空间模型 - 学习笔记

## 1. 状态空间基础

### 1.1 连续时间系统

**状态方程:**
```
dx/dt = A*x + B*u
y = C*x + D*u
```

**物理意义:**
- 状态方程描述系统内部状态的演化规律
- 输出方程描述如何从状态得到可测量的输出
- 矩阵A描述系统自由运动，B描述输入如何影响状态

### 1.2 离散时间系统

**状态方程:**
```
x(k+1) = A*x(k) + B*u(k)
y(k) = C*x(k) + D*u(k)
```

**与连续系统的关系:**
- 离散化: 连续系统 + 采样保持器 → 离散系统
- 常用方法: 零阶保持器(ZOH)、双线性变换(Tustin)

### 1.3 传递函数

**连续系统:**
```
G(s) = C*(sI - A)^(-1)*B + D
```

**离散系统:**
```
H(z) = C*(zI - A)^(-1)*B + D
```

**SISO系统:** 传递函数是标量有理函数
**MIMO系统:** 传递函数是矩阵，每个元素是一个传递函数

## 2. 系统分析

### 2.1 稳定性

**连续时间系统:**
- 稳定条件: 所有特征值的实部 < 0
- 临界稳定: 特征值实部 = 0（虚轴上）

**离散时间系统:**
- 稳定条件: 所有特征值的模 < 1
- 临界稳定: 特征值模 = 1（单位圆上）

**物理意义:** 稳定系统在有界输入下产生有界输出(BIBO)

### 2.2 可控性

**定义:** 系统可控是指存在控制输入u，能在有限时间内将系统从任意初始状态转移到任意目标状态。

**可控性矩阵:**
```
C = [B, AB, A²B, ..., A^(n-1)B]
```

**判定条件:** rank(C) = n（系统阶数）

**物理意义:** 可控性决定了我们能否通过控制输入影响所有状态

### 2.3 可观性

**定义:** 系统可观是指通过有限时间的输出测量，能唯一确定系统的初始状态。

**可观性矩阵:**
```
O = [C; CA; CA²; ...; CA^(n-1)]
```

**判定条件:** rank(O) = n（系统阶数）

**物理意义:** 可观性决定了我们能否通过输出推断所有状态

### 2.4 可控性与可观性的对偶性

系统(A, B, C)可控 ↔ 系统(A^T, C^T, B^T)可观

这使得很多算法可以利用对偶性简化设计。

## 3. 状态反馈控制

### 3.1 极点配置

**控制律:** u = -K*x + r

**目标:** 选择K使闭环系统(A-BK)的特征值位于期望位置

**Ackermann公式（SISO）:**
```
K = [0, ..., 0, 1] * Co^(-1) * phi(A)
```
其中phi(A)是将期望极点代入特征多项式后的矩阵多项式

**设计步骤:**
1. 根据性能要求确定期望极点
2. 计算可控性矩阵
3. 使用Ackermann公式求解K

### 3.2 LQR最优控制

**性能指标:**
```
J = Σ(k=0 to ∞) [x(k)^T * Q * x(k) + u(k)^T * R * u(k)]
```

**物理意义:**
- Q: 状态权重，越大表示越重视状态偏差
- R: 控制权重，越大表示越节省控制能量

**最优增益:**
```
K = (R + B^T*P*B)^(-1) * B^T*P*A
```

**Riccati方程:**
```
P = A^T*P*A - A^T*P*B*(R + B^T*P*B)^(-1)*B^T*P*A + Q
```

**LQR优势:**
- 自动确定反馈增益
- 保证闭环稳定
- 可以平衡性能和控制代价

### 3.3 LQG控制

**思想:** LQR + 卡尔曼滤波 = 输出反馈最优控制

**结构:**
1. 卡尔曼滤波器估计状态: x̂
2. LQR控制器基于估计状态计算控制: u = -K*x̂

**分离定理:** 控制器设计和滤波器设计可以独立进行

## 4. 状态观测器

### 4.1 全阶观测器

**观测器方程:**
```
x̂(k+1) = (A - L*C) * x̂(k) + B*u(k) + L*y(k)
```

**设计方法:** 选择L使(A-LC)的特征值位于期望位置

**利用对偶性:** 设计(A^T, C^T)的状态反馈增益，然后转置

**设计准则:** 观测器极点应比控制器极点快2-5倍

### 4.2 降阶观测器

**思想:** 如果部分状态可直接测量，只需估计剩余状态

**优势:** 观测器阶数更低，计算量更小

**设计步骤:**
1. 坐标变换使C = [I 0]
2. 分离可直接测量和需要估计的状态
3. 设计降阶观测器

### 4.3 分离原理

**定理:** 状态反馈控制器和观测器可以独立设计

**闭环极点 = 控制器极点 ∪ 观测器极点**

这意味着:
- 先设计控制器（假设状态已知）
- 再设计观测器（独立于控制器）
- 组合起来即可

## 5. 卡尔曼滤波

### 5.1 问题描述

**系统模型:**
```
x(k) = A*x(k-1) + B*u(k) + w(k)    # 状态方程（含过程噪声）
y(k) = C*x(k) + v(k)                # 观测方程（含测量噪声）
```

**噪声假设:**
- w(k) ~ N(0, Q)  过程噪声
- v(k) ~ N(0, R)  测量噪声
- w和v相互独立

### 5.2 卡尔曼滤波算法

**预测步骤:**
```
x̂(k|k-1) = A * x̂(k-1|k-1) + B * u(k)
P(k|k-1) = A * P(k-1|k-1) * A^T + Q
```

**更新步骤:**
```
K(k) = P(k|k-1) * C^T * (C * P(k|k-1) * C^T + R)^(-1)
x̂(k|k) = x̂(k|k-1) + K(k) * (y(k) - C * x̂(k|k-1))
P(k|k) = (I - K(k) * C) * P(k|k-1)
```

**直觉理解:**
1. 预测: 用系统模型预测状态
2. 新息: 测量值与预测值的差异
3. 更新: 根据新息修正状态估计
4. 卡尔曼增益K自动平衡模型预测和测量值的可信度

### 5.3 卡尔曼增益的含义

- K大: 更信任测量值（R小或Q大）
- K小: 更信任模型预测（R大或Q小）
- K → 0: 完全信任模型
- K → C^(-1): 完全信任测量

### 5.4 RTS平滑器

**思想:** 利用未来信息改善当前估计

**后向递推:**
```
C_k = P(k|k) * A^T * P(k+1|k)^(-1)
x̂(k|T) = x̂(k|k) + C_k * (x̂(k+1|T) - x̂(k+1|k))
```

**优势:** 平滑估计比滤波估计更准确（但需要未来数据）

## 6. 实际应用

### 6.1 倒立摆

**系统特点:**
- 四阶非线性系统
- 开环不稳定
- 需要主动控制才能保持平衡

**控制策略:**
1. 在平衡点附近线性化
2. 设计LQR控制器
3. 设计状态观测器
4. 组合实现输出反馈控制

**关键参数:**
- 小车质量M、摆杆质量m、摆杆长度l
- LQR权重Q、R的选取

### 6.2 直流电机

**系统特点:**
- 三阶线性系统
- 电气子系统（电流响应快）
- 机械子系统（转速响应慢）

**控制策略:**
- 位置控制: LQR
- 速度控制: 状态反馈
- 状态估计: 卡尔曼滤波

**关键参数:**
- 电阻R、电感L、反电动势常数Ke、转矩常数Kt
- 转动惯量J、摩擦系数B

## 7. 设计技巧

### 7.1 LQR权重选择

**Q矩阵:**
- 对角元素对应各状态的权重
- 位置/角度误差大 → Q中对应元素大
- 速度/角速度误差大 → Q中对应元素大

**R矩阵:**
- 控制能量的权重
- R大 → 控制保守，响应慢
- R小 → 控制激进，响应快

**经验法则:**
```
Q = diag([1/sigma_x1^2, 1/sigma_x2^2, ...])
R = [1/sigma_u^2]
```

### 7.2 观测器设计

**极点选择:**
- 观测器极点应比控制器极点快2-5倍
- 太快: 对测量噪声敏感
- 太慢: 状态估计延迟大

**噪声处理:**
- 测量噪声大 → 观测器极点慢一些
- 过程噪声大 → 观测器极点快一些

### 7.3 离散化方法选择

| 方法 | 特点 | 适用场景 |
|------|------|----------|
| ZOH | 保持器假设，精确 | 数字控制 |
| Tustin | 频率预畸变，保持稳定性 | 频率域设计 |
| 欧拉 | 简单，可能不稳定 | 快速原型 |

## 8. 常见问题

### 8.1 系统不可控怎么办？

**原因:** 某些状态无法通过输入影响

**处理:**
1. 检查系统建模是否正确
2. 如果不可控状态本身稳定，可以忽略
3. 如果不可控状态不稳定，需要重新设计系统

### 8.2 LQR导致大控制量

**原因:** R太小，Q太大

**解决:**
- 增大R
- 减小Q
- 使用带约束的MPC

### 8.3 观测器发散

**原因:**
- 观测器极点太靠近不稳定区域
- 模型与实际系统不匹配
- 测量噪声太大

**解决:**
- 将观测器极点移离虚轴/单位圆
- 使用卡尔曼滤波（最优观测器）
- 改善测量质量

### 8.4 离散化后系统不稳定

**原因:** 采样时间太大

**解决:**
- 减小采样时间
- 使用更精确的离散化方法
- 检查连续系统是否稳定
