# 01 - 调研报告：PCA 主成分分析

## 1. 什么是 PCA？

PCA（Principal Component Analysis，主成分分析）是一种经典的无监督降维方法，由 Karl Pearson 于 1901 年提出，后由 Harold Hotelling 于 1933 年独立发展。

### 核心思想

PCA 的核心思想是找到数据方差最大的方向（主成分），将高维数据投影到这些方向上，从而实现降维。

### 数学目标

给定 n 个 d 维样本 X = {x₁, x₂, ..., xₙ}，PCA 寻找 k 个正交方向（k < d），使得投影后的数据方差最大。

形式化地说，PCA 求解以下优化问题：

```
maximize Var(Xw) = w^T * C * w
subject to ||w|| = 1
```

其中 C 是协方差矩阵。

## 2. 数学基础

### 2.1 协方差矩阵

协方差矩阵是 PCA 的核心组件，描述特征之间的线性关系。

对于中心化后的数据矩阵 X（n×d），协方差矩阵定义为：

```
C = (1/(n-1)) * X^T * X
```

性质：
- C 是 d×d 的对称矩阵
- 对角线元素 C[i,i] 是第 i 个特征的方差
- 非对角线元素 C[i,j] 是特征 i 和 j 的协方差
- C 是半正定矩阵（所有特征值 >= 0）

### 2.2 特征值分解

对于实对称矩阵 C，特征值分解为：

```
C = V * Λ * V^T
```

其中：
- V = [v₁, v₂, ..., vₐ] 是特征向量矩阵
- Λ = diag(λ₁, λ₂, ..., λₐ) 是特征值对角矩阵
- λ₁ >= λ₂ >= ... >= λₐ >= 0

特征值的物理意义：
- λᵢ 表示第 i 个主成分方向上的方差
- λᵢ / Σλⱼ 表示第 i 个主成分解释的方差比例

### 2.3 投影

选择前 k 个特征向量 Vₖ = [v₁, v₂, ..., vₖ]，投影公式为：

```
X_projected = X_centered * Vₖ
```

投影后的数据从 d 维降到 k 维。

## 3. 算法流程

```
输入: 数据矩阵 X (n×d)，目标维度 k
输出: 降维后的数据 X_projected (n×k)

1. 中心化: X_centered = X - mean(X)
2. 计算协方差矩阵: C = (1/(n-1)) * X_centered^T * X_centered
3. 特征值分解: C = V * Λ * V^T
4. 选择前 k 个特征向量: Vₖ = V[:, :k]
5. 投影: X_projected = X_centered * Vₖ
```

## 4. 特征值分解方法

### 4.1 幂迭代法（Power Iteration）

原理：从随机向量 v 开始，反复执行 v = Av / ||Av||，v 收敛到最大特征值对应的特征向量。

优点：
- 实现简单
- 适合求解最大特征值
- 内存效率高

缺点：
- 只能求解一个特征值
- 收敛速度取决于特征值间隔
- 需要配合压缩（Deflation）技术求解多个特征值

时间复杂度：O(n² × iterations)

### 4.2 QR 算法

原理：反复执行 QR 分解 A = QR，然后令 A' = RQ。A 收敛到上三角矩阵，对角线元素即为特征值。

优点：
- 可以求解所有特征值
- 收敛速度快（带位移时二次收敛）
- 数值稳定

缺点：
- 实现复杂
- 计算量较大

时间复杂度：O(n³ × iterations)

### 4.3 numpy.linalg.eigh

NumPy 内置函数，使用 LAPACK 库实现，针对对称矩阵优化。

优点：
- 高度优化
- 数值稳定
- 使用方便

本项目实现了前两种方法作为学习目的。

## 5. PCA 的变体

### 5.1 核 PCA（Kernel PCA）

核 PCA 通过核技巧将数据映射到高维特征空间，然后在高维空间执行 PCA。这使得 PCA 能够捕获非线性结构。

**核函数类型：**

| 核函数 | 公式 | 适用场景 |
|--------|------|----------|
| RBF（高斯核） | K(x,y) = exp(-γ\|\|x-y\|\|²) | 通用非线性数据 |
| 多项式核 | K(x,y) = (γx·y + c)^d | 多项式关系 |
| Sigmoid 核 | K(x,y) = tanh(γx·y + c) | 神经网络相关 |
| 线性核 | K(x,y) = x·y | 等价于标准 PCA |

**算法流程：**
1. 计算核矩阵 K
2. 中心化核矩阵：K' = K - 1ₙK - K1ₙ + 1ₙK1ₙ
3. 特征值分解：K' = αΛα^T
4. 归一化特征向量：α = α / √Λ

**优点：**
- 能处理非线性数据
- 无需显式映射到高维空间

**缺点：**
- 计算复杂度高（需要计算核矩阵）
- 内存需求大
- 参数选择困难

### 5.2 增量 PCA（Incremental PCA）

增量 PCA 通过分批处理数据实现大规模 PCA，无需一次性加载全部数据到内存。

**算法流程：**
1. 初始化：均值、方差、主成分
2. 对每批数据：
   - 更新均值（增量公式）
   - 更新 SVD（使用矩阵分解技巧）
   - 更新方差
3. 计算解释方差

**优点：**
- 内存效率高
- 可处理流式数据
- 支持在线学习

**缺点：**
- 实现复杂
- 结果可能与标准 PCA 略有差异
- 需要选择合适的批次大小

### 5.3 稀疏 PCA（Sparse PCA）

在主成分上添加稀疏约束，使结果更易解释。

### 5.4 随机 PCA（Randomized PCA）

使用随机化算法加速大规模数据的 PCA。

## 6. PCA 的应用场景

### 6.1 数据压缩

减少数据维度，同时保留主要信息。常用于图像压缩、特征压缩。

### 6.2 噪声过滤

通过保留主要成分，去除低方差成分（通常是噪声）。

### 6.3 数据可视化

将高维数据降到 2D 或 3D 进行可视化。

### 6.4 特征工程

PCA 降维后的特征可以作为机器学习模型的输入。

### 6.5 探索性数据分析

通过分析解释方差比例，发现数据的内在维度和结构。

## 7. PCA 的局限性

### 7.1 线性假设

PCA 假设数据的主要结构是线性的，对非线性数据效果不佳。

### 7.2 方差不等于重要性

PCA 保留方差最大的方向，但方差大不一定意味着重要。

### 7.3 对异常值敏感

协方差矩阵对异常值敏感，可能影响 PCA 结果。

### 7.4 可解释性

主成分是原始特征的线性组合，可能难以解释。

## 8. 与其他降维方法的比较

| 方法 | 类型 | 优点 | 缺点 |
|------|------|------|------|
| PCA | 线性 | 简单、快速、理论完善 | 只能捕获线性结构 |
| t-SNE | 非线性 | 擅长可视化 | 计算慢、不保留全局结构 |
| UMAP | 非线性 | 快速、保留结构 | 需要调参 |
| LDA | 监督 | 利用标签信息 | 只适用于分类问题 |
| Autoencoder | 非线性 | 可学习复杂结构 | 需要大量数据、难训练 |

## 10. 调研结论

PCA 是一种经典且实用的降维方法，具有以下特点：

1. **数学基础扎实**：基于线性代数的特征值分解
2. **实现相对简单**：核心代码不超过 50 行
3. **应用广泛**：数据压缩、可视化、特征工程等
4. **理论完善**：有明确的优化目标和解释
