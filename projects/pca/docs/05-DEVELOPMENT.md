## 技术挑战与解决方案

### 挑战1：特征值分解的数值稳定性

**问题**：浮点运算可能导致协方差矩阵不对称。

**解决方案**：
```python
# 对称化处理
A = (A + A.T) / 2.0
```

### 挑战2：特征向量的符号不确定性

**问题**：特征向量可能有符号差异（v 和 -v 都是特征向量）。

**解决方案**：
- 测试时比较绝对值
- 或确保特征向量的第一个非零元素为正

### 挑战3：QR 算法的收敛速度

**问题**：无位移的 QR 算法收敛慢。

**解决方案**：
- 使用 Wilkinson shift
- 检查非对角线元素判断收敛

### 挑战4：幂迭代的收敛性

**问题**：幂迭代只能求最大特征值。

**解决方案**：
- 使用压缩（Deflation）技术
- 逐个求解特征值
