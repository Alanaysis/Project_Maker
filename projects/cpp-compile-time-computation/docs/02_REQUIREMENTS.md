| constexpr (C++14) | 5+ | 3.4+ | 2017+ |
| constexpr if (C++17) | 7+ | 5+ | 2017+ |
| consteval (C++20) | 10+ | 10+ | 2019+ |
| constinit (C++20) | 10+ | 10+ | 2019+ |

- 编译时间增加不超过 20%（相对于传统实现）
- 运行时性能提升至少 10%（对于适用场景）
- 代码可读性不低于传统实现
