# 产品设计文档

## 1. 产品定位

### 1.1 产品概述

约束求解器是一个 Python 库，用于求解约束满足问题 (CSP)。它提供了:
- 简洁的 API 接口
- 高效的求解算法
- 丰富的应用示例

### 1.2 目标场景

| 场景 | 描述 | 优先级 |
|------|------|--------|
| 数独求解 | 求解 9x9 数独谜题 | P0 |
| N 皇后 | 求解 N 皇后问题 | P0 |
| 排课问题 | 课程时间安排 | P1 |
| 资源分配 | 资源约束分配 | P2 |
| 调度优化 | 任务调度问题 | P2 |

## 2. 用户接口

### 2.1 命令行接口

```bash
# 求解数独
python examples/sudoku.py

# 求解 N 皇后
python examples/n_queens.py

# 求解排课问题
python examples/timetable.py
```

### 2.2 Python API

```python
from src import CSPSolver

# 创建求解器
solver = CSPSolver()

# 定义问题
x = solver.add_variable("x", [1, 2, 3])
y = solver.add_variable("y", [1, 2, 3])
solver.add_all_different([x, y])

# 求解
result = solver.solve()
```
