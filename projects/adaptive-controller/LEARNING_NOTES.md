# 自适应控制器 - 学习笔记

## 1. 自适应控制基础

### 1.1 什么是自适应控制

自适应控制是一种能够根据系统性能自动调整控制器参数的控制方法。与传统 PID 控制相比，自适应控制能够：

- 处理系统参数不确定性
- 应对时变系统
- 抑制未知扰动
- 保持闭环性能

### 1.2 自适应控制 vs 传统控制

```
传统控制 (PID):
┌─────────────────────────────────────────┐
│  r(t) ──▶ [PID控制器] ──▶ [被控系统] ──▶ y(t) │
│  固定参数: Kp, Ki, Kd                         │
└─────────────────────────────────────────┘

自适应控制 (MRAC):
┌─────────────────────────────────────────┐
│  r(t) ──▶ [可调控制器] ──▶ [被控系统] ──▶ y(t) │
│              ▲                              │
│              │ 参数调整                      │
│         [自适应律] ◀── 误差 e(t)             │
│              ▲                              │
│              │                              │
│         [参考模型] ◀── r(t)                 │
└─────────────────────────────────────────┘
```

### 1.3 自适应控制分类

1. **模型参考自适应控制 (MRAC)**
   - 定义参考模型描述期望行为
   - 通过自适应律调整控制器参数
   - 使系统输出跟踪参考模型输出

2. **自校正控制 (STR)**
   - 在线估计系统参数
   - 根据估计参数调整控制器
   - 适用于参数未知的系统

3. **增益调度控制**
   - 预先设计不同工作点的控制器
   - 根据运行条件切换控制器
   - 适用于已知工作范围的系统

## 2. 模型参考自适应控制 (MRAC)

### 2.1 MRAC 基本原理

MRAC 系统包含三个主要部分：

1. **参考模型**：定义期望的闭环系统行为
   ```
   ẏ_m = -a_m * y_m + b_m * r
   ```

2. **可调控制器**：参数可在线调整
   ```
   u = θ_r * r - θ_x * y + θ_d
   ```

3. **自适应律**：根据误差调整参数
   ```
   dθ/dt = f(e, φ)
   ```

### 2.2 自适应律设计

#### MIT 规则

MIT 规则基于梯度下降最小化误差平方：

```
J = 0.5 * e²
dθ/dt = -γ * ∂J/∂θ = -γ * e * ∂e/∂θ
```

**优点**：
- 实现简单
- 计算量小

**缺点**：
- 不能保证稳定性
- 可能产生振荡

**实现**：
```python
def _mit_update(self, e, r, y, dt):
    gradient_r = -e * r * 0.01
    gradient_x = e * y * 0.01
    gradient_d = -e * 0.01

    self.params["theta_r"] -= self.gamma * gradient_r * dt
    self.params["theta_x"] -= self.gamma * gradient_x * dt
    self.params["theta_d"] -= self.gamma * gradient_d * dt
```

#### Lyapunov 方法

Lyapunov 方法基于稳定性理论设计自适应律：

```
V = 0.5 * e² + 0.5 * (θ - θ*)ᵀ * Γ⁻¹ * (θ - θ*)
dθ/dt = -Γ * e * φ(x)
```

**优点**：
- 保证系统稳定
- 理论基础扎实

**缺点**：
- 需要选择合适的 Lyapunov 函数
- 计算量稍大

**实现**：
```python
def _lyapunov_update(self, e, r, y, dt):
    # 回归向量 φ(x) = [r, -y, 1]^T
    phi = np.array([r, -y, 1.0])

    # Lyapunov 自适应律: dθ/dt = -Γ * e * φ
    grad = self.gamma * e * phi

    # 更新参数
    self.params["theta_r"] -= grad[0] * dt
    self.params["theta_x"] -= grad[1] * dt
    self.params["theta_d"] -= grad[2] * dt
```

### 2.3 参考模型选择

#### 一阶参考模型

```
ẏ_m = -a_m * y_m + b_m * r
```

- 时间常数 τ = 1/a_m
- 稳态增益 K = b_m/a_m

**参数选择**：
- τ 越大，响应越慢
- K 决定稳态值

#### 二阶参考模型

```
ÿ_m + 2ζω_n * ẏ_m + ω_n² * y_m = ω_n² * r
```

- 自然频率 ω_n：控制响应速度
- 阻尼比 ζ：控制振荡特性
  - ζ < 1：欠阻尼（有振荡）
  - ζ = 1：临界阻尼
  - ζ > 1：过阻尼

**参数选择**：
- ω_n 越大，响应越快
- ζ = 0.7 为常用值（快速无超调）

## 3. 参数估计方法

### 3.1 递归最小二乘 (RLS)

#### 基本原理

对于线性系统：y = φᵀθ + ε

RLS 更新公式：
```
K = P * φ / (λ + φᵀ * P * φ)
θ = θ + K * e
P = (I - K * φᵀ) * P / λ
```

其中：
- θ 为参数估计值
- P 为协方差矩阵
- K 为卡尔曼增益
- λ 为遗忘因子

#### 遗忘因子

引入遗忘因子 λ (0 < λ ≤ 1) 以处理时变参数：
- λ = 1：无遗忘，适合常参数
- λ < 1：指数遗忘，适合时变参数
- 典型值：0.95 ~ 0.99

**实现**：
```python
def _rls_update(self, phi, error):
    # 计算卡尔曼增益
    S = np.dot(phi, np.dot(self.P, phi))
    K = np.dot(self.P, phi) / (1.0 + S)

    # 更新参数估计
    self.theta += K * error

    # 更新协方差矩阵
    I = np.eye(self.n_params)
    self.P = np.dot(I - np.outer(K, phi), self.P)
```

### 3.2 梯度下降法

最简单的参数估计方法：

```
θ = θ + γ * e * φ
```

其中 γ 为学习率。

**实现**：
```python
def _gradient_update(self, phi, error, dt):
    self.theta += self.gamma * error * phi * dt
```

### 3.3 方法对比

| 方法 | 收敛速度 | 计算复杂度 | 稳定性 | 适用场景 |
|------|---------|-----------|--------|---------|
| RLS | 快 | 中 | 好 | 线性系统 |
| 梯度下降 | 慢 | 低 | 一般 | 简单系统 |
| 带遗忘因子 RLS | 快 | 中 | 好 | 时变系统 |

## 4. 性能评估指标

### 4.1 跟踪性能

- **MSE (均方误差)**：MSE = E[e²]
- **RMSE (均方根误差)**：RMSE = √MSE
- **MAE (平均绝对误差)**：MAE = E[|e|]
- **IAE (绝对误差积分)**：IAE = ∫|e|dt
- **ISE (平方误差积分)**：ISE = ∫e²dt
- **ITAE (时间加权绝对误差积分)**：ITAE = ∫t|e|dt

### 4.2 瞬态性能

- **上升时间**：从 10% 到 90% 的时间
- **调节时间**：进入 ±2% 范围的时间
- **超调量**：(峰值 - 稳态值) / 稳态值 × 100%

### 4.3 控制性能

- **控制能量**：∫u²dt
- **最大控制量**：max|u(t)|

## 5. 实践心得

### 5.1 参数调整经验

1. **自适应增益 γ**：
   - 太小：收敛慢
   - 太大：可能振荡或发散
   - 建议：从 0.1 开始，逐步增大

2. **参考模型参数**：
   - 时间常数应大于被控对象
   - 阻尼比 0.7 左右为宜

3. **初始参数**：
   - 不需要非常准确
   - 避免极端值（太大或太小）

### 5.2 常见问题

1. **参数发散**：
   - 原因：自适应增益过大
   - 解决：减小增益或添加参数限制

2. **跟踪误差不收敛**：
   - 原因：参考模型选择不当
   - 解决：调整参考模型参数

3. **数值不稳定**：
   - 原因：时间步长过大
   - 解决：减小时间步长

### 5.3 调试技巧

1. **可视化**：
   - 绘制跟踪误差曲线
   - 观察参数变化趋势
   - 检查控制信号

2. **日志记录**：
   - 记录关键变量
   - 分析异常值

3. **单元测试**：
   - 测试各个模块
   - 验证算法正确性
