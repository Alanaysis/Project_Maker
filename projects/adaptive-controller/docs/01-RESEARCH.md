# 自适应控制器 - 调研报告

## 1. 自适应控制概述

### 1.1 什么是自适应控制

自适应控制是一种控制方法，其中控制器能够根据系统性能自动调整其参数，以应对系统动态特性的变化或不确定性。

### 1.2 自适应控制 vs 传统控制

| 特性 | 传统控制 (如 PID) | 自适应控制 |
|------|-------------------|-----------|
| 参数调整 | 手动调整 | 自动调整 |
| 系统建模 | 需要精确模型 | 可处理不确定性 |
| 时变系统 | 性能下降 | 能够适应 |
| 计算复杂度 | 低 | 较高 |
| 稳定性保证 | 容易分析 | 需要额外证明 |

### 1.3 自适应控制的分类

1. **模型参考自适应控制 (MRAC)**
   - 定义参考模型描述期望行为
   - 通过自适应律调整控制器参数
   - 使系统输出跟踪参考模型输出

2. **自校正控制 (STR)**
   - 在线估计系统参数
   - 根据估计参数调整控制器
   - 适用于参数未知的系统

3. **增益调度控制**
   - 预先设计不同工作点的控制器
   - 根据运行条件切换控制器
   - 适用于已知工作范围的系统

## 2. 模型参考自适应控制 (MRAC)

### 2.1 MRAC 基本原理

MRAC 系统包含三个主要部分：

1. **参考模型**：定义期望的闭环系统行为
   ```
   ẏ_m = -a_m * y_m + b_m * r
   ```

2. **可调控制器**：参数可在线调整
   ```
   u = θ_r * r - θ_x * y + θ_d
   ```

3. **自适应律**：根据误差调整参数
   ```
   dθ/dt = f(e, φ)
   ```

### 2.2 自适应律设计

#### MIT 规则

MIT 规则基于梯度下降最小化误差平方：

```
J = 0.5 * e²
dθ/dt = -γ * ∂J/∂θ = -γ * e * ∂e/∂θ
```

优点：
- 实现简单
- 计算量小

缺点：
- 不能保证稳定性
- 可能产生振荡

#### Lyapunov 方法

Lyapunov 方法基于稳定性理论设计自适应律：

```
V = 0.5 * e² + 0.5 * (θ - θ*)ᵀ * Γ⁻¹ * (θ - θ*)
dθ/dt = -Γ * e * φ(x)
```

优点：
- 保证系统稳定
- 理论基础扎实

缺点：
- 需要选择合适的 Lyapunov 函数
- 计算量稍大

### 2.3 参考模型选择

#### 一阶参考模型

```
ẏ_m = -a_m * y_m + b_m * r
```

- 时间常数 τ = 1/a_m
- 稳态增益 K = b_m/a_m

#### 二阶参考模型

```
ÿ_m + 2ζω_n * ẏ_m + ω_n² * y_m = ω_n² * r
```

- 自然频率 ω_n：控制响应速度
- 阻尼比 ζ：控制振荡特性
  - ζ < 1：欠阻尼（有振荡）
  - ζ = 1：临界阻尼
  - ζ > 1：过阻尼

## 3. 自校正控制 (STR)

### 3.1 STR 基本原理

自校正控制 (Self-Tuning Regulator) 结合在线参数估计和控制器设计：

1. **参数估计**：使用 RLS 等方法在线估计系统参数
2. **控制器设计**：根据估计参数设计控制器 (极点配置)
3. **控制应用**：应用控制信号
4. **重复**：循环执行以上步骤

### 3.2 极点配置

对于一阶系统 `y = a*y + b*u`，期望闭环极点为 `p`：

```
控制律: u = (p - a_hat) / b_hat * r - (a_hat + p) / b_hat * y
```

其中 `a_hat`, `b_hat` 为估计参数。

### 3.3 STR vs MRAC

| 特性 | MRAC | STR |
|------|------|-----|
| 设计思路 | 直接设计自适应律 | 估计+设计 |
| 参数估计 | 隐式 | 显式 |
| 计算量 | 较低 | 较高 |
| 灵活性 | 中 | 高 |
| 稳定性 | Lyapunov 保证 | 需额外分析 |

## 4. 参数估计方法

### 3.1 递归最小二乘 (RLS)

#### 基本原理

对于线性系统：y = φᵀθ + ε

RLS 更新公式：
```
K = P * φ / (λ + φᵀ * P * φ)
θ = θ + K * e
P = (I - K * φᵀ) * P / λ
```

#### 遗忘因子

引入遗忘因子 λ (0 < λ ≤ 1) 以处理时变参数：
- λ = 1：无遗忘，适合常参数
- λ < 1：指数遗忘，适合时变参数
- 典型值：0.95 ~ 0.99

### 3.2 梯度下降法

最简单的参数估计方法：

```
θ = θ + γ * e * φ
```

其中 γ 为学习率。

### 3.3 方法对比

| 方法 | 收敛速度 | 计算复杂度 | 稳定性 | 适用场景 |
|------|---------|-----------|--------|---------|
| RLS | 快 | 中 | 好 | 线性系统 |
| 梯度下降 | 慢 | 低 | 一般 | 简单系统 |
| 带遗忘因子 RLS | 快 | 中 | 好 | 时变系统 |

## 4. 被控对象建模

### 4.1 一阶系统

```
ẏ = -a * y + b * u
```

- 时间常数：τ = 1/a
- 稳态增益：K = b/a

### 4.2 二阶系统

```
ÿ + 2ζω_n * ẏ + ω_n² * y = ω_n² * u
```

状态空间表示：
```
ẋ₁ = x₂
ẋ₂ = -ω_n² * x₁ - 2ζω_n * x₂ + ω_n² * u
```

### 4.3 非线性系统

常见非线性特性：
- 饱和：|u| ≤ u_max
- 死区：|e| < δ 时不响应
- 滞后：输出滞后于输入
- 库仑摩擦：与速度方向相反的力

## 5. 性能评估指标

### 5.1 跟踪性能

- **MSE (均方误差)**：MSE = E[e²]
- **RMSE (均方根误差)**：RMSE = √MSE
- **MAE (平均绝对误差)**：MAE = E[|e|]
- **IAE (绝对误差积分)**：IAE = ∫|e|dt
- **ISE (平方误差积分)**：ISE = ∫e²dt
- **ITAE (时间加权绝对误差积分)**：ITAE = ∫t|e|dt

### 5.2 瞬态性能

- **上升时间**：从 10% 到 90% 的时间
- **调节时间**：进入 ±2% 范围的时间
- **超调量**：(峰值 - 稳态值) / 稳态值 × 100%

### 5.3 控制性能

- **控制能量**：∫u²dt
- **最大控制量**：max|u(t)|
