# 特征匹配学习笔记

## 什么是特征点？

特征点（Keypoint）是图像中具有独特性质的点，通常具有：
- **可重复性**: 在不同图像中能够被重复检测到
- **独特性**: 每个特征点都有独特的描述
- **局部性**: 只依赖局部图像区域
- **数量适中**: 数量足够但不过多

## 特征点类型

### 1. 角点（Corner）
- 两条边缘的交点
- 各方向灰度变化都大
- Harris角点检测器

### 2. 斑点（Blob）
- 区域内灰度相似
- 与周围区域明显不同
- LOG/DOG检测器

### 3. 边缘（Edge）
- 灰度变化剧烈的区域
- 通常不作为特征点（各向异性）

## SIFT算法（Scale-Invariant Feature Transform）

### 算法流程
1. **尺度空间构建**: 使用高斯模糊建立尺度空间
2. **关键点检测**: 在DOG（Difference of Gaussian）中找极值
3. **关键点定位**: 亚像素精度定位
4. **方向分配**: 计算梯度方向直方图
5. **描述子生成**: 16x16区域分为4x4子区域，每个8方向

### 关键公式

**高斯差分（DOG）**:
$$D(x,y,\sigma) = L(x,y,k\sigma) - L(x,y,\sigma)$$

**描述子维度**: 4 x 4 x 8 = 128维

### 优点
- 尺度不变性
- 旋转不变性
- 部分视角不变性
- 光照鲁棒性

### 缺点
- 计算量大
- 非实时

## ORB算法（Oriented FAST and Rotated BRIEF）

### 算法组成
1. **FAST角点检测**: 快速角点检测算法
2. **方向计算**: 灰度质心法计算方向
3. **BRIEF描述子**: 二进制描述子
4. **旋转BRIEF**: 根据方向旋转描述子

### FAST角点检测
- 检测圆形邻域16个像素
- 连续N个像素比中心亮/暗（N通常为12）
- 使用机器学习优化

### BRIEF描述子
- 在关键点附近随机选取点对
- 比较点对灰度值大小
- 生成256位二进制描述子

### 优点
- 速度极快（实时）
- 免费使用（无专利）
- 旋转不变性

### 缺点
- 尺度不变性较差
- 匹配精度略低于SIFT

## 特征匹配方法

### 1. 暴力匹配（Brute-Force）
- 遍历所有描述子对
- 计算距离（欧氏距离/汉明距离）
- 选择最近邻

### 2. FLANN匹配
- 快速最近邻搜索
- 使用KD树或LSH
- 适合大数据集

### 3. 比率测试（Lowe's Ratio Test）
```python
# ratio = dist_best / dist_second_best
# 如果 ratio < threshold (通常0.7)，则认为是好的匹配
if matches[0].distance < 0.7 * matches[1].distance:
    good_matches.append(matches[0])
```
