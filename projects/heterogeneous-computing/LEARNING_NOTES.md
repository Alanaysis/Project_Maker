## 第二阶段: 编程模型

### 2.1 CUDA 编程

**学习内容**:
- CUDA 编程模型
- 线程层次结构
- 内存管理

**笔记**:
```
在这里记录你的理解...

CUDA 编程模型:
1. Host (CPU) 和 Device (GPU)
2. Kernel 函数
3. 线程层次: Grid -> Block -> Thread

内存类型:
- Global Memory: 大容量，高延迟
- Shared Memory: 小容量，低延迟
- Registers: 最快，最小
```

**代码示例**:
```cpp
// CUDA 核函数示例
__global__ void vector_add(float* a, float* b, float* c, int n) {
    int idx = blockIdx.x * blockDim.x + threadIdx.x;
    if (idx < n) {
        c[idx] = a[idx] + b[idx];
    }
}

// 启动核函数
int threadsPerBlock = 256;
int blocksPerGrid = (n + threadsPerBlock - 1) / threadsPerBlock;
vector_add<<<blocksPerGrid, threadsPerBlock>>>(d_a, d_b, d_c, n);
```

**重点难点**:
- ⭐ 理解线程索引计算
- ⭐ 理解内存合并访问
- ⭐ 理解共享内存使用

**值得思考**:
- 💡 如何选择合适的线程块大小？
- 💡 如何优化内存访问模式？
