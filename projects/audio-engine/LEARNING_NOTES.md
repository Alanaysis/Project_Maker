## 核心概念笔记

### 1. 数字音频基础

#### 采样

**关键理解**：
- 采样是将连续模拟信号转换为离散数字信号的过程
- 采样率决定了能捕获的最高频率（奈奎斯特定理）
- CD 音质：44100 Hz 采样率，能捕获最高 22050 Hz 的频率

**个人笔记**：
- 采样率不是越高越好，要根据应用选择
- 电话语音 8000 Hz 就够了（语音主要在 300-3400 Hz）
- 音乐需要 44100 Hz 或更高

#### 量化

**关键理解**：
- 量化是将采样值映射到有限的数值级别
- 位深越大，动态范围越大
- 16-bit 有 65536 个级别，动态范围约 96 dB

**个人笔记**：
- 量化噪声是不可避免的
- 32-bit float 在专业音频中很常见
- 抖动（Dithering）可以改善量化质量

### 2. FFT 变换

#### 核心理解

**傅里叶变换的本质**：
- 任何信号都可以分解为不同频率的正弦波叠加
- FFT 是 DFT 的高效算法
- 时间复杂度从 O(N²) 降到 O(N log N)

**蝶形运算**：
```
X[k] = E[k] + W_N^k * O[k]
X[k+N/2] = E[k] - W_N^k * O[k]

其中：
- E[k] 是偶数项的 DFT
- O[k] 是奇数项的 DFT
- W_N^k = e^(-2πik/N) 是旋转因子
```

**个人笔记**：
- FFT 不是近似，是精确的数学变换
- 相位信息很重要，不能只看幅度
- 实数信号的 FFT 具有共轭对称性

#### 实现要点

```python
# 蝶形运算的核心
T = np.exp(-2j * np.pi * np.arange(N // 2) / N) * odd
result[:N // 2] = even + T
result[N // 2:] = even - T
```

**关键点**：
1. 旋转因子的计算
2. 分治策略的递归实现
3. 长度必须是 2 的幂次（需要填充）

#### 频谱分析

**幅度谱**：`|X[k]|` - 各频率的强度
**功率谱**：`|X[k]|²` - 各频率的能量
**相位谱**：`∠X[k]` - 各频率的相位

**帕塞瓦尔定理**：
```
Σ|x[n]|² = (1/N) Σ|X[k]|²
```
时域能量 = 频域能量（能量守恒）

### 3. 滤波器

#### 频域滤波原理

**核心思想**：
- 时域卷积 = 频域乘法
- 在频域设计滤波器更直观
- 通过修改频谱来实现滤波

**实现步骤**：
1. FFT 变换到频域
2. 设计滤波器频率响应
3. 频域乘法应用滤波器
4. IFFT 变换回时域

#### 滤波器类型

**低通滤波器**：
- 通过低频，衰减高频
- 应用：去噪、抗混叠

**高通滤波器**：
- 通过高频，衰减低频
- 应用：去除低频噪声、去直流

**带通滤波器**：
- 通过特定频段
- 应用：提取特定信号

**陷波滤波器**：
- 衰减特定频率
- 应用：去除工频干扰（50Hz/60Hz）

#### 过渡带设计

**理想滤波器**（硬截止）：
- 优点：简单
- 缺点：吉布斯现象（振铃）

**平滑过渡**（余弦过渡）：
```python
if f in transition_band:
    normalized = (f - f_low) / (f_high - f_low)
    mask = 0.5 * (1 + cos(π * normalized))
```
- 优点：无振铃
- 缺点：过渡带占用频率资源

### 4. 音频特效

#### 延迟效果

**原理**：
- 将信号延迟后与原始信号混合
- 反馈产生多次回声

**参数理解**：
- delay_time：回声间隔
- feedback：回声衰减速度
- mix：干湿比

**个人笔记**：
- 延迟时间通常在 50-500ms
- 反馈不能太大，否则会无限循环
- 延迟是很多效果的基础（混响、合唱等）

#### 混响效果

**Schroeder 混响模型**：
- 4 个并行梳状滤波器
- 2 个串联全通滤波器
- 模拟早期反射和后期混响

**梳状滤波器**：
```python
output[n] = input[n] + gain * output[n - delay]
```
- 产生梳状频率响应
- 不同延迟产生不同的音色

**全通滤波器**：
```python
output[n] = -gain * input[n] + input[n-delay] + gain * output[n-delay]
```
- 幅度响应平坦
- 只改变相位

**个人笔记**：
- 混响是音频处理中最复杂的效果之一
- 真实混响需要考虑房间形状、材料等
- Schroeder 模型是经典但简化的模型

#### 合唱效果

**原理**：
- LFO 调制延迟时间
- 产生多个略微失谐的声音
- 增加声音的厚度和宽度

**参数理解**：
- rate：LFO 频率（通常 0.5-5 Hz）
- depth：调制深度（通常 1-10 ms）
- mix：干湿比

**个人笔记**：
- 合唱效果模拟多人合唱的效果
- 调制深度影响效果的明显程度
- 可以用来让单声道声音听起来更宽

#### 失真效果

**原理**：
- 非线性处理，增加谐波
- 软削波（tanh）：温暖音色
- 硬削波（clip）：尖锐音色

**个人笔记**：
- 失真在电吉他音效中很常见
- 过度失真会损失音质
- 软削波比硬削波更自然

### 5. 降噪技术

#### 频谱减法

**原理**：
1. 估计噪声频谱（从静音段）
2. 从含噪信号频谱中减去噪声频谱
3. 使用谱下限防止过度衰减

**公式**：
```
|S(f)|² = |X(f)|² - α|N(f)|²
|S(f)| = max(|S(f)|², β|X(f)|²)
```

**参数理解**：
- α（过减因子）：控制去除多少噪声
  - 太小：噪声残留
  - 太大：音乐噪声
- β（谱下限）：防止过度衰减
  - 太小：音乐噪声
  - 太大：噪声残留

**个人笔记**：
- 频谱减法是最简单的降噪方法
- "音乐噪声"是频谱减法的主要问题
- 实际应用中需要仔细调整参数

#### 重叠相加法

**为什么需要**：
- 分帧处理会产生边界效应
- 重叠相加可以平滑过渡

**实现**：
```
帧1:   [AAAA]
帧2:       [BBBB]
帧3:           [CCCC]

重叠区域累加后归一化
```

### 6. 均衡器

#### 参数均衡器

**钟形函数**：
```python
bell = 1 / (1 + ((f² - fc²) / (f * fc / Q))²)
response = 1 + (gain - 1) * bell
```

**参数理解**：
- frequency：中心频率
- gain：增益（dB）
- Q：品质因子
  - Q 大：窄带宽
  - Q 小：宽带宽

**个人笔记**：
- 均衡器是最常用的音频处理工具
- Q 值影响效果的精细程度
- 过度均衡会损失自然感

#### 图示均衡器

**特点**：
- 固定频段（ISO 标准）
- 每个频段独立调节
- 直观易用

**ISO 标准频率**：
31.5, 63, 125, 250, 500, 1k, 2k, 4k, 8k, 16k Hz
