# 01 - 调研文档

## 音频处理基础知识

### 1. 数字音频基础

#### 采样定理（奈奎斯特定理）

**定理内容**：为了无失真地重建模拟信号，采样频率必须至少是信号最高频率的两倍。

```
采样率 >= 2 × 最高频率
```

**常见采样率**：
- 8,000 Hz：电话语音
- 22,050 Hz：低质量音频
- 44,100 Hz：CD 音质
- 48,000 Hz：专业音频
- 96,000 Hz：高解析度音频

#### 量化

**概念**：将连续的采样值映射到离散的数值级别。

**量化位深**：
- 8-bit：256 个级别，动态范围约 48 dB
- 16-bit：65,536 个级别，动态范围约 96 dB（CD 标准）
- 24-bit：16,777,216 个级别，动态范围约 144 dB（专业音频）
- 32-bit float：浮点表示，极大动态范围

**信噪比（SNR）**：
```
SNR ≈ 6.02 × N + 1.76 dB
其中 N 是量化位数
```

### 2. 频域分析

#### 傅里叶变换

**核心思想**：任何周期信号都可以分解为不同频率的正弦波叠加。

**离散傅里叶变换（DFT）**：
```
X[k] = Σ(n=0 to N-1) x[n] × e^(-j2πkn/N)
```

**快速傅里叶变换（FFT）**：
- DFT 的高效算法
- 时间复杂度：O(N log N) vs O(N²)
- 要求：N 为 2 的幂次（Cooley-Tukey 算法）

#### 频谱分析

**幅度谱**：|X[k]| - 各频率成分的强度

**功率谱**：|X[k]|² - 各频率成分的能量

**相位谱**：∠X[k] - 各频率成分的相位

### 3. 音频滤波

#### 滤波器类型

| 类型 | 作用 | 应用 |
|------|------|------|
| 低通滤波器（LPF） | 通过低频，衰减高频 | 去噪、抗混叠 |
| 高通滤波器（HPF） | 通过高频，衰减低频 | 去除低频噪声 |
| 带通滤波器（BPF） | 通过特定频段 | 提取特定信号 |
| 带阻滤波器（BSF） | 衰减特定频段 | 去除干扰频率 |

#### 滤波器实现方式

**FIR（有限脉冲响应）**：
- 优点：线性相位、稳定
- 缺点：需要较高阶数
- 实现：卷积

**IIR（无限脉冲响应）**：
- 优点：效率高、阶数低
- 缺点：可能不稳定、非线性相位
- 实现：差分方程

**频域滤波**：
- 优点：直观、易于设计
- 实现：FFT → 频域乘法 → IFFT

### 4. 音频特效

#### 延迟（Delay）

**原理**：将信号延迟后与原始信号混合

**参数**：
- delay_time：延迟时间
- feedback：反馈量（控制衰减）
- mix：干湿比

#### 混响（Reverb）

**原理**：模拟声音在空间中的反射

**Schroeder 混响模型**：
- 多个并行梳状滤波器
- 串联全通滤波器
- 模拟早期反射和后期混响

**参数**：
- room_size：房间大小
- damping：高频衰减
- mix：干湿比

#### 合唱（Chorus）

**原理**：LFO 调制延迟时间，产生多个失谐声音

**参数**：
- rate：LFO 频率
- depth：调制深度
- mix：干湿比

#### 失真（Distortion）

**原理**：非线性处理，增加谐波

**类型**：
- 软削波（tanh）：温暖音色
- 硬削波（clip）：尖锐音色

### 5. 降噪技术

#### 频谱减法（Spectral Subtraction）

**原理**：
1. 估计噪声频谱（从静音段）
2. 从含噪信号频谱中减去噪声频谱
3. 使用谱下限防止过度衰减

**公式**：
```
|S(f)|² = |X(f)|² - α|N(f)|²
|S(f)| = max(|S(f)|², β|X(f)|²)
```

**参数**：
- α：过减因子（1-3）
- β：谱下限（0-0.5）

#### 维纳滤波（Wiener Filter）

**原理**：基于最小均方误差准则的最优滤波器

**公式**：
```
H(f) = Sxx(f) / (Sxx(f) + Snn(f))
```

### 6. 均衡器

#### 参数均衡器（Parametric EQ）

**参数**：
- frequency：中心频率
- gain：增益（dB）
- Q：品质因子（带宽）

**类型**：
- Peak/Bell：钟形曲线
- Low Shelf：低频搁架
- High Shelf：高频搁架

#### 图示均衡器（Graphic EQ）

**特点**：
- 固定频段
- ISO 标准频率：31.5, 63, 125, 250, 500, 1k, 2k, 4k, 8k, 16k Hz
- 每个频段独立调节
