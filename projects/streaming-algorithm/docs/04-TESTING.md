# 测试文档

## 测试策略

### 单元测试
- 每个核心模块独立测试
- 边界条件覆盖（空窗口、满窗口、超出容量）

### 集成测试
- 多模块组合使用
- 大数据量性能验证

## 测试用例

| 模块 | 用例 | 验证点 |
|------|------|--------|
| SlidingWindow | 添加/平均/空窗口/包裹 | 正确性、边界 |
| ReservoirSampler | <k/==k/>k | 采样大小 |
| HyperLogLog | 唯一/重复/合并 | 估计精度 |
| TopK | 顺序/容量/空 | 排序、容量 |

## 运行

```bash
go test ./test/... -v
```
