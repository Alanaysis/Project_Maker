# 设备管理

物联网设备管理系统 - 设备注册、状态上报、远程控制、设备分组

## 核心功能

1. **设备注册** - 设备在线注册、认证、分配设备ID
2. **状态上报** - 定期上报设备状态（电量、信号、传感器数据）
3. **远程控制** - 远程发送控制指令、配置下发
4. **设备分组** - 按类型、区域、功能对设备分组管理

## 架构

```
设备端 ──> 设备管理系统 ──> 控制台
   │              │
   ├─ 注册请求     ├─ 设备认证
   ├─ 状态上报     ├─ 状态存储
   └─ 命令响应     └─ 命令下发
```

## 设备生命周期

```
设备注册 → 在线 → 状态上报 → 远程控制 → 离线/升级
```
