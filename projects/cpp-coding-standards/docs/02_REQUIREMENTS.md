# 需求分析

## 功能需求

### 1. 命名规范示例

**需求描述：**
提供各种命名风格的实际示例，展示良好和糟糕的命名实践。

**功能点：**
- 变量命名示例（camelCase, snake_case, PascalCase）
- 函数命名示例
- 类命名示例
- 常量命名示例
- 宏命名示例
- 命名空间命名示例
- 命名对比（良好 vs 糟糕）

**验收标准：**
- 每种命名风格有 3+ 个示例
- 包含良好和糟糕的对比
- 代码可编译运行
