# 01 - 市场调研

## 遗传算法概述

遗传算法 (Genetic Algorithm, GA) 是一种模拟自然选择过程的搜索启发式算法，由 John Holland 在 1975 年提出。它属于进化算法的一种，广泛应用于优化和搜索问题。

## 生物学背景

遗传算法模拟了生物进化过程中的几个关键机制：

1. **自然选择**: 适应环境的个体更有可能生存和繁殖
2. **遗传**: 子代继承父代的特征
3. **变异**: 随机改变个体的某些特征
4. **交叉**: 两个父代交换遗传信息产生子代

## 核心概念

### 染色体 (Chromosome)
- 表示问题解的编码
- 可以是二进制串、实数向量、排列等

### 基因 (Gene)
- 染色体上的一个位置
- 表示解的某个特征

### 适应度 (Fitness)
- 评估个体优劣的指标
- 越高表示个体越优秀

### 种群 (Population)
- 一组个体的集合
- 通常包含 50-200 个个体

## 算法流程

```
1. 初始化种群
   └─ 随机生成 N 个个体

2. 适应度评估
   └─ 计算每个个体的适应度值

3. 选择操作
   └─ 根据适应度选择父代
   └─ 常用方法：轮盘赌、锦标赛

4. 交叉操作
   └─ 父代交换遗传信息
   └─ 产生新的子代

5. 变异操作
   └─ 随机改变子代的某些基因
   └─ 增加种群多样性

6. 新种群
   └─ 用子代替换部分或全部父代
   └─ 返回步骤 2

7. 终止条件
   └─ 达到最大代数
   └─ 适应度收敛
   └─ 找到满意解
```

## 优缺点分析

### 优点
1. **全局搜索能力强**: 不易陷入局部最优
2. **适用性广**: 可处理各种类型的问题
3. **并行性好**: 种群中的个体可并行评估
4. **无需梯度信息**: 适用于不可微的问题

### 缺点
1. **计算成本高**: 需要大量适应度评估
2. **参数敏感**: 交叉率、变异率等参数需要调优
3. **收敛速度慢**: 相比梯度方法可能较慢
4. **不保证最优解**: 只能找到近似最优解

## TSP 问题详解

### 问题描述
给定一组城市和每对城市之间的距离，找到访问每个城市一次并返回起点的最短路径。

### 编码方式
- **排列编码**: 城市访问顺序的排列
- 例如: [0, 2, 1, 3] 表示 0→2→1→3→0

### 适应度函数
- 通常为路径总长度的倒数
- `fitness = 1 / total_distance`

### 特殊算子
- **顺序交叉 (OX)**: 保持相对顺序
- **交换变异**: 交换两个城市的位置
- **逆序变异**: 逆转一段路径
