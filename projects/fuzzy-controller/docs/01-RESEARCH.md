# 调研文档：模糊控制器

## 1. 模糊逻辑概述

### 1.1 什么是模糊逻辑

模糊逻辑（Fuzzy Logic）是由 Lotfi Zadeh 于1965年提出的一种处理不确定性和模糊性的数学方法。与传统的二值逻辑（非0即1）不同，模糊逻辑允许事物具有不同程度的"真"或"假"。

**核心思想**：
- 传统逻辑：一个命题要么为真（1），要么为假（0）
- 模糊逻辑：一个命题可以在0和1之间取任意值，表示其"真实程度"

### 1.2 模糊逻辑 vs 传统逻辑

| 特性 | 传统逻辑 | 模糊逻辑 |
|------|----------|----------|
| 真值范围 | {0, 1} | [0, 1] |
| 集合边界 | 清晰 | 模糊 |
| 推理方式 | 精确 | 近似 |
| 适用场景 | 确定性问题 | 不确定性问题 |

### 1.3 模糊逻辑的应用

- **控制系统**：温度控制、速度控制、工业自动化
- **人工智能**：专家系统、决策支持
- **图像处理**：边缘检测、图像分割
- **自然语言处理**：情感分析、文本分类
- **消费电子**：洗衣机、空调、相机自动对焦

## 2. 模糊集合

### 2.1 定义

模糊集合是传统集合的推广，允许元素以不同程度属于某个集合。

**数学定义**：
设 U 为论域，A 是 U 上的模糊集合，由隶属函数 μ_A: U → [0,1] 定义。

### 2.2 隶属函数

隶属函数描述元素属于模糊集合的程度。常见的隶属函数包括：

#### 三角形隶属函数 (Triangular)

```
μ(x) = max(0, min((x-a)/(b-a), (c-x)/(c-b)))
```

参数：a（左端点）、b（峰值）、c（右端点）

#### 梯形隶属函数 (Trapezoidal)

```
μ(x) = max(0, min((x-a)/(b-a), 1, (d-x)/(d-c)))
```

参数：a（左端点）、b（左肩）、c（右肩）、d（右端点）

#### 高斯隶属函数 (Gaussian)

```
μ(x) = exp(-(x - m)^2 / (2σ^2))
```

参数：m（均值）、σ（标准差）

### 2.3 模糊集合运算

- **并集 (Union)**：μ_A∪B(x) = max(μ_A(x), μ_B(x))
- **交集 (Intersection)**：μ_A∩B(x) = min(μ_A(x), μ_B(x))
- **补集 (Complement)**：μ_⌐A(x) = 1 - μ_A(x)

## 3. 模糊控制器

### 3.1 基本结构

模糊控制器通常包含以下组件：

```
输入 → 模糊化 → 规则库 → 推理引擎 → 去模糊化 → 输出
```

1. **模糊化 (Fuzzification)**：将精确输入转换为模糊值
2. **规则库 (Rule Base)**：存储模糊规则（IF-THEN）
3. **推理引擎 (Inference Engine)**：应用规则进行推理
4. **去模糊化 (Defuzzification)**：将模糊输出转换为精确值

### 3.2 模糊规则

模糊规则采用 IF-THEN 形式：

```
IF <前提> THEN <结论>
```

示例：
- IF 温度 IS 高 THEN 风扇转速 IS 快
- IF 温度 IS 低 AND 湿度 IS 高 THEN 加湿器功率 IS 低

### 3.3 推理方法

#### Mamdani 推理

1. 对每条规则计算激活强度
2. 使用激活强度裁剪输出隶属函数
3. 聚合所有裁剪后的隶属函数
4. 去模糊化得到精确输出

#### Sugeno 推理

输出是输入的线性函数，计算更简单。

### 3.4 去模糊化方法

- **重心法 (COG)**：计算隶属函数曲线下的重心
- **最大隶属度法 (MOM)**：返回最大隶属度对应值的平均值
- **面积中心法 (COS)**：类似重心法
