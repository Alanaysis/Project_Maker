# 01-RESEARCH.md - 光线追踪渲染器

## 1. 技术背景

### 1.1 光线追踪概述

光线追踪（Ray Tracing）是一种通过模拟光线传播来生成图像的渲染技术。与光栅化不同，光线追踪能够自然地处理：

- 全局光照
- 阴影
- 反射
- 折射
- 景深

### 1.2 核心算法

光线追踪的基本算法：

1. **光线发射**：从相机通过每个像素发射光线
2. **场景求交**：计算光线与场景中物体的交点
3. **光照计算**：根据材质和光源计算交点颜色
4. **递归追踪**：对于反射/折射材质，递归发射新光线

### 1.3 数学基础

#### 光线方程
```
P(t) = O + t * D
```
- P(t): 光线上参数为 t 的点
- O: 光线起点（origin）
- D: 光线方向（direction）
- t: 参数

#### 光线-球体求交
给定球心 C，半径 r，光线 P(t) = O + tD：
```
t = (-b ± sqrt(b² - 4ac)) / 2a
其中：
a = D·D
b = 2D·(O - C)
c = (O - C)·(O - C) - r²
```

#### 光线-平面求交
给定平面上一点 P₀，法线 N，光线 P(t) = O + tD：
```
t = (P₀ - O)·N / (D·N)
```

## 2. 相关技术调研

### 2.1 材质模型

#### Lambertian（漫反射）
- 最简单的材质模型
- 光线在交点处随机散射
- 颜色由反照率（albedo）决定

#### Metal（金属）
- 光线按反射定律反射
- 可添加模糊度（fuzz）模拟粗糙表面

#### Dielectric（电介质）
- 玻璃、水等透明材质
- 使用 Snell 定律计算折射
- 使用 Schlick 近似计算反射率

### 2.2 抗锯齿

多重采样抗锯齿（MSAA）：
- 对每个像素多次采样
- 取平均值作为最终颜色
- 采样数越多，图像越平滑

### 2.3 伽马校正

线性空间 vs 伽马空间：
- 渲染在颜色线性空间进行
- 显示前需要伽马校正
- 简单的伽马校正：c' = sqrt(c)
