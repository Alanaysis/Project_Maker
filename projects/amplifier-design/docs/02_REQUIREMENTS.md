# 放大器设计 - 需求文档

## 1. 项目目标

实现一个完整的模拟放大器设计库，覆盖从 BJT 基本放大器到运放应用电路的设计和分析。

## 2. 功能需求

### 2.1 BJT 基本放大器

| 编号 | 功能 | 优先级 | 说明 |
|------|------|--------|------|
| F-001 | 共射放大器 | P0 | 直流偏置计算、小信号增益、输入/输出阻抗 |
| F-002 | 共集放大器 | P0 | 电压增益≈1、高输入阻抗、低输出阻抗 |
| F-003 | 共基放大器 | P0 | 高增益、同相、低输入阻抗 |
| F-004 | 工作点分析 | P0 | IC, VCE, VB, VE, VC |
| F-005 | 温度特性 | P1 | VBE 温漂、β 温漂的影响 |

### 2.2 运算放大器

| 编号 | 功能 | 优先级 | 说明 |
|------|------|--------|------|
| F-010 | 反相放大器 | P0 | Av = -Rf/Rin，带宽，输入阻抗 |
| F-011 | 同相放大器 | P0 | Av = 1+Rf/Rin，高输入阻抗 |
| F-012 | 差分放大器 | P0 | 差模增益、共模增益、CMRR |
| F-013 | 仪表放大器 | P0 | 三运放结构，可编程增益 |
| F-014 | 有限 A_OL 影响 | P1 | 实际增益与理想增益的偏差 |

### 2.3 频率响应

| 编号 | 功能 | 优先级 | 说明 |
|------|------|--------|------|
| F-020 | 增益带宽积 | P0 | GBW = Av × BW 常数关系 |
| F-021 | 频率响应计算 | P0 | 单极点/多极点系统的增益和相位 |
| F-022 | 相位裕度分析 | P0 | 环路稳定性评估 |
| F-023 | 相位补偿 | P0 | 主极点、超前、滞后、密勒补偿 |
| F-024 | 阶跃响应估算 | P1 | 超调量、建立时间 |

### 2.4 实际应用

| 编号 | 功能 | 优先级 | 说明 |
|------|------|--------|------|
| F-030 | 信号调理 | P0 | 增益调整、电平移位、AC耦合 |
| F-031 | 热电偶放大 | P0 | 仪表放大器、高增益差分 |
| F-032 | 应变片放大 | P0 | 惠斯通电桥 + 仪表放大器 |
| F-033 | 光电二极管 TIA | P0 | 跨阻放大器 |
| F-034 | 压电传感器放大 | P1 | 高阻抗放大器 |
| F-035 | 音频前置放大 | P0 | 低噪声、适当增益 |
| F-036 | 音调控制 | P1 | Baxandall 型均衡器 |
| F-037 | 功率驱动 | P1 | 驱动级设计 |

## 3. 非功能需求

### 3.1 精度要求

| 参数 | 要求 |
|------|------|
| DC 增益精度 | < 1% 误差 (相对理想值) |
| 频率计算精度 | < 0.1% 误差 |
| 数值稳定性 | 无 NaN/Inf (合理输入范围内) |

### 3.2 代码质量

- 类型提示 (Type Hints)
- 完整文档字符串 (Docstring)
- 单元测试覆盖率 > 80%
- 模块化设计，接口统一

### 3.3 依赖

- Python 3.8+
- NumPy (数值计算)
- Matplotlib (可选，可视化)
- pytest (测试)

## 4. 数据模型

### 4.1 BJT 参数

```python
@dataclass
class BJTParams:
    beta: float = 100.0    # 电流增益
    V_A: float = 100.0     # Early 电压
    V_BE: float = 0.7      # 基极-发射极电压
    V_T: float = 0.026     # 热电压
```

### 4.2 运放参数

```python
@dataclass
class OpAmpParams:
    A_OL: float = 1e5      # 开环增益
    GBW: float = 1e6       # 增益带宽积
    SR: float = 0.5e6      # 转换速率
    V_sat: float = 13.5    # 饱和电压
    CMRR: float = 90.0     # 共模抑制比 (dB)
```

## 5. 验收标准

### 5.1 BJT 放大器

- [ ] 共射放大器增益为负，幅度 > 1
- [ ] 共集放大器增益在 0.9~1.0 之间
- [ ] 共基放大器增益为正，幅度 > 1
- [ ] 三种放大器阻抗特性符合预期
- [ ] 工作点在放大区 (VCE > 1V)

### 5.2 运放电路

- [ ] 反相放大器增益 = -Rf/Rin
- [ ] 同相放大器增益 = 1 + Rf/Rin
- [ ] 差分放大器抑制共模信号
- [ ] 仪表放大器 CMRR > 80dB
- [ ] 带宽 = GBW / |Av|

### 5.3 频率响应

- [ ] GBW = Av × BW = 常数
- [ ] 单极点系统相位从 0° 到 -90°
- [ ] 相位裕度分析正确
- [ ] 补偿技术可提高稳定性

### 5.4 应用

- [ ] 信号调理可调整增益和偏移
- [ ] 传感器放大器灵敏度计算正确
- [ ] 音频放大器带宽覆盖 20Hz~20kHz
