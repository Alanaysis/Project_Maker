# 放大器设计调研报告

## 1. 调研背景

放大器是模拟电路中最核心的功能模块之一。本调研旨在全面了解放大器的理论基础、分类方法、设计要点和应用场景。

## 2. 放大器分类

### 2.1 按器件类型分类

| 类型 | 器件 | 特点 | 典型应用 |
|------|------|------|---------|
| BJT 放大器 | 双极型晶体管 | 电流控制、高跨导 | 通用放大、射频 |
| MOSFET 放大器 | 场效应管 | 电压控制、高输入阻抗 | 集成电路、功率放大 |
| 运算放大器 | IC | 高增益、多配置 | 信号处理、仪器仪表 |

### 2.2 BJT 放大器配置

| 配置 | 增益 | 输入阻抗 | 输出阻抗 | 相移 | 主要用途 |
|------|------|---------|---------|------|---------|
| 共射 (CE) | 高 | 中 | 高 | 180° | 电压放大 |
| 共集 (CC) | ≈1 | 高 | 低 | 0° | 缓冲/阻抗匹配 |
| 共基 (CB) | 高 | 低 | 高 | 0° | 高频放大 |

### 2.3 运算放大器电路

| 电路 | 增益公式 | 特点 |
|------|---------|------|
| 反相放大器 | Av = -Rf/Rin | 虚地、反相 |
| 同相放大器 | Av = 1 + Rf/Rin | 高输入阻抗、同相 |
| 差分放大器 | Ad = Rf/R1 | 抑制共模 |
| 仪表放大器 | G = 1 + 2R1/Rg | 极高CMRR、高输入阻抗 |

## 3. BJT 放大器理论

### 3.1 BJT 工作区

BJT 有三个工作区：

1. **截止区**: V_BE < 0.7V, I_C ≈ 0
2. **放大区**: V_BE ≈ 0.7V, I_C = β × I_B
3. **饱和区**: V_CE < V_CE(sat), I_C 不受 I_B 控制

**放大器设计要求**: BJT 必须工作在放大区。

### 3.2 小信号模型

在放大区，BJT 的小信号等效模型：

```
         r_pi
B ────┤├──── B' ──── g_m * V_be ──── C
              │
              └──── E
```

参数关系：
- 跨导: g_m = I_C / V_T (V_T ≈ 26mV @ 室温)
- 输入电阻: r_pi = β / g_m = β × V_T / I_C
- 输出电阻: r_o = V_A / I_C (Early 效应)

### 3.3 偏置电路

分压偏置是最常用的偏置方式：

```
VCC
 |
 R1
 |
 +---- B
 |     |
 R2   BJT
 |     |
GND   RE
       |
      GND
```

工作点计算：
- V_B = V_CC × R2 / (R1 + R2)
- V_E = V_B - V_BE
- I_C ≈ I_E = V_E / R_E
- V_CE = V_CC - I_C × (R_C + R_E)

## 4. 运算放大器理论

### 4.1 理想运放特性

| 参数 | 理想值 | 实际典型值 |
|------|-------|-----------|
| 开环增益 A_OL | ∞ | 10^5 ~ 10^6 |
| 输入阻抗 | ∞ | 10^6 ~ 10^12 Ω |
| 输出阻抗 | 0 | 10 ~ 100 Ω |
| 带宽 | ∞ | GBW = 1~100 MHz |
| CMRR | ∞ | 80~120 dB |
| 失调电压 | 0 | 0.1~10 mV |

### 4.2 虚短虚断

理想运放分析的两个基本法则：

1. **虚短**: V+ = V- (两输入端电压相等)
2. **虚断**: I+ = I- = 0 (输入端不取电流)

### 4.3 增益带宽积 (GBW)

GBW 是运放最重要的频率参数：

```
GBW = |Av| × BW = 常数
```

含义：
- 增益越高，带宽越窄
- 增益越低，带宽越宽
- 在单位增益时，带宽 = GBW

### 4.4 稳定性

反馈放大器的稳定性由相位裕度决定：

- **相位裕度 > 60°**: 良好阻尼，快速建立
- **相位裕度 45°~60°**: 轻微过冲
- **相位裕度 < 45°**: 明显振铃
- **相位裕度 < 0°**: 不稳定，持续振荡

## 5. 频率响应

### 5.1 波特图

波特图描述放大器的频率特性：

- **幅频响应**: 增益 (dB) vs 频率 (对数)
- **相频响应**: 相位 (度) vs 频率 (对数)

单极点系统：
- 低频: 增益平坦
- 截止频率: 增益下降 3dB
- 高频: 增益以 -20dB/dec 下降

### 5.2 相位补偿技术

| 补偿方法 | 原理 | 优点 | 缺点 |
|---------|------|------|------|
| 主极点补偿 | 降低开环带宽 | 简单可靠 | 降低带宽 |
| 超前补偿 | 添加相位超前 | 保持带宽 | 增加复杂度 |
| 滞后补偿 | 降低高频增益 | 改善噪声 | 降低带宽 |
| 密勒补偿 | 极点分离 | 集成电路友好 | 需要内部节点 |

## 6. 实际应用

### 6.1 信号调理

传感器信号通常需要：
- **放大**: 将微弱信号放大到 ADC 可接受范围
- **滤波**: 去除噪声和干扰
- **电平移位**: 调整直流偏置
- **阻抗变换**: 匹配信号源和负载

### 6.2 传感器放大

| 传感器 | 输出特性 | 放大器类型 | 增益需求 |
|--------|---------|-----------|---------|
| 热电偶 | ~40μV/°C | 仪表放大器 | 100~1000 |
| 应变片 | ~1mV/V | 仪表放大器 | 100~500 |
| 光电二极管 | ~0.5A/W | 跨阻放大器 | 取决于 R_f |
| 压电传感器 | 高阻抗 | 电荷放大器 | 10~100 |

### 6.3 音频放大

音频放大链路：
1. **前置放大**: 麦克风/拾音器信号放大 (20~60dB)
2. **音调控制**: Baxandall 型均衡器
3. **功率放大**: 驱动扬声器 (几瓦~几百瓦)

## 7. 设计流程

```
需求分析 → 拓扑选择 → 偏置设计 → 增益计算
    ↓                                    ↓
器件选型 ← 补偿设计 ← 带宽分析 ← 频率响应
    ↓
PCB 设计 → 仿真验证 → 原型测试 → 调试优化
```
