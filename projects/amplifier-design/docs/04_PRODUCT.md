# 放大器设计 - 产品文档

## 1. 产品概述

本项目是一个 Python 模拟放大器设计库，提供 BJT 基本放大器、运算放大器电路和频率响应分析功能，适用于教学、原型设计和电路分析。

## 2. 功能模块

### 2.1 BJT 基本放大器

**共射放大器 (Common Emitter)**
- 输入: 偏置电阻、集电极/发射极电阻、电源电压、BJT 参数
- 输出: 电压增益、输入/输出阻抗、工作点
- 特点: 高增益、反相、中等输入阻抗

**共集放大器 (Common Collector)**
- 输入: 偏置电阻、发射极电阻、电源电压
- 输出: 电压增益 (≈1)、输入/输出阻抗
- 特点: 同相、高输入阻抗、低输出阻抗

**共基放大器 (Common Base)**
- 输入: 发射极/集电极电阻、偏置电流
- 输出: 电压增益、输入/输出阻抗
- 特点: 高增益、同相、低输入阻抗

### 2.2 运算放大器电路

**反相放大器**: Av = -Rf/Rin
**同相放大器**: Av = 1 + Rf/Rin
**差分放大器**: 差模增益、CMRR
**仪表放大器**: 可编程增益 G = 1 + 2R1/Rg

### 2.3 频率响应

- 增益带宽积分析
- 相位补偿设计 (主极点、超前、滞后、密勒)
- 环路稳定性评估 (相位裕度、增益裕度)

### 2.4 实际应用

- 信号调理: 增益调整、电平移位、AC 耦合
- 传感器放大: 热电偶、应变片、光电二极管、压电
- 音频放大: 前置放大、音调控制、功率驱动、分频器

## 3. 使用指南

### 3.1 快速开始

```python
from src.bjt import CommonEmitter, BJTParams
from src.opamp import InvertingAmp
from src.applications import SensorAmplifier

# BJT 共射放大器
ce = CommonEmitter(R_B1=47e3, R_B2=10e3, R_C=4.7e3, R_E=2.2e3, V_CC=12)
print(f"增益: {ce.voltage_gain():.1f}")

# 反相放大器
inv = InvertingAmp(R_in=10e3, R_f=100e3)
print(f"增益: {inv.gain()}")

# 热电偶放大
sa = SensorAmplifier()
tc = sa.thermocouple_amp(R_g=100)
print(f"灵敏度: {tc['sensitivity']*1e3:.2f} mV/°C")
```

### 3.2 典型工作流

1. 确定信号源特性 (幅度、频率、阻抗)
2. 选择放大器拓扑
3. 配置参数 (电阻、增益)
4. 调用 `summary()` 查看完整参数
5. 验证带宽和稳定性

## 4. 性能指标

| 参数 | BJT 放大器 | 运放电路 |
|------|-----------|---------|
| 增益范围 | 1 ~ 100 | 1 ~ 10000 |
| 带宽 | DC ~ 数百 kHz | DC ~ GBW/Av |
| 输入阻抗 | 1k ~ 100k Ω | 10k ~ TΩ |
| 输出阻抗 | 1k ~ 10k Ω | ~0 Ω (理想) |
| THD | 较高 | 低 |

## 5. 局限性

- 小信号分析，不包含大信号非线性
- 简化的温度模型
- 不包含噪声分析
- 不包含 PCB 寄生参数
- 运放使用一阶模型 (单极点)
