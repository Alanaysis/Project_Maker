# 市场调研报告

## 1. 问题定义

### 要解决的问题
实现图层混合和蒙版操作，支持图像合成。

### 为什么这个问题重要
图层合成是 Photoshop、GIMP 等图像编辑软件的核心功能。

## 2. 同类型项目概览

| 项目 | 核心特点 | 技术栈 |
|------|---------|--------|
| Photoshop | 商业级图像编辑 | C++ |
| GIMP | 开源图像编辑 | C |
| Skia | 图形引擎 | C++ |
| Cairo | 2D 图形库 | C |

## 3. 技术变体分析

**核心循环**：
```
图层输入 → 混合计算 → 遮罩应用 → 合成输出
```

**混合模式**：Normal, Multiply, Screen, Overlay, Darken, Lighten, Difference

**蒙版操作**：Apply Mask, Erode, Dilate

## 4. 我们的选择

实现核心图层合成引擎：
- 9 种混合模式
- Alpha 蒙版
- 腐蚀/膨胀形态学操作
- 多层合成
