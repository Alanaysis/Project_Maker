## 1. 核心概念

### 图层混合
将多个图层的像素按特定算法组合。

### Alpha 混合
`result = top * alpha + bottom * (1 - alpha)` 是基础。

### 蒙版
通过 Alpha 通道控制像素的可见性。

## 2. 重点难点

### 混合模式数学
不同混合模式有不同的数学公式，需要理解每种模式的视觉效应。

## 3. 延伸学习
