# 键盘驱动及优化

## 项目简介

实现一个键盘驱动，支持按键扫描、中断处理、按键优化。通过这个项目，深入理解键盘硬件原理、中断处理机制和按键去抖算法。

## 学习目标

- ⭐ **键盘硬件原理**：理解矩阵键盘扫描原理、按键物理特性
- ⭐ **中断处理机制**：掌握硬件中断、中断处理函数、中断上下文
- ⭐ **按键去抖算法**：学习硬件去抖和软件去抖的实现
- **DMA传输**：了解DMA在键盘数据传输中的应用
- **输入子系统**：理解Linux输入子系统架构

## 核心功能

1. **按键扫描**：支持矩阵键盘扫描
2. **中断处理**：响应按键中断事件
3. **按键映射**：将扫描码转换为键值
4. **去抖处理**：消除按键抖动干扰
5. **输入事件**：生成标准输入事件

## 重点难点

### ⭐ 矩阵键盘扫描

矩阵键盘通过行列交叉实现按键检测：
- 行线输出扫描信号
- 列线读取按键状态
- 需要处理多键同时按下的情况

### ⭐ 中断处理

中断处理的关键点：
- 中断上下文不能睡眠
- 需要快速响应和处理
- 中断处理程序要尽量简短

### ⭐ 去抖算法

按键抖动的处理：
- 硬件去抖：RC电路滤波
- 软件去抖：定时器检测、状态机

## 值得思考

💡 **为什么需要去抖？**
按键按下时会产生机械抖动，通常持续5-20ms，不处理会导致重复触发。

💡 **中断和轮询哪个更好？**
中断响应快但开销大，轮询简单但延迟高，需要根据场景选择。

💡 **如何处理多键同时按下？**
需要考虑按键冲突（ghosting）和按键翻转（rollover）问题。
