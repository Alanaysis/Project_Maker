**学习内容**:
- [ ] 理解按键抖动原因
- [ ] 学习去抖方法
- [ ] 比较不同去抖算法

**笔记**:
```
在这里记录去抖算法的要点...

按键抖动：
- 机械抖动持续5-20ms
- 会产生多个脉冲
- 需要去抖处理

去抖方法：
1. 硬件去抖：RC电路、施密特触发器
2. 软件去抖：定时器、计数器、状态机
```

**算法比较**:
```
定时器去抖：
- 优点：简单、响应快
- 缺点：固定延迟

计数器去抖：
- 优点：可调阈值
- 缺点：需要多次采样

状态机去抖：
- 优点：逻辑清晰、可扩展
- 缺点：实现复杂
```

#### 事件队列
**学习内容**:
- [ ] 设计事件队列
- [ ] 实现队列操作
- [ ] 处理队列满/空

**笔记**:
```
在这里记录事件队列的要点...

循环队列：
- 入队：tail = (tail + 1) % SIZE
- 出队：head = (head + 1) % SIZE
- 判空：head == tail
- 判满：(tail + 1) % SIZE == head
```

**代码示例**:
```c
// 循环队列实现
typedef struct {
    key_event_t events[64];
    int head;
    int tail;
} event_queue_t;

void queue_push(event_queue_t *queue, key_event_t *event)
{
    int next = (queue->tail + 1) % 64;
    if (next != queue->head) {
        queue->events[queue->tail] = *event;
        queue->tail = next;
    }
}
```
