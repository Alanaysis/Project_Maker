# 学习笔记: 傅里叶变换

## 核心概念

### 1. 傅里叶变换的本质

傅里叶变换的本质是**将信号从时域变换到频域**。

时域告诉我们信号随时间如何变化，频域告诉我们信号包含哪些频率成分。

**直觉**: 就像白光可以分解为彩虹（不同颜色=不同频率），任何信号都可以分解为不同频率的正弦波。

### 2. DFT vs FFT

- **DFT** (Discrete Fourier Transform): 离散傅里叶变换，O(N^2) 复杂度
- **FFT** (Fast Fourier Transform): 快速傅里叶变换，O(N log N) 复杂度

FFT 不是一种新的变换，而是 DFT 的快速算法。就像快速排序不是一种新的排序，而是排序的快速实现。

### 3. 旋转因子 (Twiddle Factor)

```
W_N^k = e^{-j*2*π*k/N}
```

旋转因子是 FFT 的核心。它是一个复数，在复平面上表示一个角度为 -2πk/N 的单位向量。

**关键性质**:
- 周期性: W_N^{k+N} = W_N^k
- 对称性: W_N^{k+N/2} = -W_N^k
- 这两个性质使得 FFT 的分治策略成为可能

### 4. 蝶形运算

```
X[k] = E[k] + W * O[k]
X[k+N/2] = E[k] - W * O[k]
```

蝶形运算的名字来源于数据流图的形状。它是 FFT 的基本计算单元。

### 5. 幅度谱和相位谱

- **幅度谱** |X[k]|: 每个频率分量有多强
- **相位谱** ∠X[k]: 每个频率分量的起始相位

幅度谱告诉我们"有哪些频率"，相位谱告诉我们"这些频率的相对位置"。

## 数学推导

### DFT 推导

从连续傅里叶变换出发:

```
X(f) = ∫ x(t) e^{-j2πft} dt
```

离散化: t = nT, f = k/(NT), dt = T

```
X(k/(NT)) ≈ T * Σ x(nT) e^{-j2πkn/N}
```

忽略常数 T，得到 DFT:

```
X[k] = Σ x[n] e^{-j2πkn/N}
```

### FFT 分治推导

将 N 点 DFT 拆分为两个 N/2 点 DFT:

```
X[k] = Σ_{n=0}^{N-1} x[n] W_N^{kn}
     = Σ_{n even} x[n] W_N^{kn} + Σ_{n odd} x[n] W_N^{kn}
     = Σ_{r=0}^{N/2-1} x[2r] W_N^{2rk} + Σ_{r=0}^{N/2-1} x[2r+1] W_N^{(2r+1)k}
     = Σ_{r=0}^{N/2-1} x[2r] W_{N/2}^{rk} + W_N^k Σ_{r=0}^{N/2-1} x[2r+1] W_{N/2}^{rk}
     = E[k] + W_N^k O[k]
```

其中 E[k] 和 O[k] 分别是偶数和奇数索引的 N/2 点 DFT。

利用周期性 W_N^{k+N/2} = -W_N^k:

```
X[k] = E[k] + W_N^k O[k]
X[k+N/2] = E[k] - W_N^k O[k]
```

## 实现要点

### 1. 补零到 2 的幂

```python
if N & (N - 1) != 0:  # N 不是 2 的幂
    next_power = 1
    while next_power < N:
        next_power <<= 1
    x = np.pad(x, (0, next_power - N))
```

补零不会改变频谱的形状，只是增加了频率分辨率。

### 2. 位反转

迭代 FFT 需要位反转排列:

```
索引 0 (000) → 0 (000)
索引 1 (001) → 4 (100)
索引 2 (010) → 2 (010)
索引 3 (011) → 6 (110)
索引 4 (100) → 1 (001)
索引 5 (101) → 5 (101)
索引 6 (110) → 3 (011)
索引 7 (111) → 7 (111)
```

### 3. 利用 FFT 计算 IFFT

```python
def ifft(X):
    return np.conj(fft(np.conj(X))) / len(X)
```

利用 IDFT 公式中指数符号相反的特点，通过共轭变换复用 FFT 代码。

## 频谱分析

### 1. 频率分辨率

```
Δf = fs / N
```

- fs: 采样率
- N: FFT 点数
- 更多采样点 → 更高的频率分辨率

### 2. 频谱泄漏

当信号频率不在 FFT 的频率 bin 上时，能量会泄漏到相邻的 bin。

解决方法: 加窗 (Windowing)

### 3. Parseval 定理

```
Σ|x[n]|^2 = (1/N) Σ|X[k]|^2
```

时域和频域的总能量相等。这是能量守恒的体现。

### 4. 卷积定理

```
FFT(x * h) = FFT(x) · FFT(h)
```

时域卷积 = 频域乘法。这使得卷积可以通过 FFT 高效计算。

## 常见陷阱

### 1. 频谱对称性

实数信号的频谱关于 N/2 对称:
```
X[k] = conj(X[N-k])
```

因此只需看前一半 (0 到 N/2)。

### 2. 补零的影响

补零增加频率分辨率（bin 更密），但不增加频率精度（不能分辨更近的频率）。

### 3. 采样定理

采样率必须大于信号最高频率的 2 倍:
```
fs > 2 * f_max
```

否则会出现混叠 (Aliasing)。
