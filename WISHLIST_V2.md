**一句话描述**：实现基于 Redis/ZooKeeper 的分布式锁

**学习目标**：
- 理解分布式锁原理
- 掌握 Redlock 算法
- 学会锁续期

**技术栈**：
- 主语言：Go / Python
- 框架：无
- 其他：Redis / ZooKeeper

**核心循环**：
```
锁请求 → 锁获取 → 业务执行 → 锁释放
```

**参考项目**：
- Redlock
- Curator
- etcd

**优先级**：P1
**预估时长**：6 小时
